# Digests

各カテゴリ（Claude / GameDev / Fables / Maxims）に直近1週間で追加された情報を分析し、
**今週読むべき情報・活用案・ゲーム企画の種**を1枚にまとめる Weekly Digest の保管場所。

単なる更新ファイル一覧ではなく、日曜に Digest を1本読むだけで今週の追加情報が把握できることを目的とする。

## 方式

- 週次生成（毎週日曜 08:00 JST、各カテゴリ収集の終了後）
- 対象期間: 実行日の7日前 00:00 JST 〜 実行日 07:59 JST
- ワークフローは `.claude/skills/weekly-digest/SKILL.md` に定義

## ファイル構成

| パス | 役割 |
|------|------|
| `README.md` | 本ファイル（概要） |
| `INDEX.md` | 生成済み Digest 一覧（新しいものが上） |
| `YYYY/YYYY-MM-DD.md` | 各週の Digest 本体 |

## 実行方法

手動実行のトリガーワード:

- 「Weekly Digestを作成して」「今週のDigestを作って」
- 「今週追加された情報をまとめて」「今週の情報を整理して」
- 「Weekly Digestを再生成して」（既存ファイルを上書き）

同一日付の Digest が既に存在する場合、通常実行では新規作成しない（再生成指示時のみ上書き）。
