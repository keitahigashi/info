# 逃げちゃダメだ

> 逃げちゃダメだ、逃げちゃダメだ、逃げちゃダメだ……やります、僕が乗ります！

## 著者

庵野秀明（あんの ひであき）。アニメ監督。『新世紀エヴァンゲリオン』は1995年放送のテレビアニメで、巨大人型兵器エヴァに乗る少年少女と、彼らの内面を深く描いた作品として社会現象を巻き起こした。

## 解説

主人公・碇シンジが第壱話「使徒、襲来」で、初めてエヴァンゲリオン初号機への搭乗を迫られる場面の独白。恐怖から逃げ出したい気持ちと闘いながら「逃げちゃダメだ」と自らに言い聞かせ、最後に搭乗を受け入れる。困難から目を背けたくなる弱さと、それでも向き合おうとする決意の両方を抱えた、シンジというキャラクターを象徴する言葉として広く知られる。
