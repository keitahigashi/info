# ここから始めましょう。1から。いいえ、ゼロから

> ここから始めましょう。1から。いいえ、ゼロから。

## 著者

長月達平（ながつき たっぺい）。ライトノベル作家。『Re:ゼロから始める異世界生活』は2014年から刊行されている、死に戻りの能力を持つ少年スバルの異世界生活を描く作品。アニメ化もされ高い人気を得た。

## 解説

すべてに打ちのめされ、何もかも失って絶望するナツキ・スバルに、ヒロインのレムが寄り添ってかける言葉（アニメ第18話で特に印象的に描かれる）。今が最低でゼロどころかマイナスに見えても、ここを起点にもう一度始めればいい——過去にとらわれず再出発を促す優しい励ましである。落ち込んだ人を立ち上がらせる言葉として、作品の枠を超えて愛されている。
