# 撃っていいのは、撃たれる覚悟のある奴だけだ

> 撃っていいのは、撃たれる覚悟のある奴だけだ。

## 著者

『コードギアス 反逆のルルーシュ』は2006年放送のサンライズ制作のテレビアニメ。谷口悟朗が監督、大河内一楼がシリーズ構成を務めた。亡国の皇子ルルーシュが「ギアス」の力を得て帝国へ反逆する物語。

## 解説

主人公ルルーシュが第1話で「絶対遵守の力」ギアスを手にし、自分を狙う軍へ向けて放つ台詞。他者を攻撃する（撃つ）以上、自分も同じ報いを受ける覚悟がなければならない——加害には必ず相応の責任が伴うという冷徹な原則を突きつける。革命を志すルルーシュ自身の覚悟を表すと同時に、力を振るう者の倫理を問う言葉として、作品を象徴する名台詞となっている。
