# 本気の失敗には、価値がある

> 本気でやった場合に限るよ。本気の失敗には、価値がある。

## 著者

小山宙哉（こやま ちゅうや）。漫画家。『宇宙兄弟』は2007年から『モーニング』で連載されている、宇宙飛行士を目指す兄弟・南波六太と日々人の挑戦を描いた作品。

## 解説

主人公・南波六太（ムッタ）が、宇宙飛行士選抜試験の過程で語る言葉。失敗そのものを恐れるのではなく、本気で取り組んだ末の失敗にこそ学びと価値が宿る、という前向きな姿勢を示す。中途半端な取り組みの失敗とは区別し、「本気でやった場合に限る」と条件を添えるところに説得力がある。挑戦を後押しする言葉として、ビジネスや教育の場でもよく引用される。
