# あきらめたら、そこで試合終了だよ

> 最後まで…希望を捨てちゃいかん。あきらめたらそこで試合終了だよ。

## 著者

井上雄彦（いのうえ たけひこ）。漫画家。『SLAM DUNK』は1990年から『週刊少年ジャンプ』で連載された、高校バスケットボールを題材にした作品で、日本のバスケ人気を押し上げた金字塔。

## 解説

湘北高校バスケ部監督・安西光義（通称・安西先生）の言葉。三井寿の中学時代の回想で、県大会決勝・残り時間わずかで諦めかけた三井に語りかけた台詞であり、インターハイの山王戦では桜木花道に「あきらめたらそこで試合終了ですよ…？」と問いかける形でも登場する。勝負がつく前に自分から諦めてしまえば、可能性はそこで尽きる——勝敗そのものよりも、最後まで希望を捨てない姿勢の大切さを説く、スポーツ漫画屈指の名言。
