# スプーンは存在しない

> There is no spoon.

## 著者

ウォシャウスキー姉妹（The Wachowskis、ラナ／リリー）。アメリカの映画監督・脚本家姉妹。SF哲学とアクションを融合させた『マトリックス』三部作で知られる。

## 解説

預言者オラクルの待合室で、念力でスプーンを曲げる少年がネオに告げる。「スプーンを曲げようとするな。それは無理だ。ただ真実に気づけ——スプーンなど無い、と。そうすれば曲がるのはスプーンではなく自分自身だと分かる」。ネオはこれを受けて「スプーンは存在しない」と口にする。仮想現実の可変性と、現実とは知覚の産物にすぎないという作品の中心思想を凝縮した名台詞。
