# 名は前兆（名前はその人・物の本質を言い当てる）

> nomen atque omen quantivis iam est preti.

## 著者

ティトゥス・マッキウス・プラウトゥス（前254頃–前184）は共和政ローマ最盛期の喜劇作家。ギリシア新喜劇を翻案し、活気ある口語ラテン語と言葉遊びで大衆に愛された。

## 解説

劇中で奴隷トクシルスが「ルクリス（Lucris ＝利益の意）」という名の少女を売り込む際、その名と縁起（omen）は値千金だと語る言葉遊び。名が運命や本質を暗示するという古来の観念を表す。
