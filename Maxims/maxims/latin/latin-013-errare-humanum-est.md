# 誤るのは人間の常である

> Errare humanum est.

## 著者

ストア派の哲学者・劇作家セネカ（前4頃–後65）。ネロ帝の師として知られ、『道徳書簡集』『怒りについて』など多くの倫理的著作を残した。同趣旨の観念は古代に広く共有されていた。

## 解説

「errare」は「誤る・迷う」（不定詞）、「humanum est」は「人間的なことである」。「人は誰でも過ちを犯すもの」という人間理解を表す。完全形は「errare humanum est, perseverare autem diabolicum（誤るのは人間的だが、誤りに固執するのは悪魔的だ）」とされ、後者は後世（中世の説教やキケロ『ピリッピカ』12.5「cuiusvis hominis est errare」の系譜）で付け加えられたもの。過ちそのものより、過ちを認めず繰り返す態度を戒める格言として用いられる。
