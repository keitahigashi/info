# 平和を欲するなら、戦争に備えよ

> Si vis pacem, para bellum.

## 著者

4世紀末から5世紀のローマの著述家ウェゲティウス。軍事教範『軍事論（De re militari / Epitoma rei militaris）』の著者として知られ、その軍事思想は中世・近世ヨーロッパに長く影響を与えた。

## 解説

「si vis pacem」は「もし平和を欲するなら」、「para bellum」は「戦争を準備せよ」（命令）。平和を保つには軍備を怠ってはならない、という抑止の思想を端的に表す。標語化した「si vis pacem, para bellum」の直接の元は、ウェゲティウスの「したがって平和を望む者は戦争に備えよ（Igitur qui desiderat pacem, praeparet bellum）」とされる。自動拳銃「ルガーP08（パラベラム）」の名称や弾薬「9mmパラベラム」の語源にもなった。
