# 人は概して、自分が望むことを喜んで信じてしまうものだ

> ...quod fere libenter homines id quod volunt credunt.

## 著者

ユリウス・カエサル（前100–前44）は共和政末期ローマの政治家・軍人・文人。自らのガリア遠征を簡潔明快な文体で記録した『ガリア戦記』で知られる。

## 解説

敵の指揮官の心理を読み、偽情報で誘い込む場面に添えられた地の文。人間は自分の願望に沿う情報を無批判に信じやすい、という認知の傾向を鋭く突く。確証バイアスの古典的表現としてしばしば引用される。

## 出典

- カエサル『ガリア戦記（Commentarii de Bello Gallico）』3.18
- 原文は従属節中で副詞「fere」（概して）を伴う。独立格言としては「Libenter homines id quod volunt credunt」と短縮引用されることが多い
