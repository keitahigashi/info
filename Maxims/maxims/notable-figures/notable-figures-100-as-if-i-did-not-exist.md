# 教師にとって最大の成功のしるしは、「子どもたちは今、まるで私が存在しないかのように活動している」と言えるようになることである

> The greatest sign of success for a teacher ... is to be able to say, "The children are now working as if I did not exist."

## 著者

マリア・モンテッソーリ（1870–1952）。イタリアの医師・教育学者。イタリア初の女性医師の一人で、子どもの自発性を尊重する「モンテッソーリ教育法」を創始した。世界的に普及した幼児教育のパイオニア。

## 解説

大人（教師）の理想は、指示・介入する存在から、子どもが自立して集中的に活動できるよう環境を整える「見えざる案内役」へと自らを消していくことにある——というモンテッソーリ教育の核心を端的に表す。
