# これは、すべての人のために

> This is for everyone.

## 著者

ティム・バーナーズ＝リー（1955– ）。World Wide Web（WWW）を発明したイギリスの計算機科学者。Webの基本技術であるHTTP・HTML・URLを考案し、それらを特許化せず無償で公開した。

## 解説

2012年ロンドン夏季オリンピックの開会式で、バーナーズ＝リー本人がその場でツイートし、会場8万人の客席に並んだLEDライトに大きく表示された言葉。Webを特定の企業や国家のものではなく全人類の共有財産とする理念を端的に示し、オープンで公平なインターネットという彼の思想を象徴する。
