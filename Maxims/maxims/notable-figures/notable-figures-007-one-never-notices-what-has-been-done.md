# 成し遂げられたことに気づく者はいない

> One never notices what has been done; one can only see what remains to be done.

## 著者

ポーランド出身の物理学者・化学者マリ・キュリー（マリア・スクウォドフスカ＝キュリー、1867–1934）。放射能の研究で女性初のノーベル賞、かつ二度（物理学賞・化学賞）の受賞者となった。

## 解説

達成に安住せず、常に次の課題へ目を向ける研究者の姿勢を率直に語った言葉。私的な書簡に記されたものであり、飾らない本心がうかがえる。

> 誤帰属に注意: キュリーに帰される「人生に怖れるべきものはない、ただ理解すべきものがあるだけだ」は、後半部分がグレン・シーボーグの言葉と混ざった疑わしい引用である。本句は1894年の兄宛て書簡という出典が特定でき、娘エーヴ・キュリーの伝記に収録されているため採用した。
