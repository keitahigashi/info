# 全ての混沌には調和が、無秩序の中には秘められた秩序がある

> In all chaos there is a cosmos, in all disorder a secret order.

## 著者

スイスの精神医学者カール・グスタフ・ユング（1875–1961）。分析心理学の創始者。

## 解説

一見、混沌や無秩序に見えるものの奥にも、解釈可能な秩序・法則が潜んでいる——というユングの洞察。アニマ（男性の無意識に宿る女性的元型）の捉えがたい性質を論じる文脈で語られた言葉で、原文は「すべて作用するものはその対立物に根ざしている」と続く。無意識やこころの一見ばらばらな現象の背後に構造を見ようとするユングの姿勢を示す。
