# 我々が見通せるのはほんの少し先までだが、そこには為すべきことが山ほどあるのが見える

> We can only see a short distance ahead, but we can see plenty there that needs to be done.

## 著者

アラン・チューリング（1912–1954）。英国の数学者・計算機科学者。チューリングマシンの概念で計算理論の基礎を築き、第二次大戦中はドイツの暗号機エニグマの解読に貢献した。機械の知能を判定する「チューリング・テスト」の提唱者としても知られる。

## 解説

機械が思考しうるかを論じた論文全体を締めくくる最終文。見通せる範囲は限られていても、目の前には取り組むべき課題が山ほどあると述べ、人工知能研究の将来への楽観と、いま為すべき実行を促す。壮大な未知に対して謙虚さと行動主義を両立させる研究者の姿勢を示す言葉として広く引用される。
