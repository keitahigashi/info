# 住宅は住むための機械である

> Une maison est une machine à habiter.
> （A house is a machine for living in.）

## 著者

ル・コルビュジエ（本名 Charles-Édouard Jeanneret、1887–1965）。スイス生まれ、フランスで活動した近代建築の巨匠。機能主義・モダニズム建築の理論的指導者で、サヴォア邸やユニテ・ダビタシオンで知られる。

## 解説

著書『建築をめざして』の中心命題。住宅を自動車や船と同じく「機能から形が導かれる工業製品」として捉えるべきだという近代建築宣言の核心であり、装飾過多な歴史主義との決別を示す。20世紀の建築・デザイン思想に決定的な影響を与えた。
