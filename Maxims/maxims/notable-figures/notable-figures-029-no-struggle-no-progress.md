# 闘いがなければ、進歩はない

> If there is no struggle, there is no progress. ... Power concedes nothing without a demand. It never did and it never will.

## 著者

アメリカの奴隷制廃止論者・演説家・著述家フレデリック・ダグラス（1818–1895）。自らも奴隷として生まれ脱走した経歴を持ち、19世紀アメリカ最大の黒人指導者の一人として奴隷解放と公民権の拡大に尽力した。

## 解説

1857年、西インド諸島での奴隷解放23周年を記念してニューヨーク州カナンデーグアで行った演説の一節。彼の数ある演説の中でも最も引用される言葉で、続く「権力は要求なしには何も譲り渡さない（Power concedes nothing without a demand）」とともに、自由や権利は受け身では得られず、道徳的・物理的な闘争を通じてのみ勝ち取られる、という歴史認識を鋭く示している。
