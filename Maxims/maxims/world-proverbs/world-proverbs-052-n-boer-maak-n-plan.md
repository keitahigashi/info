# 農夫は必ず策を講じる

> 'n Boer maak 'n plan.

## 著者

作者不詳（アフリカーンス語のことわざ）。

## 解説

直訳は「農夫（boer）は計画（plan）を立てる」です。資源の乏しい厳しい環境で土地を耕したボーア人農民の歴史に由来し、どんなに困難な状況でも機転と創意工夫で必ず解決策を見つけ出す、という不屈の精神を表します。

アフリカーンス語でもっとも頻繁に使われる慣用句のひとつで、南アフリカの自助・たくましさの象徴とされます。日本語の「窮すれば通ず」や「火事場の馬鹿力」に近いニュアンスを持ちます。
