## 収集記事

### 645. Natural Funativity
- **URL**: https://www.gamedeveloper.com/design/natural-funativity
- **ソース**: Game Developer (Noah Falstein, 2004)
- **反映先**: 00-foundations
- **サマリー**: 進化心理学に基づくゲームの楽しさ3分類フレームワーク。身体的（狩猟採集スキル練習）・社会的（部族的結合・言語）・精神的（パターン認識）の3種に楽しさを整理し、「洗練された砂糖症候群 (RSS)」として現代ゲームの魅力を説明。新作ゲームの評価を主観から客観的フレームワークへ移行させる実践的手法。

### 646. Gamification: Thoughts on Definition and Design
- **URL**: https://www.gamedeveloper.com/design/gamification-thoughts-on-definition-and-design
- **ソース**: Game Developer (Zachary Fitz-Walter, 2012)
- **反映先**: 00-foundations
- **サマリー**: ゲーミフィケーションの定義（非ゲームコンテキストへのゲームデザイン要素の活用）を整理し、バッジ・ポイント・リーダーボードのみに依存した外発的動機設計の限界を批判。自己決定理論の3欲求（自律性・有能性・関連性）に基づく内発的動機設計の重要性を論じる。

### 647. Procedural Generation: Digging Deeper
- **URL**: https://www.gamedeveloper.com/design/procedural-generation-digging-deeper
- **ソース**: Game Developer (Luis Diaz, 2015)
- **反映先**: 07-technical-systems
- **サマリー**: プロシージャル生成 (procedural generation) をマップ・ダンジョン生成から**ナラティブ生成**へ拡張する可能性を論じる。ルールベースのナラティブ生成により、少数のルールから創発的な物語が生まれるConway's Game of Life型のアプローチを提示。Shadow of Mordor のNemesisシステムを革新的実装例として高く評価。

### 648. Design, Doubt, & the Scientific Method
- **URL**: https://www.gamedeveloper.com/design/design-doubt-the-scientific-method
- **ソース**: Game Developer (Tim Conkling, 2015)
- **反映先**: 08-production
- **サマリー**: ゲームデザインの反復プロセスに科学的方法（仮説→実験→データ収集→反復）を適用する手法。設計課題を「失敗」ではなく「仮説が正しくなかったという発見」と捉え直すことで、開発中の不安と疑念サイクルを克服する。最小プロトタイプ+プレイテスターで前進できるという実践的メッセージ。

### 649. Design 101: Playtesting
- **URL**: https://www.gamedeveloper.com/design/design-101-playtesting
- **ソース**: Game Developer (Dan Felder, 2015)
- **反映先**: 08-production
- **サマリー**: 5段階のプレイテスト手法（コンセプト検証・散弾的テスト・体験テスト・負荷テスト・アクセシビリティテスト）を体系化。「問題探しより成功パターンを複製するブライトスポット戦略 (bright spot strategy)」を中心的思想とし、プレイペア観察法・Good/Bad/Meh記録など実践的テクニックを多数紹介。
