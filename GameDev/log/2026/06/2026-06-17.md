### 620. Mechanics - Depth Knows Best
- **URL**: https://www.gamedeveloper.com/game-platforms/mechanics-depth-knows-best
- **ソース**: Game Developer (Max Pears, Ubisoft, 2017)
- **反映先**: 01-mechanics
- **サマリー**: 「機能の数」より「1つのメカニクスの使い方の多様性」を優先するという設計哲学。マリオのジャンプやアンチャーテッド4のグラップリングフックを事例に、単一メカニクスが複数の文脈で機能することが深い体験を生む仕組みを解説。TF2のロケットジャンプのようなメカニクス間の相互作用が意図しない深みを生み出すことも示す。

### 621. What is depth and how do you add it to your game?
- **URL**: https://www.gamedeveloper.com/game-platforms/what-is-depth-and-how-do-you-add-it-to-your-game-
- **ソース**: Game Developer (Alex Kubodera, 2022)
- **反映先**: 01-mechanics
- **サマリー**: 深さ (depth) と複雑さ (complexity) を対比定義し、深さは「可能性空間」であると規定。セットアップ・インベストメント・ビルド・マイクロの4カテゴリで深さを分類し、各カテゴリへの設計アプローチを解説。メタゲームの形成とクリエイティビティ制限のトレードオフについても言及。

### 622. Key Ingredients For Compelling Video Game Characters
- **URL**: https://www.gamedeveloper.com/design/key-ingredients-for-compelling-video-game-characters
- **ソース**: Game Developer (Natalie Mikkelson, 2020)
- **反映先**: 03-narrative
- **サマリー**: 魅力的なゲームキャラクター構築の7要素（動機・プレイヤー関係性・3次元的キャラクター・キャラクターアーク・強い敵対者・予期しない感情・Show Don't Tell）を包括的に解説。理想的ファンタジーと白紙状態の2アプローチ、25項目のキャラクター構築チェックリストを提供。

### 623. Structures of Narrative
- **URL**: https://www.gamedeveloper.com/design/structures-of-narrative
- **ソース**: Game Developer (Gregory Pellechi, 2018)
- **反映先**: 03-narrative
- **サマリー**: ゲームにおける4つのナラティブ構造（枠物語・エピストラリー・キアスティック・リング）を定義し、各構造をゲーム事例で解説。Halo CE の上下対称構造、What Remains of Edith Finch の文書型構造など具体例が豊富。4構造は組み合わせ可能で、レベルデザインとの統合についても示唆。

### 624. Min/Max Mastery in Game Design
- **URL**: https://www.gamedeveloper.com/design/min-max-mastery-in-game-design
- **ソース**: Game Developer (Josh Bycer, 2014)
- **反映先**: 01-mechanics
- **サマリー**: ミニマックス (min/max) プレイの否定的・肯定的両視点を整理し、デザイナーが取り得る3つの対応策（予測不可能性の確保・数値隠蔽・コンテンツ設計）を提示。Disgaeaのポストゲームコンテンツ設計とXCOMの最適化問題を事例に、ゲームタイプによって対応戦略が異なることを示す。
