### 615. Paper Playtesting and Roleplaying as an NPC to Design "The Irregular"
- **URL**: https://www.gamedeveloper.com/design/paper-playtesting-and-roleplaying-as-an-npc-to-design-quot-the-irregular-quot-
- **ソース**: Game Developer (Lanie Dixon, Octothorpe, 2017)
- **反映先**: 08-production
- **サマリー**: VR謎解きゲーム「The Irregular」の開発で実施したペーパープロトタイプ×NPCロールプレイ手法の事例。開発者がNPC役を演じながらプレイヤーの思考プロセスを観察し、デジタル実装前にクエスト5件修正・1件追加を実現。タイムスタンプ付き詳細ログとプレイヤーの自発的メモ取り行動からゲーム内ノート機能を設計した実践的事例。

### 616. 6 Steps to a Successful Playtesting Process for an Indie Developer
- **URL**: https://www.gamedeveloper.com/programming/6-steps-to-a-successful-playtesting-process-for-an-indie-developer
- **ソース**: Game Developer (Maxim Kiselev, HighFleet共同制作者, 2024)
- **反映先**: 08-production
- **サマリー**: インディー開発者向けの6段階プレイテストプロセス（準備→デモ作成→チャネル構築→テスト実施→フィードバック識別→統合）。1セッション1〜2要素に絞り込む原則、itch.io/Steam Playtest/Discordの段階別プラットフォーム選定、反復的フィードバックテーマへの注目、コミュニティとの透明なコミュニケーションがロイヤルプレイヤー形成につながることを説く実践ガイド。

### 617. How I Teach Game Design: The Game Design Process
- **URL**: https://www.gamedeveloper.com/design/how-i-teach-game-design-lesson-1-the-game-design-process-
- **ソース**: Game Developer (Eric Zimmerman, NYU, 2013)
- **反映先**: 00-foundations
- **サマリー**: Eric Zimmermanによるゲームデザイン教育論。反復的設計プロセス（プロトタイプ→テスト→分析の繰り返し）の5原則（アイデアは安い・即座にプロトタイプ開始・失敗を受け入れる・批判的視点を保つ・独創性に応じて反復量を増やす）を解説。Tic-Tac-Toe改造演習（3つ並べたら負け）で単純なルール変更が体験を根本的に変えることを実証する教育実践。

### 618. Strange Love: Game Theory vs. Game Design
- **URL**: https://www.gdcvault.com/play/1019141/Strange-Love-Game-Theory-vs
- **ソース**: GDC Vault (Frank Lantz, NYU Game Center, 2013)
- **反映先**: 00-foundations
- **サマリー**: ゲーム理論（数学的決定分析）とゲームデザイン（創造的実践）の「奇妙な関係」をGDCで講演。ナッシュ均衡に達したゲームは退屈になる（= 最適解が存在するゲームはデザイン的失敗の可能性）、合理的プレイヤー前提の限界、ゲーム理論は分析ツールであって設計処方箋ではないという示唆を歴史的・哲学的視点から展開。

### 619. Level Design Workshop: The Illusion of Choice
- **URL**: https://www.gdcvault.com/play/1023142/Level-Design-Workshop-The-Illusion
- **ソース**: GDC Vault (Jim Brown, Epic Games, 2016)
- **反映先**: 02-level-design
- **サマリー**: プレイヤーが「自由に選んでいる」と感じながら実際にはデザイナーが意図した行動を選択する「選択の幻想 (illusion of choice)」の設計論。選択過負荷・錨効果・損失回避の心理学的メカニズムを解説し、視覚的誘導（照明・コントラスト）・空間的誘導（ファネル設計）・報酬配置による具体的な実装手法を示す。意図的設計と無意識の設計ミスの対比表付き。
