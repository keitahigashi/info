### 605. Has Tides of Tomorrow cracked the narrative multiplayer formula?
- **URL**: https://www.gamedeveloper.com/design/has-tides-of-tomorrow-cracked-the-narrative-multiplayer-formula-
- **ソース**: Game Developer (Bryant Francis, 2025)
- **反映先**: 03-narrative
- **サマリー**: Road 96開発元Digixartの新作における非同期ナラティブマルチプレイシステムの設計解説。プレイヤーの選択結果が他プレイヤーのゲームセッションに波及する仕組みを、Death Stranding/Elden Ringの非同期マルチプレイから発展させた形で実現。プリプロダクションで「プロトテキスト」検証を行いスコープを厳選する手法を提唱。

### 606. Revisiting the MDA Framework
- **URL**: https://www.gamedeveloper.com/design/revisiting-the-mda-framework
- **ソース**: Game Developer (Luiz Claudio Silveira Duarte, 2015)
- **反映先**: 00-foundations
- **サマリー**: MDAフレームワークの批判的再考。デジタルゲームを前提とした線形モデル（デザイナー: M→D→A、プレイヤー: A→D→M）がボードゲームには不整合であると指摘。ボードゲームではプレイヤーが事前にルール（メカニクス）を学ぶため逆順プロセスが発生する。業界の「デジタルゲーム盲目性」を克服し、動的・非線形なMDA適用を提唱。

### 607. The New Dramaturgy Pt 1: Introduction to Gamic Dramaturgy
- **URL**: https://www.gamedeveloper.com/design/the-new-dramaturgy-pt-1-introduction-to-gamic-dramaturgy
- **ソース**: Game Developer (Evan Hill, 2013)
- **反映先**: 00-foundations
- **サマリー**: 演劇のドラマトゥルギー理論をゲームに応用した分析フレームワーク。「経験層（Idiom・Suspense・Catharsis）」と「構造層（Design Goal・Dramatic Substance・Pacing）」の二層モデルでゲームの感情設計を体系化。シネマティック vs. ゲーム性の対立を解消するブリッジツールとして機能し、MDAが扱わない感情的・劇的側面を補完する。

### 608. Hop 'Til You Drop: Finding the fun with rapid iteration
- **URL**: https://www.gamedeveloper.com/design/hop-til-you-drop-finding-the-fun-with-rapid-iteration
- **ソース**: Game Developer (Jordan Maynard / Manticore, 2020)
- **反映先**: 08-production
- **サマリー**: 2人チームが2週間で100回以上（1日平均7ビルド）の反復を行いマルチプレイヤーパズルゲームを完成させた実践報告。既存アセット活用で初期実装を2日に短縮、ストループ効果導入・勝利条件変更・コインシステム追加等のデータ主導の設計変更を実施。「コンパイル時間×10,000=完成予定時間」という格言が示す通り、反復速度の最大化が生産性の核心。

### 609. Storytelling with Interface: The Narrative Design of User Interface in Video Games
- **URL**: https://www.gamedeveloper.com/design/storytelling-with-interface-the-narrative-design-of-user-interface-in-video-games
- **ソース**: Game Developer (Sisi Yuan, 2021)
- **反映先**: 04-ui-ux
- **サマリー**: ゲームUIを物語語りのツールとして設計するフレームワーク。物語的UI要素をキャラクター・オブジェクト・雰囲気 (atmosphere)・イベントの4構成要素に分類し、世界観との整合性・感情設計の一致・不可能な三角形のバランス・段階的設計プロセスの4原則で実装する。「コンテキスチュアルデザインだけが解ではない」というメッセージで、機能性と物語性の統合を訴える。
