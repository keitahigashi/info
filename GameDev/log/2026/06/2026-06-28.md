### 670. Performing a Heuristic Analysis of Your Game
- **URL**: https://www.gamedeveloper.com/design/performing-a-heuristic-analysis-of-your-game
- **ソース**: Game Developer (Suchaaver Chahal, 2015)
- **反映先**: 04-ui-ux
- **サマリー**: ニールセンの10の使いやすさ原則をゲームUIに適用するヒューリスティック評価 (heuristic evaluation) 手法を体系化。大規模ユーザーテスト前の低コスト評価として有効で、The Sharpest Blade（フォント・ナビゲーション問題）とHearthstone（ダイアログ・フィードバック・ボード配置の好例）のケーススタディを通じて、評価→重大度付け→修正計画の実施手順を解説する。

### 671. Deep Dive: Multiple menu systems & UI design in APICO
- **URL**: https://www.gamedeveloper.com/design/deep-dive-multiple-menu-systems-ui-design-in-i-apico-i-
- **ソース**: Game Developer (Ell R, 2022)
- **反映先**: 04-ui-ux
- **サマリー**: 養蜂シム APICO が採用した無制限マルチウィンドウメニューシステムの設計判断を解説。OSウィンドウパターン（タイトルバードラッグ・×ボタン）を活用して学習コストを排除し、シフトクリック転送用ターゲットメニュー・ピン留め機能で複数ウィンドウ開放時の操作効率を確保。ターン制・ファーミング・クラフト系ゲームへの汎用パターンとして応用可能。

### 672. Game HUD Design Techniques: Enhance UI/UX and Visual Hierarchy for Indie Game Developers
- **URL**: https://www.indiedevguide.com/articles/game-hud-design-techniques-ui-ux-indie-devs/
- **ソース**: Indie Dev Guide (Ravi Singh, 2025)
- **反映先**: 04-ui-ux
- **サマリー**: HUD設計の基礎から2025年トレンドまでを網羅する実践ガイド。重要度別の視覚的差別化（サイズ・色・配置）、スクリーンエッジの慣例的活用（左上にHP・右下に弾薬等）、文脈的HUD・アダプティブ透明度などの現代的手法、および視覚・聴覚・触覚のマルチチャンネルフィードバック統合を解説。反復的改善プロセス（プレイテスト→ヒートマップ分析→修正）も含む。

### 673. UX and UI in game design: exploring HUD, inventory, and menus
- **URL**: https://medium.com/@brdelfino.work/ux-and-ui-in-game-design-exploring-hud-inventory-and-menus-5d8c189deb65
- **ソース**: Medium (Bruna Delfino, 2025)
- **反映先**: 04-ui-ux
- **サマリー**: ゲームUIの3主要領域（HUD・インベントリ・メニュー）を認知負荷最適化と視覚的階層の観点から横断的に解説。インベントリはグリッド型・リスト型・カテゴリタブ型・タグフィルタ型の4パターンを比較し、メニューはPC・コンソール・モバイルのプラットフォーム別入力特性に応じた設計の違いを整理。「3クリックルール」や一貫した戻る動作など普遍的なナビゲーション原則も提示。
