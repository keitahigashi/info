### 650. The User Interface Continuum: A Study Of Player Preference
- **URL**: https://www.gamedeveloper.com/design/the-user-interface-continuum-a-study-of-player-preference
- **ソース**: Game Developer (Kristine Jørgensen, Bergen University, 2011)
- **反映先**: 04-ui-ux
- **サマリー**: 22名のプレイヤーを対象にHUDへの受容性を定性研究し、フィクショナリスト・システマティスト・相対主義者の3アーキタイプを発見。多数派の相対主義者はHUDを慣習として受け入れており、「HUD=没入破壊」という懸念は過剰であると実証。インタラクティビティ自体が没入を駆動するため、ゲームには映画的な「第四の壁」が存在しないという核心的知見。

### 651. What Does Accessibility Mean in Game Design
- **URL**: https://www.gamedeveloper.com/design/what-does-accessibility-mean-in-game-design
- **ソース**: Game Developer (Josh Bycer, 2021)
- **反映先**: 04-ui-ux
- **サマリー**: 「アクセシビリティ（知覚・操作支援機能）」と「アプローチビリティ（UI/UXとメカニクスの設計改善）」を明確に区別し、それぞれ異なるアプローチが必要であることを論じる。「良いゲームデザインはアクセシブルなゲームデザイン」という中心原則のもと、困難の「理由」の分析と開発初期段階からの組み込みを推奨。

### 652. 5 Misconceptions about UX (User Experience) in Video Games
- **URL**: https://www.gamedeveloper.com/business/5-misconceptions-about-ux-user-experience-in-video-games
- **ソース**: Game Developer (Celia Hodent, 2015/2019更新)
- **反映先**: 04-ui-ux
- **サマリー**: 元Epic Games UX DirectorでフォートナイトのUXを統括した神経科学者Celia Hodentが、「UXは芸術を歪める」「常識に過ぎない」「単なる意見」「コストがかかる」「独立した工程」という5つの誤解を論破。認知バイアス（呪いの知識・確認バイアス等）とワーキングメモリ限界（約4項目）の科学的根拠をもとに、UXを医学的推奨と同等の専門的判断として位置づける。

### 653. UX Walkthrough: Anatomy of a Usability Test in Video Games (Part 1)
- **URL**: https://www.gamedeveloper.com/production/ux-walkthrough-anatomy-of-a-usability-test-in-video-games-part-1-
- **ソース**: Game Developer (Om Tandon, Steven Gaston, 2015)
- **反映先**: 04-ui-ux
- **サマリー**: 社内プレイテストが抱える7つの構造的欠陥（開発環境実施・社員担当・グループ形式・記録なしなど）と、それを生む5種の心理的バイアス（実験者・サンプリング・応答・手続き・確認バイアス）を体系化。「コンセプト段階の発見コスト×1 → リリース後×1000」という発見コストの法則を示し、早期の客観的テスト設計の重要性を訴える。

### 654. Should we abandon the HUD? Examining its impact on player flow
- **URL**: https://www.gamedeveloper.com/design/should-we-abandon-the-hud-examining-its-impact-on-player-flow
- **ソース**: Game Developer (Reinard Baertsoen, 2024)
- **反映先**: 04-ui-ux
- **サマリー**: HUD廃止がフロー状態に与える二重効果を分析。視覚ノイズ排除によるフロー強化の可能性がある一方、制御感の喪失（フロー達成の必須条件）を招くリスクも存在する。フロー状態中は自己反省能力が低下し記憶に残りにくいという副作用も指摘。HUD廃止の是非に一律の答えはなく、ゲームの性質に応じた実験と試行錯誤を推奨。
