# CLAUDE.md

## このリポジトリについて

ゲーム開発のナレッジベース。エンジン非依存のゲームデザイン・技術記事を収集・整理し、トピック別に蓄積する。

## 絶対ルール（必ず守ること）

- 収集済みURLへの重複追加は禁止（`REFERENCES.md` で重複チェック）
- 1回の実行で最大5記事まで
- ナレッジベースファイル（`00-09`）への追記は所定フォーマットに従う
- ログは必ず `log/{YYYY-MM-DD}.md` に作成する
- 特定エンジン固有の記事は、汎用パターンとして抽象化可能な場合のみ採用する
- 英語の専門用語は原語を併記する（例: "ジュース (juice)"）

## Skillsの使い方

タスク開始前に必ず該当するSKILL.mdを読んでから作業すること。

| やること | 参照するSKILL.md | トリガーワード |
|----------|-----------------|---------------|
| 記事収集・自動収集ワークフロー | `.claude/skills/article-collector/SKILL.md` | 「記事収集して」「ゲーム開発の記事探して」「集積して」「続けて」 |

## ファイル構成（参照用）

| パス | 役割 |
|------|------|
| `00-09` のナンバリングファイル | ナレッジベース本体 |
| `README.md` | 目次・システム概要 |
| `REFERENCES.md` | 収集済み記事テーブル（重複チェックの単一ソース） |
| `references/` | 記事詳細ファイル（Git管理） |
| `log/` | デイリー収集ログ |
| `.claude/skills/article-collector/` | 記事収集ワークフロー定義 |
