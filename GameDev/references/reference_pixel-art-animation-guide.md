## キャラクター設計の4原則

| 原則 | 内容 |
|------|------|
| **シンプルな形状** | 基本的な幾何学形を使用 |
| **明確なシルエット** | 背景なしでも認識可能 |
| **限定的な色彩** | スプライトあたり**2〜4色**の厳選 |
| **表現力のある特徴** | 目、口、ボディランゲージへの注力 |

## アニメーション制作4ステップ

1. **フレーム計画** — キーポーズのスケッチ
2. **ベーススプライト作成** — 中立姿勢の設計
3. **動きの追加** — フレーム間の段階的変化
4. **細部の洗練** — 髪や衣服の二次アニメーション (secondary animation)

## フレームレート設定

| スタイル | FPS | 印象 |
|---------|-----|------|
| レトロ | 8〜12 FPS | チョッピーで懐かしい |
| 標準 | 12〜24 FPS | 滑らかで自然 |

## 重要な技術的留意点

- **一貫性の維持**: フレーム間のピクセル位置のズレ防止
- **色彩パレット統一**: 全フレーム共通の色使用
- **タイミングの自然性**: 流暢で自然な動きの実現

## 回避すべき一般的誤り

- 不均一な輪郭線 (jaggies)
- 過度な詳細（低解像度に詰め込みすぎ）
- 貧弱なタイミング（フレーム間の速度不整合）
- フレームの欠落（中間ポーズ不足）

## サブピクセルアニメーション

限られたピクセル数で滑らかな動きを表現する技法:
- 「中間ピクセル」を追加してトランジションを滑らかに
- より微妙な動きにはより多くの色が必要
- 低解像度スプライトでの深度表現に効果的
