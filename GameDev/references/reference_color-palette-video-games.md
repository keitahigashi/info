## 色彩心理の基礎

| 色 | 心理的連想 |
|----|-----------|
| **赤 (Red)** | 情熱、暖かさ、怒り、愛 |
| **橙 (Orange)** | 喜び、熱意、新鮮さ |
| **黄 (Yellow)** | 幸福、臆病さ |
| **緑 (Green)** | 自然、嫉妬、病気、貪欲 |
| **青 (Blue)** | 落ち着き、冷たさ、企業的、男性的 |
| **紫 (Purple)** | 王族、贅沢、品質 |
| **黒 (Black)** | 神秘、悪、悲嘆 |
| **白 (White)** | 誠実、純粋、神聖、哀悼 |

## 5つのカラースキーム

### 1. 暖色トーン (Warm Tones)
Firewatch のような秋のパレット。快適さまたは攻撃性を文脈に応じて伝達。安全感または戦闘準備を示唆。

### 2. 寒色トーン (Cool Tones)
SFタイトルが明るく冷たいブルーを使用し、無感情な未来を示唆。暖色アクセントとの組み合わせで視覚的関心を創出。

### 3. 西部劇/ディストピア (Western/Dystopian)
茶、灰、生気のないパレットが終末的ナラティブを支える。「明るい色が劇的に目立ち、重要情報をハイライト」。

### 4. ノワール/サイバーパンク (Noir/Cyberpunk)
深い影と明るいハイライトのコントラスト。Far Cry: Blood Dragon のようなネオンライティングがプレイヤーを引きつけつつ危険を警告。

### 5. モノクロ/限定色 (Monochrome/Limited)
制限的な色使いでドラマチックなミニマリズムを達成。補完テーマとの組み合わせで特に印象的に。

## 実践的応用

### ウェイファインディング (Wayfinding)
色のコントラストでプレイヤーの注意を自然に誘導。
- キーオブジェクト以外を脱彩度化して集中を強化
- ノワール照明の心理学で明るさへの移動を促進

### 安全性インジケーター
色がゲームプレイ状態の変化を伝達。
- 赤い画面の警告がダメージを示す
- Splinter Cell: Conviction の脱彩度化がステルス安全を信号

### ナラティブ展開
色の不在 vs 彩度がストーリーの意味を伝達。NieR: Automata の日常的な基地エリア vs 鮮やかな地球探索のコントラスト。

## 設計プロセスのヒント

- デザインソフトのレイヤーブレンドモードによる実験で補色関係を有機的に発見
- 意図的な目的（ウェイファインディング/ムード設定/ストーリーテリング）から始めてパレット選択を導く
