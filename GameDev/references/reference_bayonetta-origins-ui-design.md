## 基本情報

UIアーティスト近藤知之による解説。完成1年前に参加し、自由制作期間は約6ヶ月。既定コンセプト「絵本らしいUI」に基づき、共通デザインルールを再策定して統一感を復元。

## コア設計哲学

**基本方針:** 「ゲーム内すべてが絵本内で起きている」— 各画面が「絵本の1ページに感じられる」デザインを追求。西洋風おしゃれな絵本の雰囲気が目標。

**視覚的原則:**
- イラスト・文字を主役に配置
- 装飾は最小限
- 余白を区切り要素として活用

## 色彩体系

内容に応じた配色ルール:

| 基調色 | 用途 |
|--------|------|
| **白紙** | 共通要素 |
| **緑** | 森シーン関連 |
| **セピア調** | シナリオ系 |
| **水色** | ティルナノーグ関連 |

## イラスト配置戦略

| 画面タイプ | 手法 |
|-----------|------|
| メニュー画面 | セレッサ・チェシャの描き下ろしイラスト |
| イラストなし画面 | 物語シーン関連のシルエット（ヒトダマ、探索風景） |
| 深度表現 | 「システム」→「設定」でシルエットの前後感演出 |
| 登場人物 | ランダム形ボタン、森イラスト囲み |
| ワールドマップ | ランドマーク描写。未開放時は生き物・ヒトダマ配置 |
| ストーリー連動 | 進行で画面イラスト変化 |

## プレイ時HUD設計

- セレッサとチェシャで**色・配置を明確に区別**
- 体力強化: アート班制作アイテム
- 魔力強化: UI班デザインのゲージ（対比表現）
- 属性関連: チェシャデザインに統一
- 視認性: コントラスト維持しつつアイコン化

## アニメーション哲学: CG感の排除

「CG感」を排除する**2D的表現**を一貫して採用:

| 手法 | 説明 |
|------|------|
| **ページめくり** | 画面切り替えトランジション |
| **じわっと描かれる出現** | 段階的描画エフェクト |
| **パラパラ漫画風** | アイコンアニメーション |
| **2D的強調** | CG発光を回避した強調表現 |

## 汎用化ポイント

1. **世界観ファーストUI**: ゲーム世界のメタファー（絵本の1ページ）がUI全体の設計基盤となる汎用的アプローチ
2. **コンテンツ別カラーコーディング**: 画面内容に応じた基調色の切り替えが情報構造を暗黙的に伝達
3. **アニメーション言語の統一**: UI遷移・エフェクトのスタイルを統一することで全画面で一貫した触感を実現
4. **HUDのキャラクター分離**: 複数操作キャラクターのHUD要素を色・位置で明確に分離する設計パターン
