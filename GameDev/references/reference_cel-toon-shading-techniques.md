## コア概念

セル/トゥーンシェーディングはスタイライズドな2D外観を3D深度を維持しつつ生む非フォトリアリスティックレンダリング技法。物理ベースレンダリングと対照的に、リアリズムより芸術的表現を優先。

## 量子化ライティング (Quantized Lighting)

離散的な色バンドを使用する基本的アプローチ。

```
shade = floor(shade * 3.) / 3.
```

光の角度に基づいて3-4つのフラットな色領域を生成し、滑らかなグラデーションを明確なトーン段階に置き換え。サーフェス法線と光方向のドット積を計算し、floor関数で量子化。

## アウトライン描画手法

### Inverted Hull（反転ハル）

- メッシュを複製しわずかに拡大
- **背面のみ描画**してクリーンなアウトラインを生成
- 実装がシンプル

### ポストプロセス

- レンダリング後にエッジ検出を適用
- ノイズやハッチングパターンを含む、より洗練された輪郭エフェクト

## 芸術的応用

- **2D美学**: フラットな色でコミックブック/イラスト風の外観
- **深度の明確性**: アウトラインが重なる形状間の混乱を防止
- **バンドパターニング**: シェードバンド内にハッチングやテクスチャを戦略的に使用
- **白黒バリアント**: エッジ定義を強調した高コントラストレンダリング

## ゲーム事例

Zelda、Hades、Dragon Ball、Genshin Impact — 各々が独自のビジュアルアイデンティティのためにこの技法を活用しつつ、ゲームプレイの明確性を維持。
