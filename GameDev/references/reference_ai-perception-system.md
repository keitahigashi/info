## 知覚チャネル（4種）

| チャネル | 説明 |
|---------|------|
| **Vision（視覚）** | 距離チェック・視野角錐・視線検証（レイキャスト） |
| **Hearing（聴覚）** | 足音・銃声・爆発・環境音の検出 |
| **Environmental（環境）** | 煙・照明変化・障害物による視界影響 |
| **Social（社会的）** | チームメイトの役割認識・グループ協調 |

## 段階的認識レベル vs 二値検出

知覚は「二値的なYes/Noではなく、認識レベルについて」動作。

「ガードはプレイヤーの位置を即座に知らない。"ほぼ見えている"状態で躊躇し、それから反応する」

→ 疑念構築メカニクスを通じた**テンション創出**。NPCは調査してからアクションにコミット。

## パフォーマンス最適化: 階層的検証

コストの高い計算を防ぐ段階的アプローチ:

```
1. Distance filtering（距離フィルタ）  ← 最も安価
   ↓ 範囲内なら
2. FOV angle check（視野角チェック）   ← 中程度
   ↓ 角度内なら
3. Raycast（レイキャスト）            ← 最も高価
```

- レイキャストは**必要な場合のみ**実行
- 複数トレースで部分的可視性を測定
- 検出状態間の自然なトランジションを維持

## 行動システムとの統合

知覚が「AIが何を知っているか」を定義 → アクション選択とパスファインディングの判断を誘導。

## 社会的協調

役割ベースの認識が明示的指示なしに**創発的分隊行動**を実現:
- NPCが利用可能な戦術ポジションを知覚
- 役割を予約し重複を回避
- 中央集権的指示不要
