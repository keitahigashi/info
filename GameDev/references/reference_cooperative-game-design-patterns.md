## 3つのマルチプレイヤーモードの区別

| モード | 定義 |
|--------|------|
| Competitive（競争） | プレイヤーが直接対立する目標・戦略を追求 |
| Co-operative（協力） | 利害が「完全に対立も完全に一致もしない」、個人報酬あり |
| Collaborative（協調） | チームが共有結果に向けて活動。チーム勝利=全員勝利、チーム敗北=全員敗北 |

設計上の核心課題: プレイヤーが持ち込む**競争性**をナビゲートし、チームワークを強制しつつ利己的行動を抑制するメカニクスを実装すること。

## 6コアデザインパターン（Rocha et al., 2008）

1. **Complementarity（相補性）**: 固有のキャラクターロールや情報が他プレイヤーの貢献を強化
2. **Synergies between abilities（能力間シナジー）**: 異なる能力が他キャラのアクションを支援・修正
3. **Exclusive team abilities（チーム専用能力）**: チームメイトにのみ使用可能なアクション（例: TF2のメディック回復）
4. **Shared goals（共有目標）**: 統一チームワークを要求する非排他的目標
5. **Goal synergies（目標シナジー）**: 相互依存を促進するタスク依存関係
6. **Team-specific rules（チーム固有ルール）**: フレンドリーファイア防止等の協力促進メカニクス

## 7追加デザインパターン（Seif El-Nasr et al., 2010）

1. **Camera settings（カメラ設定）**: 分割画面 / シングルフォーカス / 全キャラ表示
2. **Shared object interaction（共有オブジェクトインタラクション）**: 複数プレイヤーが同時にゲームオブジェクトを操作
3. **Shared puzzles（共有パズル）**: 協力で解く障害・チャレンジ
4. **Shared characters（共有キャラクター）**: 異なるプレイヤーが順番に制御するNPC
5. **Anti-lone-wolf NPCs（一匹狼対策NPC）**: 孤立・非協力的プレイヤーをターゲットにする敵
6. **Vocalisation（自動発声）**: 戦略的決定・危険をチームメイトに警告する自動キャラ発声
7. **Limited resources（リソース制限）**: 共有・交換を促進する希少性

## チャレンジ分類

### Pure Challenges（純粋チャレンジ）
- 身体的努力・協調・反射・空間認識を要求
- 個人プレイヤーがソロ完了**不可能**に設計
- 同期された動きと戦略的認識が必要

### Applied Challenges（応用チャレンジ）
- **レース条件**: 協調完了を強制するタイム制イベント
- **探索**: 同時スイッチ操作・協力トラップ回避
- **経済**: トレードと配分議論を要求するリソース管理

## 研究知見（25セッション・各3時間のビデオ分析）

最も協力行動に影響を与えたメカニクス:

| パターン | 促進する行動 |
|---------|-------------|
| **Complementarity** | 共有興奮、戦略的コミュニケーション、役割分化 |
| **Shared goals/puzzles** | チームワーク議論の動機付け |
| **Object interaction** | 笑い、相互支援、戦略計画 |
| **Camera setup** | 全キャラ表示が最良（待機時間摩擦が最小） |
| **Difficulty levels** | ピア支援頻度を有意に増加 |

## 実践的設計ガイドライン

1. **コミュニケーション基盤**: ボイス/テキストチャット、仮想ジェスチャー、環境ベースシグナリング
2. **アバターデザイン**: 社会的表現がマルチプレイヤー体験中の社会化パターンに影響
3. **メカニクス選定**: 相補性・共有目標・オブジェクトインタラクションが**最高インパクト**
4. **アンチグリーフィング**: 協調フレームワーク内で競争的行動を抑制するルールとNPC

## 相互依存性の効果（関連研究）

- 高い相互依存性 → **コミュニケーション増加**、**フラストレーション低下**
- 共有コントロール → 知覚される有能感・自律性が低下
- 非対称協力ゲームは接続感・社会的エンゲージメント・没入感・コントロール快適性を向上
- 6種の非対称性: 能力 / チャレンジ / インターフェース / 情報 / 投資 / 目標・責任
