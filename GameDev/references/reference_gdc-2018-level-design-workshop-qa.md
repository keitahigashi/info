## GDC 2018 レベルデザインワークショップ エキスパートラウンドテーブルQ&A

### 概要

GDC 2018のレベルデザインワークショップ（2018年3月20日開催）に先立ち、業界10人の専門家がコミュニティからの質問に回答。Guerrilla Games・Naughty Dog・Fullbright・Bithell Gamesなど多様なスタジオの観点が集約されている。

### プレイヤー誘導のテクニック

視覚的戦略の推奨事項：
- ランドマーク・ライティングコントラスト・カラーコーディング・アフォーダンスを活用する
- **Shaver（Naughty Dog）の実践**: 「目標物の周囲をすべて暗くして、目標に光を当てる。人は光に引き寄せられる」
- **Totten**: 「環境アートを言葉として、レベルデザインをプレイヤーとの会話として捉える」

### 設計哲学：美観とゲームプレイの統合

美観とゲームプレイを切り離さず相互補完として設計する。Dishonored のサンドボックス設計（複数の機械的アプローチによるプレイヤーエージェンシー）が参照事例として挙がる。

### オープンワールドのナビゲーション設計

Kevin Lynch の著書『Image of the City』のフレームワークを応用：

| 要素 | 内容 |
|------|------|
| ノード (nodes) | 集中活動ポイント・交差点 |
| 地区 (districts) | 特徴を持つエリア |
| ランドマーク (landmarks) | 目立つ参照点 |
| エッジ (edges) | 境界・線形要素 |
| パス (paths) | 移動ルート |

このフレームワークは都市設計を超えて、あらゆるゲームのナビゲーション設計に適用可能。

### テスト課題・採用評価

- Shaver（Naughty Dog）: レベルデザイナーにはブロックメッシュレイアウト、スクリプターには擬似コードを1日作業範囲で評価
- 採用時は実際の「設計能力の証明」が重視される

### 推奨学習リソース

**書籍:**
- Francis D.K. Ching『Architecture: Form, Space, and Order』
- Jesse Schell『The Art of Game Design』
- Kremers『Level Design: Concept, Theory, and Practice』

**分析推奨ゲーム:** Super Metroid・Half-Life 2・Dishonored 2

### 実践的アドバイスまとめ

1. 紙スケッチから始め → エンジン内ブロックメッシュ → 最終アート
2. 即座に反復的プレイテストを実施
3. コントラスト（色・ライティング・ジオメトリ）でプレイヤーの注意を誘導
4. 設計制約確立時にカメラ視点を考慮する
5. 建築原則を空間設計の理解に活用する
