## スタジオ概要

Envar Studio: スウェーデン拠点、32名から約200名へ成長。Riotでの長年の経験（VALORANT開発を10名→150名で経験）を基に設立。2024年12月にスウェーデン株式市場上場。

## アウトソーシングの運営原則

### プロダクション原則
「品質が最優先」— 信頼構築の基盤。

### プロジェクト開始プロセス
1. クライアント目標・パイプライン・期待値の完全理解
2. 初日からの明確なスコープ・スケジュール・コミュニケーションルーチン・レビュー周期設定
3. 一貫性のある更新、透明なコミュニケーション、期限内納品を優先

## 差別化要因

| 要因 | 説明 |
|------|------|
| **スタイライズドアート専門性** | 2Dコンセプト + 3D（キャラ/環境/テックアート） |
| **大規模一貫性** | 長期間・大規模チームで同じ品質基準を維持 |
| **パートナーシップ志向** | ベンダーではなくクライアントチームの拡張として機能 |
| **文化的理解** | Riot 6年のオンサイト経験で企業文化・品質基準を内在化 |

## クライアント関係の進化

「ベンダーから内部チームの拡張へ転換するには、外部供給業者ではなく**真のパートナー**として行動すること」が鍵。

### 業界の集約化傾向
- 多数の小規模ベンダー → 信頼できる少数スタジオへ統合
- 一回限りの納品 → 長期パートナーシップへシフト
- 信頼性・コミュニケーション・オーナーシップがアート品質と同等に重要

## スケーリングの課題

| 規模 | 課題 |
|------|------|
| 30→60名 | コミュニケーション構造の変化 |
| 60→120名 | 一貫性と品質管理の複雑化 |
| 120→200名 | 文化維持・リーダーシップ構造・採用基準 |

### 対応策
- **有機的成長**: 品質・文化・安定性を守る拡大
- **採用基準**: スキル以上にマインドセット・文化適合性を重視
- **オンボーディング**: 構造化された学習プロセスと明確な標準設定

## 二重事業構造（サービス + 内製ゲーム）

- 独立した運営体制（各々に専任リーダーシップ）
- **相乗効果**: サービス業務の品質管理・スコーピング精度がゲーム開発に活用
- 文化面: 高い期待値と反復作業による専門性の維持

## 2026年の業界展望

- COVID期ブーム後の構造調整（レイオフ・シャットダウン多発）
- 内部採用は3〜6ヶ月要するが、ニーズは1〜4週間で発生
- 高品質アウトソーシングの価値が増加
- 信頼できるパートナーは「迅速な統合、長期的オーバーヘッド不在」を提供

## 汎用化ポイント

- パートナーシップ志向のアウトソーシング原則はスタジオ規模を問わず適用可能
- スケーリング時の文化維持戦略は内部チーム成長にも応用
- プロジェクト開始時のスコープ/コミュニケーション設定は汎用的ベストプラクティス
