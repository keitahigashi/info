## 古典的ナラティブ構造フレームワーク

### 三幕構成 (Three-Act Structure)
アリストテレスの『詩学』に由来する基本構造:
- **第一幕**: 始まり（設定・提示）
- **第二幕**: 中盤（対立・展開）
- **第三幕**: 終わり（クライマックス・解決）

現代的応用:
| フレームワーク | 提唱者 | 特徴 |
|-------------|--------|------|
| 中心プロット構造 (Central Plot Structure) | Robert McKee | 映画脚本向け・葛藤の中心性 |
| パラダイム (Paradigm) | Syd Field | 三幕構成の具体的ページ配置 |
| 英雄の旅 (Monomyth / Hero's Journey) | Joseph Campbell | 普遍的神話パターン・試練と変容 |

### 英雄の旅の進行
プロタゴニストが試練を通じてクライマックスと解決に到達するまでの普遍的旅路。

> **重要原則**: 「ツールはプロジェクト固有のニーズに合わせて柔軟に適用すべきであり、硬直したフォーマットに従うべきではない」

## スタイル技法 (Style Devices)

### 文学的技法のゲームへの応用

| 技法 | 説明 | ゲーム事例 |
|------|------|-----------|
| **寓意 (Allegory)** | 倫理的葛藤を架空の設定で描く | Papers, Please（移民審査を通じた全体主義批判） |
| **暗喩 (Metaphor)** | 物質・要素を象徴として用いる | BioShock（ADAM = 薬物依存の暗喩） |
| **誇張 (Hyperbole)** | 効果を強調した誇張表現 | – |
| **パロディ (Parody)** / **サタイア (Satire)** | 既存形式の揶揄・批評 | – |

### 信頼できない語り手 (Unreliable Narrator)
- **タイプ1**: 誠実な記憶の間違い（意図しない誤情報）
- **タイプ2**: 意図的な欺き（プレイヤーへの意図的誤誘導）

ゲーム事例:
- *Prince of Persia* — 語り手の記憶が物語を再構築する
- *Disco Elysium* — 主人公自身が自分の過去を理解していない

### その他のナラティブ技法
- **省略 (Omission)**: 語られない空白がプレイヤーの想像力を刺激する
- **オープンエンド (Open-ended Storytelling)**: 解釈の余地を残し、プレイヤーの解読を報酬化する

## ドラマティックフォース (Dramatic Forces)

Alexander Swords による、ナラティブの推進力の分類:

### 主要フォース
| フォース | 定義 |
|---------|------|
| **逆転の運命 (Reversal of Fortune)** | 状況の予期しない反転 |
| **エスカレーション (Escalation)** | 緊張・危険の段階的上昇 |
| **変容 (Transformation)** | キャラクターや世界の本質的変化 |
| **介入 (Intervention)** | 外部要因による流れの変化 |

### 要素間フォース
- **対立 (Opposition)**: 相反する要素の衝突
- **相乗 (Synergy)**: 要素の組み合わせによる効果増幅

## 進行と整合 (Progression and Alignment)

ドラマティックテンションの中に「呼吸できる余地」を設ける設計:

| 概念 | 説明 |
|------|------|
| **進行 (Progression)** | 予測可能な展開・期待通りのイベント |
| **整合 (Alignment)** | 自由プレイの瞬間・緊張からの解放 |

プレイヤーが常に緊張状態にならないよう、意図的にテンションを緩める区間を設計することが重要。

## 設計的示唆

1. **既存の物語フレームワークをそのまま適用するのではなく、プロジェクトに合わせてカスタマイズする**
2. **信頼できない語り手はゲームの強みを活かした技法**（プレイヤーが主体的に謎を解く構造と相性が良い）
3. **省略とオープンエンド設計で世界観の奥行きを演出できる**（全て説明しすぎない）
4. **ドラマティックフォースを意識してビートを配置**することで、感情的な山谷のリズムを制御できる
