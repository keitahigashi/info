## 5つのコアメカニクス

### 1. Tower Placement（タワー配置）

「プレイヤーは最適な位置にタワーを戦略的に配置し、効果を最大化する必要がある」

- 位置ごとの戦略的価値の差異が基本基準
- 長いパス区間に到達できる位置が高価値
- 分岐点はタワー配置の戦略的ポジション

### 2. Resource Management（リソース管理）

敵撃破/パッシブ収入でリソース蓄積 → 新タワー建設 vs 既存強化への戦略的配分。

### 3. Enemy Waves（敵ウェーブ）

段階的難易度上昇。多様な敵タイプ（飛行ユニット/重装甲等）が防御戦略の適応を強制。

### 4. Tower Upgrades（タワーアップグレード）

既存タワー強化 vs 新防御構造建設の判断。即時ニーズと長期利益のバランス。

### 5. Map Layout（マップレイアウト）

複数パス・チョークポイントの物理的設計がタワー配置と全体戦略に根本的影響。

## プレイヤーが楽しむ要素

| 要素 | 説明 |
|------|------|
| 戦略的計画 | 防御最適化の知的挑戦 |
| インクリメンタルプログレッション | 脆弱→有能への段階的成長感 |
| 高リプレイ性 | 難易度/マップバリエーション |

## 設計上の課題

| 課題 | 対策 |
|------|------|
| 反復性 | タワー/敵のバリエーション増加 |
| 急な難易度スパイク | 段階的な複雑性導入 |
| リソース管理の複雑さ | 新規プレイヤー向け簡易モード |

## バランス調整の核心

- タワーパワーと敵強度の**精密な均衡**
- 新タワー/敵/マップの段階的導入で停滞を防止
- 特殊能力・多様なリソースタイプ・動的環境要素で**戦略的深度**のレイヤー追加
- 明確なビジュアルフィードバック＋満足感のあるオーディオキューで没入感向上

## 模範作品

- **Plants vs Zombies**: グリッドベース配置＋経済レイヤリング
- **Bloons TD**: タワーバリエーション＋アップグレード深度
- **Kingdom Rush**: 固定位置＋ヒーロー統合＋戦闘経済
