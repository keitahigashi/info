## The User Interface Continuum: A Study Of Player Preference（UIの連続体：プレイヤー選好研究）

著者: Kristine Jørgensen（ベルゲン大学シニアレクチャラー） | 公開: 2011年4月12日

### 概要

ゲームのHUD（ヘッドアップディスプレイ）に対するプレイヤーの受容性を定性的に研究した学術記事。Diablo II・The Sims 2・Crysis・Command & Conquer 3 の4作品で22名のプレイヤーを調査し、インターフェイスに対する3種のプレイヤーアーキタイプを発見。「HUDは没入を壊す」という一般的な懸念を覆す。

---

### 研究方法

| 項目 | 内容 |
|-----|------|
| 対象ゲーム | Diablo II / The Sims 2 / Crysis / Command & Conquer 3 |
| 参加者 | 22名（様々なゲーム経験） |
| 手法 | 定性インタビュー・プレイ観察 |
| 目的 | HUDへの態度・受容パターンの類型化 |

---

### 3つのプレイヤーアーキタイプ

#### 1. フィクショナリスト (Fictionalist)
- ゲームの物語世界への没入を最重視
- HUD等のインターフェイス要素が「虚構の断絶 (breaking fiction)」を引き起こすと感じる傾向
- 現実世界と切り離されたゲーム体験を求める
- **少数派**

#### 2. システマティスト (Systematist)
- ゲームの形式的ルールシステム・メカニクスを最重視
- フィクション（物語的側面）は二次的な関心事
- インターフェイスの説明や学習コストを気にしない
- **少数派**

#### 3. 相対主義者 (Relativist)
- フィクションと機能的情報伝達の両立を支持する **最多派**
- 情報伝達の明瞭さを優先するが、物語的没入も完全には否定しない
- HUDを「慣習 (convention)」として受け入れており、虚構を壊すものとは認識しない

---

### 核心的発見：「インタラクティビティが没入を駆動する」

> 「ゲームには見えない第四の壁が存在しない」

映画や演劇では観客と作品の間に「第四の壁」が存在するが、ゲームではプレイヤー自身がインタラクティブに参加するため、HUDのような慣習的UIは「没入の破壊」ではなく「ゲームプレイの一部」として機能する。

インタラクティビティ自体が没入を生み出すため、ほとんどのプレイヤーはオーバーレイUIを受け入れられる。

---

### UIデザインへの示唆

1. **「HUD=没入破壊」の過剰心配は不要** — 相対主義者（多数派）は慣習化されたUIを問題視しない
2. **ターゲットプレイヤー層の把握が先決** — フィクショナリストが対象なら没入型UI（ダイエジェティック）を検討
3. **情報伝達の明瞭さは多数が求める最低条件** — システマティストも含め機能的な情報提供は必須
4. **UI慣習の学習コストは存在する** — ただしジャンル経験者には問題にならない

---

### ゲームUIの連続体（概念図）

```
フィクション優先 ←───────────────────→ 機能優先

完全ダイエジェティック UI              非ダイエジェティック UI
（Dead Space のリング体力表示）      （従来型 HUD バー）

フィクショナリストが好む              システマティストが好む
         ↑
    多数派の相対主義者は中間を許容
```
