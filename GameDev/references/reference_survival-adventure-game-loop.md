## コアピラー（4本柱）

「探索 (exploration)、生存 (survival)、戦闘 (combat)、プログレッション (progression)」の4テネットを中心に相互接続。戦闘偏重ループを超えた、よりリスキーで魅力的な体験を構築。

## リソース収集とプログレッション

### RNDシステム

- ユニークな生物・植物のスキャンで研究ポイント獲得（敵対的/希少な標本ほど多い）
- 遺跡・宝箱に散在するサイエンスノートが研究を補完
- 破壊分析器: アイテムを破壊して技術をアンロック → 発見素材に紐づくテックツリー

### 惑星リソース

フローラが複数目的に機能: 医療・食料・クラフト素材・環境耐性バフ。惑星の難易度ティアがルート品質を決定。

## クラフトシステム

アイテムの稀少性がクラフトを生存の中心に据える。薬・武器・防具・ツールの継続的製造が必要。装備メンテナンスに惑星リソース（旋盤用のオイル/潤滑剤）を要求 → 探索がオプションではなく**必須**に。

## 基地建設と惑星オペレーション

ドロップポッドで初期物資と地上基地用シェルターカプセルを投下。採取装備を構築し、燃料と労力を要するポッドランチャーで軌道に帰還 → 意味のあるロジスティクスチャレンジ。

## 脅威設計

- ラウンド開始時の装備を削減し慎重な計画を強制
- 環境ハザード（放射線、酸性雨）にクラフトした耐性アイテムが必要
- 危険な惑星を準備要件の背後にゲート
- PvPゾーン: 優れたリソースとフル戦闘リスク

## システムの相互接続

探索 → 素材獲得 → クラフトでツール/耐性作成 → より難しい惑星アクセス → レア研究/ルート → 技術進歩。このサイクルが複数のプレイスタイルでエンゲージメントを維持。
