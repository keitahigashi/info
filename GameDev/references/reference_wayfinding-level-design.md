## ウェイファインディング (Wayfinding) とは

**ウェイファインディング**は2つのプロセスで構成される:
- **オリエンテーション (Orientation)**: プレイヤーが現在地を認識すること
- **ナビゲーション (Navigation)**: 目的地への経路を発見すること

従来の「プレイヤーを正しい方向へ操作する」発想ではなく、「プレイヤーは様々な方法でゲームを遊び、自らの遊びを構成する」という認識に立つ。

## 心理学的基盤

### メンタルマップ (Mental Map)

プレイヤーが空間を概念的に表現する構造。5要素:

| 要素 | 説明 |
|------|------|
| **パス (Path)** | プレイヤーが移動する経路 |
| **エッジ (Edge)** | 地域の境界・壁・崖 |
| **ディストリクト (District)** | 特徴で区別される地域エリア |
| **ノード (Node)** | 経路の交点・重要な場所 |
| **ランドマーク (Landmark)** | 空間内の目印となる要素 |

### デッド・レコニング (Dead Reckoning)

出発点・移動方向・速度・経過時間から現在位置を推定する航海術的能力。プレイヤーは常にこれを行っており、デザインはこれを支援または妨害する。

## 9つのナビゲーション戦略

1. **Track following**: 可視のパスを辿る
2. **Route following**: 記憶した経路を再現
3. **Educated seeking**: 論理的推測による探索
4. **Inference**: 環境の手がかりから推論
5. **Screening**: 選択肢を消去していく
6. **Aiming**: 見えている目標へ向かう
7. **Map reading**: UI マップを参照
8. **Compassing**: コンパス/方位を利用
9. **Social navigation**: NPC や他プレイヤーに追従

## 技法の4カテゴリ（確実性段階別）

### 1. Subtle（1〜20% 確実性）
*微妙な合図・控えめなヒント*

- Allegory（象徴・寓意）
- Worldbuilding（世界観の一貫性）
- NPC フォロー
- Unusual detail（異常な細部の強調）
- Sound design（音による誘導）
- Typology（建築様式・空間類型）

### 2. Coarse（35〜60% 確実性）
*提案的・既知パターンの活用*

- Composition（構図・三分割法）
- Lighting（照明で視線を誘導）
- In-world signage（世界内の看板・標識）
- Environmental storytelling（物語的環境配置）
- Ground composition（床の模様・テクスチャの流れ）
- Repetition（同じ要素の繰り返しによる方向付け）

### 3. Situational（70〜93% 確実性）
*強い促進・状況依存的誘導*

- Scripted sequences（スクリプトシーン）
- Resources / breadcrumb trails（リソース配置によるパン屑誘導）
- Cutscenes（カットシーン）
- Dynamic UI（動的ミニマップ・目標マーカー）
- Active threats（敵や危険による経路強制）

### 4. Direct（95〜98% 確実性）
*物理的・恒久的方向指定*

- Static barriers（動かせない壁・障害物）
- Always-on HUD（常時表示 HUD ナビゲーション）
- Hard obstacles（通過不可能な物理障壁）

## 実践の3原則（Half-Life 2 事例より）

1. プレイヤーは目を引くものがない限り**上方を見ない**
2. プレイヤーは**進行方向に注視**する
3. プレイヤーは**コントラスト**（色・形・照明・動き）に焦点を当てる

→ 導線上に明るい色・動くオブジェクト・コントラストの強い要素を配置することで自然に誘導できる。

## アンチ・ウェイファインディング (Anti-Wayfinding)

意図的に迷子を作る逆説的設計。**野生性・真正性・探索感**を重視するゲームに有効。

- ランドマークやサインポストを最小化
- 「ランダムに散布されたジオメトリ」でプレイヤー自身のルート創造を促進
- 例: サバイバルゲーム、ホラーゲームの迷子体験

## 検証の重要性

「有効な手法はプレイテスト検証を通じて初めて確認可能」。心理学理論だけでは不十分。空間構成への効果はプレイテストで初めて証明される。

## カテゴリとの関連

- **すべての要素がウェイファインディング情報として機能する**: ライティング・サウンド・NPCの動き・床のテクスチャなど
