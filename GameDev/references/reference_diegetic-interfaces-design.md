## 定義

ダイエジェティックインターフェースは、外部オーバーレイではなくゲームのナラティブ世界に論理的に統合されたUI要素。Fallout の Pip-Boy や Dead Space の Isaac Clarke の RIG スーツのように、ワールドビルディングを強化する機能的ツール。

## 没入感効果

ダイエジェティックインターフェースは**没入感の乗数 (immersion multiplier)** として機能。情報が有機的に提示されることで、プレイヤーはキャラクターを離れずに情報にアクセスでき、認知負荷 (cognitive load) を軽減。外部観察者ではなくアバターの視点でワールドを体験する。

## 設計原則 (4つの基盤)

| 原則 | 説明 |
|------|------|
| **一貫性 (Consistency)** | ゲームの確立されたロアとビジュアルスタイルに合致 |
| **機能性 (Functionality)** | 情報を明確かつ簡潔に提示 |
| **統合 (Integration)** | インターフェースを環境にシームレスに埋め込む |
| **フィードバック (Feedback)** | プレイヤーのアクションに対する即座の直感的な応答 |

## 実装課題と解決策

| 課題 | 解決策 |
|------|--------|
| 不明瞭性 (Obscurity) | 厳密なユーザーテスト |
| 侵入性 (Intrusiveness) | 反復的洗練 |
| 不整合な適用 | 文脈化と情報の優先順位付け |
| 過度な依存 | カスタマイズオプションとアクセシビリティ配慮 |

## 代表的ゲーム事例

| タイトル | ダイエジェティックUI |
|---------|-------------------|
| **Dead Space** | RIGスーツにHP/弾薬表示。従来のHUDを排除 |
| **Metro 2033** | 腕時計が時間と放射線レベルを示す。サバイバルの象徴 |
| **Mirror's Edge** | 環境色彩でルートを誘導 |
| **Far Cry 2** | 地図とコンパスを物理的に手に持つ |
| **Red Dead Redemption 2** | 没入感とユーザビリティのバランスを達成 |

## 核心的助言

微妙さ (subtlety) を受け入れること — 複雑さでプレイヤーを圧倒するのではなく、最小限の戦略的要素を使用する。
