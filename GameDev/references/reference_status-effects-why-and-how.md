## ステータスエフェクトの価値

### 戦略的深さ
- 直接ダメージより「計画性」を重視
- ターン順序や資源管理に影響
- 複数効果の相乗作用による複雑な戦術

### 予測不可能性と緊張感
- 確率ベースの「ハイリスク・ハイリターン」
- 劇的な逆転の可能性

### システム習熟の報酬
- 効果の相互作用を理解するプレイヤーが優位
- 専門的なチーム構成の可能性

## 6つの分類

| 種類 | 説明 | 例 |
|------|------|-----|
| **Over Time** | ターンごとの段階的変化 | 毒、再生 |
| **Buff/Debuff** | 数値ステータス変更 | ヘイスト、弱化 |
| **Debilitating** | 行動制限 | スタン、沈黙 |
| **Loss of Control** | 意思決定奪取 | 混乱、魅了 |
| **Null** | 耐性付与 | シールド、火炎耐性 |
| **Transformative** | 状態変化 | 模倣、元素反応 |

## 設計失敗パターン

| 問題 | 影響 |
|------|------|
| **ボス免疫** | 効果系の選択肢をプレイヤーが放棄 |
| **信頼性の欠如** | 確率失敗が多すぎて利用を躊躇 |
| **難易度不足** | ステータス効果が不要に見える |
| **行動奪取の過剰** | 長時間スタンによるフラストレーション |

## 設計ベストプラクティス

- **累積成功率**: 失敗が続くと成功確率上昇
- **ゲージシステム**: 複数回適用後に確定発動
- **ペース維持**: スタンより出血/ディフェンスブレイク優先（攻撃的効果で戦闘加速）
- **差別化**: 各効果に独自の役割付与、重複排除

## 戦闘外での応用

| ゲーム | 応用 |
|--------|------|
| **ゼルダ BotW** | 氷・火・電気の環境反応 |
| **ダーケストダンジョン** | 心理状態としてのナラティブ表現 |
| **Stardew Valley** | 食事バフによるシミュレーション |
