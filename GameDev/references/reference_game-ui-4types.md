## 概要

ゲームUIはプレイヤー体験の充実、没入感の向上、戦略の支援に寄与する。4つの主要タイプが存在する。

## 4つのUIタイプ

### 1. ダイエジェティック (Diegetic)

ゲーム世界の「一体的な部分」であり、プレイヤーとキャラクターの両方に見える。

| 項目 | 内容 |
|------|------|
| **強み** | 没入感を破るオーバーレイを排除。キャラクターとのリアルなインタラクション |
| **弱み** | 情報の明確性が低下する可能性。アクションシーンで読みにくい |
| **代表例** | Dead Space（スーツに表示されるHP）、GTA V |
| **適したケース** | 没入感とナラティブ一貫性を優先するゲーム |

### 2. 非ダイエジェティック (Non-Diegetic)

「プレイヤーのみに表示される」グラフィックオーバーレイ。キャラクターは認識できない。

| 項目 | 内容 |
|------|------|
| **強み** | 明確でアクセシブルな情報提供。迅速な戦略判断を支援 |
| **弱み** | 人工的・侵入的に感じうる。画面の乱雑化リスク |
| **代表例** | World of Warcraft（HP/マナ/アビリティのHUD） |
| **適したケース** | ストラテジー、シミュレーション、競技ゲーム |

### 3. スペーシャル (Spatial)

ゲーム世界内に表示されるが、キャラクターには見えない。ダイエジェティックと非ダイエジェティックのハイブリッド。

| 項目 | 内容 |
|------|------|
| **強み** | 没入感と機能性のバランス。自然なナビゲーション支援 |
| **弱み** | 実装が複雑。やや人工的に感じうる |
| **代表例** | Assassin's Creed（敵の頭上の名前）、Splinter Cell: Conviction（ミッションガイドの光路） |
| **適したケース** | ナビゲーション支援を没入感を損なわず提供したいゲーム |

### 4. メタ (Meta)

キャラクターの状態を反映するが、ゲーム世界の「物理的オブジェクト」としては表示されない。

| 項目 | 内容 |
|------|------|
| **強み** | 侵入的なHUDなしにドラマティックな緊張感。直感的な危険伝達 |
| **弱み** | 繊細すぎて重要情報を見逃すリスク。ステータス固有の指標に限定 |
| **代表例** | Call of Duty: Modern Warfare（被弾時の画面血しぶき） |
| **適したケース** | アクション・シューターでの強度と緊迫感の演出 |
