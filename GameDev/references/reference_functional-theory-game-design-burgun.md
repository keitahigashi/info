## 機能的理論（Functional Theory）：ゲームデザインに客観的評価基準を

### 問題提起：40年間の方向性の誤り

- ビデオゲーム業界は40年間、グラフィクス技術・新入力デバイスに注力してきた
- ゲームデザインそのものの体系化・理論化はほぼ進んでいない
- 「より多くのゲームを作れば良くなる」という量的拡大論では不十分

### 中心的な主張：機能的理論（Functional Theory）の必要性

- 音楽理論が音楽の多様な発展を可能にしたように、ゲームにも機能的理論が必要
- 理論があれば：
  - 客観的批評基準の確立
  - 設計指針の共有
  - 搾取的メカニズム（スキナーボックス等）の識別と排除が可能になる

### ゲームの再定義：3種類の混同を解く

| 用語 | 定義 |
|------|------|
| コンテスト (Contest) | 競争要素を含むシステム |
| パズル (Puzzle) | 明確な目標があるが競争がない |
| ゲーム (Game) | 曖昧な意思決定を伴う競争システム |

現在「ビデオゲーム」と呼ばれるものはこれら3種が混在しており、それぞれ異なる設計原則が必要。

### 評価基準の提案

- 「意思決定の質と頻度」をゲーム評価の軸とする
- スキナーボックス (Skinner box) 的メカニクス（報酬サイクルへの依存）はゲームの質を下げると明言

### 批判への反論

- 「理論化は創造性を損なう」という懸念：音楽理論が音楽の多様性を損なわなかったように、理論は創造の足枷にはならない
- 「作り続けることで自然と良くなる」という楽観論：方向性のない量的増加はむしろ市場を悪化させる
