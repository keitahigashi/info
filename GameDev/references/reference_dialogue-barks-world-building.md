## バーク (Barks) の定義

NPCが会話メニューを介さず、特定状況で一方的に発する短い台詞。正式な対話ではなく、雰囲気とワールドビルディングのための自発的発声。

## 2つの配信方式

| 方式 | 説明 | 効果 |
|------|------|------|
| **内的独白** | NPCが思考を口に出す | 個人的な性格・願望の露出 |
| **断片会話** | プレイヤーが途中から聞く会話の断片 | 世界が自律的に動いている感覚 |

「必要な情報だけを捉えさせる」ことで情報過多を回避。

## 動詞ベース発想法 (Verb-Based Ideation)

バーク執筆は**動作動詞**から開始する:
- NPCが何をしているか（エンジン修理、休憩、紛争回避）
- 行動がトリガーとなる台詞を設計
- 棒立ちNPCの不自然さを排除

## コンテンツテーマ

### 日常生活
- 職場の不満と達成
- 家族関係と人間関係の緊張
- ゲーム世界内での個人的野望

### ワールドビルディング
- 政治的対立と派閥間の緊張
- 伝説の宝物や目標
- 地域のゴシップと文化的意見

## 性格差別化

複数の性格アーキタイプで均一化を防止:
- 陽気な鉱夫 vs 苦々しい鉱夫 → 同じ状況で異なる反応
- 話し方の癖・感情的傾向で自然な多様性
- 過度に目立たない程度のバリエーション

## 視点と世界観の統合

NPCはその社会の価値観を自然に反映すべき:
- ゲーム世界独自の道徳体系に基づく意見
- 現実世界の倫理をフィクションに投影しない
- 住民の意見は「慣れた生活様式」に寄り添う

## プレイヤー認知バーク

プレイヤーを直接参照するバークで没入感を創出:

| タイプ | 例 |
|--------|-----|
| 外見への言及 | 装備・見た目へのコメント |
| 行動への反応 | プレイヤーの選択に対する賞賛/非難 |
| 評判の反映 | 過去の行動に基づく態度変化 |

「プレイヤーが世界に本当に関わっている」と感じさせる。

## ゲームプレイ統合機能

雰囲気を超えた実用的役割:

| 機能 | 説明 |
|------|------|
| 危険警告 | 脅威の事前通知 |
| 目標明確化 | プレイヤーの混乱を防止 |
| 暗黙の報酬 | 完了した行動への微妙な称賛 |
| 結果の強化 | ストーリー進行の帰結を反映 |

## ベストプラクティス

- **情報階層の維持**: バークはゲーム目標を妨げない補助的役割
- **対立する視点の活用**: 複数派閥の見解で社会学的深みを創出
- **機能と雰囲気のバランス**: ナビゲーション支援と物語フレーバーの両立
- **反復の管理**: 同一バークの過剰反復は没入感を破壊
