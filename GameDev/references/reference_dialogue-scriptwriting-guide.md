## ダイアログの4核要素

| 要素 | 説明 |
|------|------|
| **キャラクター性** | 背景・性格・役割を反映した独自の話し方・語彙 |
| **文脈的関連性** | プロット展開、背景情報、キャラクター特性の開示 |
| **相互作用選択肢** | 分岐物語に対応した選択肢の提供 |
| **感情的影響** | 笑い、恐怖、悲しみ、興奮などの喚起 |

## 4ステップ執筆プロセス

1. **ゲーム世界の理解**: 世界観の内部に没入し、設定ルール・歴史を理解
2. **キャラクター開発**: 背景設定、動機付け、個人的成長曲線の構築
3. **プロット統合**: ストーリー推進の手段としての対話活用
4. **説明と没入感のバランス**: 自然な流れで必要情報を伝達

## 実践的スキル

- キャラクターに**真実性**を持たせる
- 自然な流れとリズムの創出（**音読検証**推奨）
- 台詞の裏にある含意（**サブテキスト**）の活用
- インタラクティブ性への対応

## 課題と解決策

| 課題 | 解決策 |
|------|--------|
| 複数分岐での一貫性維持 | 綿密なアウトライン作成 |
| メカニクスとの同期 | チーム間の効果的な協力 |
| 品質保証 | ベータテストに基づく反復修正 |

## データ分析との統合

プレイヤー行動分析でどのダイアログが効果的かを測定。分析ツール活用により、成功するナラティブパターンを特定。
