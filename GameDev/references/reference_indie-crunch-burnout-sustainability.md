## 自己課題クランチの3つの原因

### 1. ロマンチックな神話

「苦しむアーティスト」概念が苦痛を「必要悪、通過儀礼」と位置づけ、自己破壊を芸術的メリットと等価に。

### 2. 経済的脆弱性

限られたリソースと雇用不安が「ついに成功する」圧力を生み、合理的限界を超えて駆動。

### 3. コントロールの幻想

追加時間が完璧を保証すると信じるが、「ゲーム開発は本質的に予測不可能」。

## バーンアウトパターン

情熱から毒性への進行: 仕事と生活の境界溶解 → 睡眠が犠牲に → 「創造の喜びが強迫的な駆動に変容」。身体的/精神的健康の悪化、人間関係の緊張、創造性の低下 → 自己永続的サイクル。

## 持続可能性10項目ツールキット

| # | 戦略 |
|---|------|
| 1 | **スコープ管理**: MoSCoW法で不要フィーチャー排除 |
| 2 | **タイムボックス**: ポモドーロ（25分集中ブロック） |
| 3 | **タスク分解**: Trello/Asanaで管理可能なステップに |
| 4 | **自動テスト**: デバッグ時間削減 |
| 5 | **バージョン管理**: Git/GitHubで安全とストレス軽減 |
| 6 | **定期的な休憩**: 勤務日中の切断期間をスケジュール |
| 7 | **現実的なデッドライン**: 予測不能性を認めた余裕 |
| 8 | **チームコミュニケーション**: 進捗を率直に議論し期待を調整 |
| 9 | **委任**: チーム内でワークロードを効果的に分配 |
| 10 | **睡眠優先**: 認知パフォーマンスのため7-8時間 |

## 成功事例: Pixel Dreams

アジャイル導入 + 厳格な残業禁止 + プロジェクト管理ツールへの投資 → 生産性・士気・ゲーム品質が向上。ウェルビーイングを犠牲にせず成功可能と実証。
