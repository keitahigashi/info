## 水面レンダリングの4主要要素

### 1. Reflection（反射）

環境、ゲームオブジェクト、光源、自己反射の描画。

**平面反射 (Planar Reflection) の実装手順**:
1. 水面に対してカメラを対称にミラーリング
2. 反転クリッピングプレーンを使用して新フレームバッファにレンダリング
3. 結果テクスチャをシェーダー技法で水面プレーンに適用

**コスト**: 水面プレーンの数に比例してスケール。

### 2. Refraction（屈折）

水中要素の色・波・深度に基づく歪み表現。

- 現在のカメラとクリッピングプレーンで水中要素のみをレンダリング
- 別フレームバッファに出力後、シェーダーエフェクトを適用
- 反射より実装がシンプル

### 3. Waves（波）

自然力による表面歪み。水体タイプ（海/湖/川/池）によって異なる。

#### DuDv Mapping（歪みマッピング）

- R/Gチャネルで方向歪みを表現
- フラグメントシェーダーレベルでリプルエフェクト生成

#### Normal Mapping（法線マッピング）

- RGBチャネルでXYZ方向を表現
- 表面ディテールのイリュージョンを提供
- DuDvと組合せでリアリスティックな波表現

### 4. Caustics（コースティクス）

水面上下に見える光パターン。

- **事前ベイク済みアニメーションテクスチャ**を使用
- **加算ブレンディング**で適用
- 対象の深度に基づいて透明度を増加

## フレネル効果 (Fresnel Effect)

反射率を視線角度で変調するリアリズム向上技法:

| 視線角度 | 反射強度 |
|---------|---------|
| 浅い（水面に平行に近い） | **強い**反射 |
| 急な（水面に垂直に近い） | **弱い**反射（水中が透けて見える） |

## レンダリングパイプライン概要

```
1. 反射パス: ミラーカメラ → フレームバッファ → 反射テクスチャ
2. 屈折パス: 通常カメラ + クリッピング → 屈折テクスチャ
3. 波動処理: DuDv + Normal Map → UV歪み
4. 合成: フレネル係数で反射/屈折をブレンド
5. コースティクス: 加算ブレンドで光パターン重畳
```
