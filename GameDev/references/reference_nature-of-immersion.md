Game Developer (Jonathan Michalik): https://www.gamedeveloper.com/design/the-nature-of-immersion

## 問題提起

ゲーム業界では「没入感（immersion）がある」という表現が過剰に使われており、意味が希薄化している。開発者は「どのような種類の没入感か」「どのような方法で没入するか」を具体的に定義する必要がある。

> 「単に『没入感がある』と述べるだけでは、ゲームの独自性が明確に伝わらない」

## 没入感の分類フレームワーク

### 1. 延長的没入感（Extensional Immersion）

プレイヤーの身体がゲーム世界の延長として機能する感覚。

- **典型例**: タッチスクリーンの指スラッシュゲーム（指 = 仮想の刃）
- **特徴**: 物理的な入力がゲーム内アクションと1対1で対応
- **感覚**: 「自分の身体でゲーム世界に触れている」

### 2. 空間的没入感（Spatial Immersion）

プレイヤーがゲーム空間を「自分の生活世界」のように自然にナビゲートできる感覚。

- **典型例**: MMORPG（既知の世界に住んでいるように自然に移動できる）
- **特徴**: ゲーム空間への深い親密性と直感的なナビゲーション
- **感覚**: 「ゲーム世界に『存在している』」

## 設計への応用: 「方法と感覚の公式」

```
ゲームジャンル（プレイ方法） + 没入感の種類（プレイ時の感覚）
= 体験の完全な説明
```

**活用例:**
- モバイルアクション + 延長的没入感 → 「指でキャラクターを直接操る一体感」
- オープンワールド RPG + 空間的没入感 → 「この世界に住んでいるような感覚」
- VR ゲーム + 延長的没入感 → 「身体ごとゲーム世界に入り込む体験」

## ゲーム企画・マーケティングへの応用

この分類法によって:

| 場面 | 効果 |
|------|------|
| 企画書 | 「没入感がある」から「〇〇型没入感を提供する」への具体化 |
| プレイテスト | 没入感の種類別に評価設問を設計できる |
| マーケティング | 競合との明確な差別化説明が可能 |
| チーム内共有 | 設計方針の共通語彙として機能 |

## 没入感を妨げる要因の分類

没入感の種類を特定できると、何が没入を妨げているかも特定しやすくなる:

- **延長的没入の阻害**: 入力と結果のラグ、操作感の違和感、フィードバック不足
- **空間的没入の阻害**: 一貫性のない世界ルール、ローディング断絶、マップの不自然な制約
