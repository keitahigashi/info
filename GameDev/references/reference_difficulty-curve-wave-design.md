## 波状の難易度設計

固定的な難易度ではプレイヤーは「同じ感覚しか味わえない」。難易度を意図的に上下させることで、変化に富んだゲーム体験を創出する。

## 難易度変動の効果

| 局面 | 効果 |
|------|------|
| **上昇局面** | チャレンジ感を提供し緊張感を高める |
| **下降局面** | 休息ポイントを作り、達成感を与える |
| **サイクル化** | 繰り返しにより長時間プレイを維持 |

## 設計アプローチ

- ゲーム進行に応じた段階的な難易度調整が重要
- 視覚的グラフで理想的な曲線パターンを設計
- 難易度曲線を**意図的に**設計することで、開発者の意図した流れでプレイヤーを「飽きさせずに遊んでもらう」ことが可能

## フロー理論との関連

波状設計はフロー理論の実践的応用。上昇局面でフロー状態に入り、下降局面で回復し、再び上昇局面でチャレンジに向かうサイクルが、持続的なエンゲージメントの基盤。
