## オープンワールドナビゲーションの設計哲学

プレイヤーが「目に見えない手に導かれるような」感覚で自然に探索するよう誘導するのが優れたナビゲーション設計。迷子になりながらも発見の喜びを感じるメカニズムが重要。

## 主要誘導技法

### ブレッドクラミング (Breadcrumbing)

視覚的・環境的手がかりを連続して配置し、プレイヤーを無意識的に誘導する手法。

- **ランドマーク活用**: 山、塔、特徴的建物など遠くから見える目印で方向感覚を補助
- **光と色彩**: エルデンリングのグレース遺跡の光柱など、明るい光源で注意を引く
- **視線誘導**: 道の形状、樹木の配置、壁の方向で自然に進路を示す

### マップ構成アーキテクチャ

| 構造 | 説明 | 使用例 |
|------|------|--------|
| **ハブ&スポーク型** | 中心拠点から複数ルートへ放射状に延びる構造 | 多くのRPG |
| **段階的解放** | レベル・ツール習得に応じてエリアをアンロック | ゼルダBotW |
| **ハンドクラフト型** | 意図的な配置による密度の高い世界 | ウィッチャー3 |
| **手続き生成型** | アルゴリズムによる広大な世界生成 | No Man's Sky |

## デュアルレイヤーアプローチ

**オプション構造の重要性**: ウェイポイントや進行マーカーは「オプション」として提供し、初心者は依存でき、経験者は無視できる二層設計。

- マーカーOFFでも環境手がかりだけで進行可能な世界を作る
- Red Dead Redemption 2のHUDオン/オフ切り替えが好例
- プレイヤーの探索スタイル（マーカー依存型 vs 自由探索型）を両立

## 高速移動システムの設計

- 発見済みのポイントのみアンロック → 探索のモチベーションを維持
- 全ポイントへのテレポートは探索価値を損なう
- 徒歩移動と高速移動のコスト差で世界の広さを体感させる

## 代表的な優良事例

| ゲーム | 特徴的なナビゲーション手法 |
|--------|--------------------------|
| **ゼルダBotW** | 遠くから見える塔・シーカーセンサー・段階的エリア解放 |
| **エルデンリング** | 光の柱・場所の碑文・マルチパスルート |
| **ウィッチャー3** | クエストマーカーとミステリーポイントの併用 |
| **GTA V** | 三人称メガシティの密度と道路網の自然な誘導 |

## 設計上の注意点

- マップが大きすぎると「空虚さ」が探索の障壁になる
- ランドマーク密度は移動速度に比例して調整が必要
- ミニマップ依存を避けるには、環境自体が読みやすくある必要がある
