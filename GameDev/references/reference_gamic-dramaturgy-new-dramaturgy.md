## ゲーミックドラマトゥルギー入門

### 問題提起
MDAフレームワーク等の機械論的分析手法は普及しているが、ゲームの「感情的・劇的な側面」を扱う形式的システムが不足している。

### ドラマトゥルギー (Dramaturgy) とは
「物語の主要素の劇的構成と表現の芸術」。古典劇（アリストテレスのポエティクス等）に起源を持つ概念をゲーム設計に援用する試み。

デザイン部門とナラティブ部門の間で起きがちな「シネマティック vs. ゲーム性」の対立を解消するブリッジツールとして機能する。

### 二層フレームワーク

#### 経験層 (Experience Layer)
プレイヤーが直接感じる要素:

| 要素 | 説明 |
|------|------|
| **Idiom** | ゲームのアイデンティティと表現形式 |
| **Suspense** | 不確定な結果への投資による緊張感 |
| **Catharsis** | 感情的解放・浄化の感覚 |

#### 構造層 (Structure Layer)
経験を生み出す設計の基盤（プレイヤーには間接的にのみ知覚される）:

| 要素 | 説明 |
|------|------|
| **Pacing** | 感情的事象のリズム構造 |
| **Dramatic Substance** | サスペンスを生成する5要素（エージェンシー・キャラクター表現・コンプリケーション・プレイ統合・プレゼンテーション） |
| **Design Goal** | 作品全体を形作る中核概念 |

### 相互依存フロー
Design Goal → Idiom → Dramatic Substance → Suspense → Pacing → Catharsis

上位要素が下位要素の基盤となり、各層が階層的かつ相互依存的に機能する。

### 活用シーン
- ゲームの感情設計レビュー・批評に有効
- 機能的なゲームプレイが「なぜ感情的に響かないか」を診断するツール
- シネマティックとインタラクション設計の統合評価に応用可能
