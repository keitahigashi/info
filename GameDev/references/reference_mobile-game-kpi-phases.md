## 出典

gamebiz (AppLovin 林宜多氏): https://gamebiz.jp/news/195310

## 概要

520万個のアプリが存在する競争市場でのモバイルゲームKPI最適化戦略を4段階で解説。

## 4つのフェーズ別KPI

### Phase 1: 新規ユーザー獲得

| 指標 | 定義 |
|------|------|
| **CPI** (Cost Per Install) | インストール1件あたりのコスト |
| **CPA** (Cost Per Action) | チュートリアル完了等の特定イベント達成コスト |
| **CAC** (Customer Acquisition Cost) | 営業・マーケティング総コスト |

アトリビューション分析により成果の大きい流入経路への投資集中が可能。

### Phase 2: ユーザー体験・エンゲージメント

- **セッション分析**: セッション数・時間・間隔・深さをトラッキング
- 「ロード時間が長すぎるとセッションが短くなりLTV低下」
- アプリクラッシュ・フリーズはユーザー離脱・悪評に直結

### Phase 3: 収益化・評価

| 指標 | 定義 |
|------|------|
| **DAU** | 日単位のアクティブユーザー数 |
| **MAU** | 月単位のアクティブユーザー数 |
| **定着率** | DAU ÷ MAU（1に近いほど定着度高い） |
| **ARPU** | ユーザー1人あたり平均収益 |
| **LTV** | ユーザー生涯収益予想値 = 収益化 × リテンション × バイラリティ |

バイラリティはK-factorで計測し口コミ・ネットワーク効果を可視化。

### Phase 4: ユーザー維持

- **チャーンレート**: アンインストール率を測定
- 離脱理由とARPU/LTVを総合判断して復帰施策の優先度を決定

## 実践的活用ポイント

- 各フェーズで異なるKPIに焦点を当てることが最適化の鍵
- CPA基準でマーケティング予算を配分
- DAU/ARPU/LTV相関を継続的に監視し施策調整
