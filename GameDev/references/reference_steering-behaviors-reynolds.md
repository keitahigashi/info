## 3層行動階層

| 層 | 機能 | 例 |
|----|------|-----|
| Action Selection（上位） | 目標設定・戦略決定 | 「逃げる」「追跡する」の切替 |
| Steering（中位） | 目標を単純な操舵行動に分解 | Seek/Flee/Wander等 |
| Locomotion（下位） | 制御信号を物理的移動に変換 | 車輪/脚/翼の動き |

## 単純ビークルモデル

点質量近似 + 位置・速度ベクトル + 最大力・最大速度パラメータ。前進オイラー物理積分。車輪/航空機/潜水艦/歩行すべてに汎用的に適用可能。

## 操舵行動カタログ

### 移動行動

| 行動 | 説明 |
|------|------|
| **Seek** | 固定目標に向けて速度を放射状に整列。操舵ベクトル＝目標速度−現在速度 |
| **Flee** | Seekの逆。目標から離れる方向に操舵 |
| **Arrival** | Seekに減速を追加。目標接近時にランプダウンして目標上で停止 |
| **Pursuit** | 移動目標の**未来位置を予測**してSeek。遠距離→大きな予測時間T |
| **Evasion** | Pursuitの逆。移動脅威の予測未来位置からFlee |
| **Offset Pursuit** | 移動目標のオフセット位置に沿って操舵（航空機の掃射等） |

### 環境・障害物行動

| 行動 | 説明 |
|------|------|
| **Obstacle Avoidance** | 前方に仮想シリンダーの自由空間を維持。最も脅威的な障害物を回避 |
| **Path Following** | 予め定義されたパス（回廊/道路）に沿って移動。逸脱が半径超過時にSeek |
| **Wall Following** | 表面からの指定オフセット距離を維持 |
| **Containment** | 許可領域内に移動を制限。境界超過予測時に内側へ操舵 |
| **Flow Field Following** | ベクトル場のローカル接線に動きを整列。アーティストが「フロー」マップで方向制御可能 |
| **Unaligned Collision Avoidance** | 任意方向のキャラクター間衝突防止。最接近点を予測し横方向に回避 |

### ロコモーション行動

| 行動 | 説明 |
|------|------|
| **Wander** | 球面上の制約付きランダムウォーク。Perlinノイズでより滑らかな変動も可能 |

### 群れ行動（Group Behaviors）

| 行動 | 説明 |
|------|------|
| **Separation（分離）** | 近傍キャラクターとの最小距離を維持。**1/r²** 重み付けで反発力を計算 |
| **Cohesion（凝集）** | 近傍グループメンバーの重心に引き寄せ |
| **Alignment（整列）** | 近傍キャラクターの速度ベクトル平均に同期 |
| **Flocking（群れ）** | Separation + Cohesion + Alignment の組合せで**創発的**群れ行動 |
| **Leader Following** | リーダー後方のオフセット点にArrival + 互いのSeparation + リーダー前方回避 |

### 特殊行動

| 行動 | 説明 |
|------|------|
| **Interpose** | 2つの移動キャラ間に位置取り |
| **Hide** | 障害物の遮蔽側を特定してSeek |
| **Shadow** | 獲物に接近後、Alignmentで速度・方向を合わせる |

## 行動結合手法（5種）

| 手法 | 説明 | 特徴 |
|------|------|------|
| **Linear Combination** | 操舵ベクトルの重み付き和 | シンプルだが対立力の相殺リスク |
| **Momentum Filtering** | フレームごとに1行動選択、慣性で自然ブレンド | 軽量 |
| **Prioritized** | 優先順位付き、最初の非ゼロ結果を使用 | 競合防止 |
| **Prioritized Dithering** | 確率的に優先順評価 | ばらつきのある自然な行動 |
| **Prioritized Acceleration Allocation** | 高優先度が力の大きさを決定、残余を低優先度に | 最も精密 |

## 実装のポイント

- 操舵ベクトルは容易に結合可能（会話/感情行動とは異なる）
- フレーム間慣性が行動変化を自然にフィルタリング
- **空間分割**（グリッド/四分木）で近傍検索を最適化
- 高速エンティティには予測メカニズムが不可欠
