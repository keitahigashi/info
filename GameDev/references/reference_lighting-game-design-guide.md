## ライティングの核心的役割

照明は単なる照らしを超え、「ストーリーテリングツール、ゲームプレイメカニクス、感情の指揮者」として同時に機能。

## 4つの基本ライトタイプ

| タイプ | 説明 |
|--------|------|
| **アンビエント (Ambient)** | 基礎的な環境照明（空、グローバルライト）。ベースの視認性とムードを確立 |
| **ディレクショナル (Directional)** | 特定方向からの光（太陽光模倣）。影・深み・時間帯エフェクト |
| **ポイント (Point)** | 特定場所からの放射（たいまつ、電球、爆発）。ローカルな雰囲気 |
| **スポット (Spot)** | 集中ビーム。注意の誘導、精密なターゲティングで緊張感を生む |

## ムードと雰囲気の構築要素

- **色温度**: 暖色 = 快適/興奮、寒色 = 落ち着き/謎
- **強度レベル**: 明暗のコントラスト比
- **動きのパターン**: ダイナミックで応答的な照明が没入を強化

## ジャンル別応用

| ジャンル | ライティング戦略 |
|---------|----------------|
| **ホラー** | 暗闘をメカニクスに。懐中電灯と影のダイナミクス |
| **アドベンチャー** | ナラティブを強化する雰囲気的ガイダンス |
| **アクション** | 戦闘視認性を維持するクラリティ |
| **RPG** | ロアを反映する地域的雰囲気 |
| **パズル** | インタラクティブ要素を区別する明確なビジュアル階層 |

## よくある失敗

- 過剰照明がビジュアルノイズとパフォーマンス問題を生む
- パフォーマンスへの影響を無視
- 一貫性のないスタイリング
- プレイヤーガイダンスの不足
- ダイナミックストーリーテリングの潜在能力の無視
