## インタラクティブストーリーテリングの5核心要素

### 著者・背景

Thomas Grip（Amnesia・PENUMBRAシリーズ開発元 Frictional Games）が2013年に発表した、ナラティブゲームデザインの実践的フレームワーク。自社作品の試行錯誤から導出した原則を体系化。

### 5つの核心要素

**1. ストーリーテリングに焦点を当てる (Storytelling Focus)**
- ゲームの存在理由がナラティブ体験にあること
- パズル・戦闘などのゲームプレイ要素はナラティブのサポートに徹する
- 「なぜプレイヤーがここにいるのか」をナラティブで答え続ける設計

**2. プレイ時間が大部分を占める (Gameplay Majority)**
- 映像視聴や読書ではなく「プレイする」時間が中心
- インタラクション自体が「存在感 (presence)」を生み出すメカニズム
- プレイヤーは受動的な観客ではなく能動的な参加者であるべき

**3. インタラクションがナラティブ的意味を持つ (Narrative Interaction)**
- プレイヤーの行動が物語の重要な出来事に直結していること
- 「ドアを開ける」「スイッチを押す」という行為が物語の一部となる設計
- 操作と物語の乖離（ルドナラティブ乖離）を最小化する

**4. 反復的アクションの排除 (No Repetition)**
- 同一メカニクスの繰り返しは想像力・没入感を阻害する
- Braid/Limboの例：基本メカニクスは同一でも「常に変動する使用方法」を提供
- 「同じ動作で別の体験」を生み出す文脈の変化が鍵

**5. 主要な進行ブロックの排除 (No Blocking)**
- 難解なパズルや迷路で詰まるとプレイヤーの注意がナラティブから逸れる
- 「プレイヤーの注意力が常にナラティブに向かう状態」を維持
- 難易度とナラティブ没入のトレードオフを意識的に管理する

### 設計の優先順位

```
ナラティブ体験
    ↑
インタラクションの意味
    ↑
ゲームプレイ要素（メカニクス）
```

ゲームプレイ要素はナラティブを届けるための「器」であり、器自体が目的にならないよう注意する。

### 実例・参照ゲーム

| ゲーム | 該当する原則 |
|--------|------------|
| Amnesia | 1・3・5：恐怖体験をインタラクションで体験させる |
| Braid | 4：時間操作メカニクスを常に新しい文脈で活用 |
| Limbo | 4：シンプルなアクションを多様な状況で変化させる |
| 一般的なFPS | 2（失敗）：移動・射撃の繰り返しがナラティブを希薄化 |
