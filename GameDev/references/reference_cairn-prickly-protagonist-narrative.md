## Cairnの主人公設計分析

### 記事の前提

「優れたゲーム執筆はあらゆるゲームに存在している。私たちはそれを見つけるための道具を持っていないだけ」— Bryant Francis

### 主人公アヴァの設計

『Cairn』（The Game Bakers）の主人公アヴァは、従来のゲーム主人公と対比的に設計されている：

| 従来の不可能目標キャラクター | アヴァ |
|--------------------------|------|
| 熱心・前向き | 気難しく・無愛想で・無情 |
| 成長を目指す | 人間性を失う危険を抱える |
| 感情的につながりやすい | 意図的に距離を置く |

### 感情設計の手法

#### サブテキスト (Subtext) の活用
- 台詞やメッセージを通じて明示せず、プレイヤーが推測するよう設計
- カットシーン以外（登山中のセリフ）でも性格を表現し、没入を維持
- メッセージへの怒り・若い登山者マルコへの無礼・Climberbotへの最初の冷淡さ

#### 強制的なモーメント
- プレイヤーはマルコに対して暴言を吐く場面でのみ進行できる
- これにより「プレイヤーがアヴァを体験する」のではなく「アヴァとして行動する」状況を作る

#### 複雑なキャラクター弧 (Character Arc)
アヴァの物語は従来の「他者を受け入れることを学ぶ」成長弧ではない：

「頂上到達の時だけ自由を感じる」極度の執着
→ 物語中盤で家族からの悲報を受けても山を下りない選択
→ フレッシャー風アニメーションの悪夢シーンで葛藤を表現
→ プレイヤーへの問い：「この目標達成後、この人物に何が残るのか？」

### 設計上の示唆

#### 不快なキャラクターを成立させる条件
1. **一貫性**: 行動・セリフ・ゲームプレイすべてがキャラクターの性格と矛盾しない
2. **サブテキストの深度**: 明示的説明ではなく、行間・選択・物理的表現で伝える
3. **プレイヤーへの問いかけ**: 感情移入の代わりに倫理的・感情的な問いを投げかける
4. **道徳的ニュアンス**: 成長物語の拒否が「不完全な設計」ではなく「意図的な選択」と伝わること

#### ゲームプレイとの統合
- アヴァの無愛想さは単なるキャラクター描写でなく、登山（孤独・過酷・上だけを向く行動）とメカニクスレベルで一致
- ゲームプレイそのものがキャラクターの心理状態を体験させる

### 汎用化ポイント

「気難しい主人公」の設計原則は特定IPに依存しない：
- **ゲームプレイとの一貫性**: キャラクターの心理状態をメカニクスで表現する
- **成長を拒否する弧**: プレイヤーに「このキャラクターは変わるべきか？」を考えさせることが目的
- **サブテキストの量と質**: 説明しすぎはキャラクターの複雑性を殺す
