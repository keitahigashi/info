## 心理的基盤

アチーブメントは明確な目標と測定可能な進捗を提供し、内発的動機に作用。構造化されたマイルストーン達成を通じて達成感を形成。

## 3層ティアアーキテクチャ

| ティア | タイミング | 目的 |
|--------|---------|------|
| **Starter** | 最初の30分以内 | 早期離脱防止、システム理解 |
| **Mid-tier** | 数時間のプレイ | スキル発達の要求、エンゲージメント深化 |
| **Mastery** | 長期プレイ | ベテランプレイヤー向け継続的目標 |

→ 早期放棄とバーンアウトの両方を適切な難易度カーブで防止。

## リテンションデータ

| 条件 | リテンション率 |
|------|-------------|
| Day 0 で初アチーブメント解除 | **56.9%** |
| Day 7 時点で未解除 | **8.5%** |

→ 初回アチーブメントの早期提供がリテンションに決定的。

## リテンション駆動メカニクス

- 反復的エンゲージメントループの創出
- 非金銭的プログレッションマーカーの提供
- 完了ストリークによる心理的投資の構築
- 可視性有効時のソーシャル比較機会

## 設計ベストプラクティス

| 原則 | 説明 |
|------|------|
| 目的の明確な伝達 | プレイヤーが何を目指すか理解できる |
| 努力に比例した報酬感 | 難易度と報酬の釣り合い |
| 可視的・追跡可能な進捗 | プログレッションバーやカウンター |
| 適切なタイミングの通知 | 正の行動を強化、侵入的にならない |

## アンチパターン

| 問題 | 影響 |
|------|------|
| 過度に複雑な階層 | プレイヤーの混乱 |
| 達成不能な目標 | フラストレーション |
| 難易度と無関係な報酬 | 価値感の喪失 |
| エンゲージメント指標優先 | プレイヤー満足度の犠牲 |

## 実装ガイドライン

- アクセシビリティ（新規プレイヤーの早期勝利）とアスピレーション（ベテランの長期目標）のバランス
- 即時かつ明確なフィードバック（ビジュアルエフェクト/サウンドキュー/アニメーション）
- バッジの希少性・難易度・関連性が価値を維持
- 過剰使用や目標無関係なバッジは効果を低下
