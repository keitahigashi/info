## ゲームリワードの階層構造

著者: Chelsey Webster (2013)

### フレームワーク: 5層のリワード階層

マズローの欲求段階説 (Maslow's Hierarchy of Needs) をゲームのリワード設計に応用した構造。**下の層が上の層の必須条件**となっており、任意の層が欠けるとそれより上の層は効果を発揮できない。

```
       ┌────────────────────────────────┐
       │  5. コスメティック・装飾的報酬   │  ← 最上層
       ├────────────────────────────────┤
       │  4. ショート・短期報酬          │
       ├────────────────────────────────┤
       │  3. メジャー・中期報酬          │
       ├────────────────────────────────┤
       │  2. コア・長期報酬              │
       ├────────────────────────────────┤
       │  1. プレイヤー体験（基盤）       │  ← 最下層
       └────────────────────────────────┘
```

### 各層の詳細

**第1層: プレイヤー体験（基盤）**
- ゲームプレイそのものの楽しさと没入感
- 「このゲームを遊ぶこと自体が楽しい」という根本的な体験
- **これがなければ他のすべてのリワードが無意味になる**
- 失敗事例: 装飾的リワードが充実していても基本的ゲームプレイが楽しくなければ継続されない

**第2層: コア・長期報酬**
- 永続的な大きな報酬（新ゲーム機能の解禁、大規模ストーリー展開）
- 数十時間のプレイを通じて達成される目標
- 例: メインストーリーのクリア、最強装備のコンプリート

**第3層: メジャー・中期報酬**
- レベルアップ、クエスト完了などの一時的だが重要な報酬
- 数時間〜数日のプレイスパンで達成される目標
- 例: ボス撃破、スキルツリーのブランチ解放

**第4層: ショート・短期報酬**
- 小さく頻繁な報酬（少額の経験値、ゴールドドロップ）
- 数分〜数十分のセッション内で達成される目標
- 例: 敵撃破時の経験値、宝箱の発見

**第5層: コスメティック・装飾的報酬**
- 視覚的進捗表示（バー、実績、マップ解禁、コスチューム）
- プレイヤーの進捗を「見える化」する要素
- 例: アチーブメント獲得、コレクション完成、外観変更

### 重要な原則

**依存性の原則**: 下層が満たされていない状態で上層に投資しても効果がない
```
× 装飾的報酬を豊富にしても → 基本体験が楽しくなければ離脱
× 長期目標を設定しても → 短期フィードバックがなければモチベーション維持困難
```

**設計優先順位**: 常に下の層から設計・改善する

### 成功事例: World of Warcraft

- 第1層: 戦闘・探索・社会的体験の根本的な楽しさ
- 第2〜4層: 分単位〜年単位の多層的な報酬設計
- 第5層: 実績・マウント・外観装備の豊富なコスメティック

**結果**: プレイヤーは常に「次に達成できる目標」を持ち続けた。

### 設計への提言

1. **第1層の妥協は致命的** — 「面白い体験」を最初に検証する
2. **層ごとに適切な密度を保つ** — 短期報酬が多すぎると価値が薄れる
3. **層間の時間スパンを意識する** — 各層が異なる頻度で達成されるよう設計する
4. **コスメティックは補完要素** — 実質的なゲームプレイ価値の代替にはならない
