## エージェンシーの心理学的基盤

**エージェンシー (Agency)** = 「コントロールの感覚 — 自分の行動が重要であるという感覚」

これは人間が本質的に持つ自律性への欲求 (自己決定理論 SDT の自律性欲求) を満たす。

> 「偽の選択はプレイヤーの信頼を即座に損なう」

本物の結果が伴わない選択はエージェンシー感を破壊する。

## 感情的投資の仕組み

選択が記憶に残る条件:
- **責任 (Responsibility)**: 自分が選択した結果への当事者感
- **喪失 (Loss)**: 何かを犠牲にした実感
- **犠牲 (Sacrifice)**: 価値あるものを捧げること
- **裏切り (Betrayal)**: 信頼関係への選択
- **救済 (Redemption)**: 過去の失敗を取り戻す機会

プレイヤーはキャラクターと世界に投資しているとき、選択を深く体験する。

## 選択アーキテクチャ (Choice Architecture)

見えないデザイン要素が選択を形成する — エージェンシーを排除せず、感情的解釈を微妙に誘導する:

| 要素 | 説明 |
|------|------|
| **提示方法** | 選択肢の見た目・順序・言葉遣い |
| **タイミング** | いつ選択が提示されるか |
| **フレーミング** | 選択がどのように文脈付けられるか |
| **コンテキスト** | 直前の出来事・感情状態 |

## 5種の選択タイプ

### 1. 道徳的選択 (Moral Decisions)
個人の倫理観とアイデンティティを探索。善悪の明確な答えがない真のジレンマ。

### 2. 関係的選択 (Relational Choices)
仲間の動態と社会的結果に影響。キャラクターとの絆の深さを規定。

### 3. 戦略的選択 (Strategic Selections)
効率または生存を優先。ゲームプレイ最適化とナラティブの交差点。

### 4. 表現的選択 (Expressive Options)
主要なプロット変更なしに個性を表現。世界観への参加感を高める。

### 5. マクロ選択 (Macro Choices)
全体的な進路を規定する大きな決断 vs. 色調を着色するミクロな決断。

## 5プレイヤーアーキタイプと動機

| アーキタイプ | 選択動機 |
|------------|---------|
| **ロールプレイヤー** | キャラクターのアイデンティティに基づいて選択 |
| **モラリスト** | 個人の倫理を優先 |
| **プラグマティスト** | 効率・生存を重視 |
| **エクスプローラー** | 新しいコンテンツ・結果を探索 |
| **カオスプレイヤー** | 予測不可能な選択を実験的に探索 |

→ 効果的なデザインは複数の動機パターンに対応する

## 結果の可視性 (Consequence Visibility)

結果の4種類:

1. **即時的 (Immediate)**: 選択直後に発生
2. **遅延的 (Delayed)**: 後のシーンで現れる
3. **隠された (Hidden)**: バックグラウンドで蓄積し後で明示
4. **感情的 (Emotional)**: 数値変化ではなくキャラクター関係の変化

## 構造的アプローチの比較

| 構造 | 特徴 | 適合するゲーム |
|------|------|--------------|
| **分岐ナラティブ** | 異なるパスで明確な選択結果 | RPG・アドベンチャー |
| **収束構造** | 類似した弧・感情的パーソナライゼーション | アクション・主流ゲーム |

> 「知覚されたエージェンシーは、分岐の複雑さよりも機械的に重要である」

プレイヤーが選択に重みを感じるかどうかは、実際の分岐数より設計の質と感情的文脈に依存する。

## バランスの設計課題

- **圧倒を避ける**: 選択肢が多すぎると決断麻痺が発生
- **制限を避ける**: 選択肢が少なすぎると自由感が失われる
- **明確さと自由の均衡**: プレイヤーが選択の重みを理解しつつも選択できる設計
