## 現実世界の散乱現象

| 現象 | 説明 |
|------|------|
| **空と空気遠近法 (Aerial Perspective)** | レイリー散乱により遠景が青みを帯びる |
| **雲** | 高濃度の水粒子が光を複雑に散乱、曇天下で柔らかい拡散照明 |
| **霧とヘイズ** | 視界を低下、地表近くの湿った場所で集中 |
| **ボリュメトリックライティング** | 粒子を光源が照らし「ゴッドレイ」/光の筋を形成 |

## ゲームでの散乱効果の目的

- リアリスティックな空のレンダリング
- 空気遠近法による**深度知覚**
- LOD/ストリーミングアーティファクトの隠蔽
- 雰囲気的ムードの確立
- 魔法/技術のVFX
- シーン構図とフォーカス誘導

## 物理的基盤: エネルギー保存モデル

光が媒質を通過する3プロセス:

| プロセス | 説明 |
|---------|------|
| **Transmittance（透過）** | 光が変化なしに通過 |
| **Scattering（散乱）** | 粒子により偏向 |
| **Absorption（吸収）** | 熱エネルギーに変換 |

3プロセスの合計 = 入射エネルギー総量。

## ゲーム向け簡略化

「ゲームでは計算を簡略化し、最も重要で可視的な部分に注力する」

**単一散乱近似 (Single Scattering)**: 光がカメラに到達する前に1回だけ散乱すると仮定。

### Out-Scattering（外散乱）

**Beer-Lambert法則**に従い、光の消衰は消衰係数（散乱＋吸収）に基づく指数関数的減衰:

```
T(d) = e^(-σ_e × d)
```

均一媒質では単一の指数関数で表現。

### In-Scattering（内散乱）

視線経路に入る光の計算。非均一密度媒質では**数値積分**が必要 → 計算負荷が高い。

## ボリュメトリックフォグアルゴリズム

Assassin's Creed 4 で開発された手法:

「近距離・中距離の光の単一散乱を物理ベースで効率的かつ統一的に再現」

- リアルタイムパフォーマンスと視覚的真正性のバランス
- シャドウマッピングに依存してボリューム内の影を定義

## レイマーチングアプローチ

1. カメラからボリュームを通してレイを発射
2. 一定間隔で「マーチ」しながらデータを蓄積
3. 霧が光の移動に与える影響を計算
4. 光の吸収・散乱・ソフトシャドウ・光の筋をシミュレート
