## 目的

現実世界の建築原理をゲームのレベルデザインに応用し、ゲーム空間と環境制作に活用する汎用的フレームワークを提供。建築・都市計画の何千年にもわたる知見を、プレイヤーの感情と体験を形成する空間設計に転用する。

## 7つの建築的レベルデザイン要素

| 要素 | 説明 |
|------|------|
| 視線誘導 (sightline guidance) | プレイヤー視点の操作、環境配置によるナビゲーション |
| ライティング (lighting) | 感情的トーン形成、ゲームプレイ情報の提供 |
| 影と薄暗さ (shadows and shade) | 心理的効果、ナラティブ強調 |
| 探索性 (explorativity) | プレイヤーの発見欲望、環境に隠された要素の配置 |
| 方向性 (directionality) | 空間内での移動方向性、プレイヤー心理的方向付け |
| 空間のリズム (spatial rhythm) | オープンスペース・制限スペースの交替、テンション・解放サイクル |
| 壮大な空間の演出 (grand spaces) | スケール感による感情喚起、クライマックス的配置 |
