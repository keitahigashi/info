## ジンバルロック問題

オイラー角の3回転軸のうち2つが整列 → 3自由度が2自由度に崩壊 → その構成を通過するスムーズな回転が不可能。

## クォータニオンの優位性

| 特性 | 説明 |
|------|------|
| ジンバルロック回避 | 軸の突然の整列が発生しない |
| スムーズ補間 | SLERPで方位間の最短経路補間 |
| 数値安定性 | 長時間アニメーションでも安定 |

## 数学的構造

**4Dハイパー複素数**: q = w + xi + yj + zk

| 部分 | 成分 |
|------|------|
| スカラー部 | w（実数部） |
| ベクトル部 | v = (x, y, z)（虚数部） |

### 基本性質

- **非可換**: pq ≠ qp（回転順序が重要）
- **結合則**: (pq)r = p(qr)（安定した合成）
- i² = j² = k² = ijk = -1

## 回転エンコード公式

軸 **u** 周りに角度 θ で回転する単位クォータニオン (|q|=1):

```
q = cos(θ/2) + sin(θ/2)(u_x·i + u_y·j + u_z·k)
```

**θ/2 の理由**: クォータニオン乗算による正しい回転合成を保証。

## SLERP（球面線形補間）

「クォータニオン間を最短4Dアーク上でスムーズに補間」

- 二重被覆特性: q と -q は同一3D回転を表現
- ドット積が負なら一方を否定して最短経路を保証
- **一定角速度**の最短大円弧パス

### Euler LERP との比較

| 手法 | パス | 速度 |
|------|------|------|
| SLERP | 最短大円弧 | **一定角速度** |
| Euler LERP | 曲がった非最適経路 | 不均一 |

## 歴史的背景

### Hamilton の発見（1843年）

William Rowan Hamilton が複素数の高次元拡張を試行。3次元では可換性が不可能と認識し、可換性を放棄して虚数単位 k（ij=k, ji=-k）を導入。

「数学史上初の複素数の高次元への成功した拡張」

初期は物理学（Maxwellの電磁気学）で使用→ベクトル解析に取って代わられた→固有の回転特性で復活。
