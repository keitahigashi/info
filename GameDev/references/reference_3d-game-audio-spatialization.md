## 3Dオーディオの定義

3次元空間の特定の位置から音が発生するような知覚を生む技術。ゲーム環境の真正性とエンゲージメントを向上。

## 主要技法

| 技法 | 説明 |
|------|------|
| **HRTF** | 頭部・耳・胴体との音の相互作用をシミュレート。FPSで特に有用 |
| **オクルージョン (Occlusion)** | 物理的障害物による音の減衰をシミュレート |
| **リバーブ & エコー** | 空間的深みを追加。リバーブ=閉鎖空間、エコー=オープンワールドの距離感 |
| **距離減衰 (Distance Attenuation)** | リスナーからの距離に比例して音源がフェード |

## 実装フレームワーク

3つの必須コンポーネント:
1. **オーディオリスナー**: プレイヤーの耳をシミュレート（通常プレイヤーキャラに付加）
2. **オーディオソース**: 空間化設定された音源オブジェクト
3. **設定調整**: 距離・減衰・ドップラー効果の微調整

## ミドルウェアツール

- **FMOD**: 複雑なアダプティブオーディオの作成
- **Wwise**: ダイナミック音楽と応答的サウンドエフェクト

## ジャンル別応用

| ジャンル | オーディオ戦略 |
|---------|-------------|
| **FPS** | 精密なローカリゼーション（敵の位置特定） |
| **サバイバルホラー** | 不穏なサウンドスケープで緊張感 |
| **オープンワールド** | 没入と規模感の確立 |
