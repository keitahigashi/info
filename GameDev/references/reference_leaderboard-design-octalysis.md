## 中心問題

従来のリーダーボードは「1人の幸福な勝者と9,900人の意気消沈した敗者」を生む。上位到達が非現実的だと、脳が努力の期待リターンを計算し、エンゲージメントが蒸発する。

## マイクロスプライシング: 6次元

単一グローバルランキングの代わりに、重複するリーダーボードを6次元で構築:

| 次元 | 説明 | 効果 |
|------|------|------|
| **時間ベース** | 日/週/月でリセット | 新たな勝利機会を懲罰感なく提供 |
| **アクティビティベース** | 異なる行動別ランキング | 多様なプレイスタイルに報酬 |
| **ティア/リーグベース** | スキルレベル内での競争 | グランドマスターとの不毛な競争を回避 |
| **地理的** | 地域別ランキング | 全国順位が到達可能に感じられる |
| **ソーシャルグラフ** | フレンドベースの競争 | 他人より知人との競争がエンゲージメント向上 |
| **チームベース** | グループ競争 | 達成感と帰属感を同時に活性化 |

→ 1人の支配的勝者ではなく、カテゴリ横断で複数の勝者を生成。

## 表示アーキテクチャ: 近傍ビュー

各ユーザーに「自分の順位 ± 5位」を表示:
- 心理的に到達可能な進行ギャップを創出
- グローバルトップ10表示 → 比較のドゥームループを誘発
- 近傍フォーマット → 次の順位が征服可能に感じられる

## スクリーンショットテスト

#1ランキングの自発的共有 = 成功した設計のシグナル。ユーザーが外部で成果を表示したがる時、リーダーボードが真のステータスシグナルとして機能。

## 時間的戦略

3つのリーダーボードを同時に階層化:

| 期間 | 活性化するCore Drive |
|------|---------------------|
| **週間** | CD2: 達成可能な達成感 |
| **月間** | CD7: 予測不可能性・好奇心 |
| **全期間** | CD5: レガシー・関連性 |

異なるプレイヤーペルソナに異なるタイムスケールが必要。

## 毒性とゲーミングの防止

| 対策 | 説明 |
|------|------|
| 3-5のプライマリボードに制限 | ランキング疲労の回避 |
| 複数メトリクスの重み付け | 単一指標のゲーミング防止 |
| チームランキング（集団主義文化） | 個人競争の毒性緩和 |
| 品質/多様性メトリクス | 量ではなく質の評価 |

## Octalysis統合

リーダーボードは複数のCore Driveを同時に活性化する時に機能:
- 純粋な競争フォーカス → 失敗
- 可視的進行 + 公正な競争 + 複数の達成パス → 持続的エンゲージメント
- 心理的帰属感と具体的達成感のハイブリッドがゼロサム競争を超える
