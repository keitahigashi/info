## 2026年UX/UIデザイン7つの潮流

### ビジュアルスタイル系トレンド

| 潮流 | 概要 |
|------|------|
| リキッドグラスデザイン (liquid glass) | 半透明ガラス質感。光の屈折・透過をUIに取り込む |
| ベントボックスレイアウト (bento box layout) | 格子状モジュール構造。情報密度を保ちながら整理された構造を実現 |
| 3Dマテリアルデザイン | 金属・粘土・布など実素材の質感をUI要素に適用 |
| インタラクティブスクロール | スクロール操作で物語が展開するストーリードリブン体験 |

### 機能・体験系トレンド

| 潮流 | 概要 |
|------|------|
| AI環境連動型パーソナライズ | 時間帯・行動パターン・文脈に応じてUIが自動適応 |
| AIネイティブインターフェース | チャット型対話でUIを動的生成する新形式 |
| アクセシビリティ優先設計 | インクルーシブデザインが設計の出発点に（後付けオプションではなく） |

### ゲームUIへの応用

- **リキッドガラス**: SF・ファンタジーゲームのHUD・メニュースクリーンに。空間的透明感とレイヤー奥行きを同時に表現
- **ベントボックス**: インベントリ・スキルツリー・ビルドメニューの情報整理に。モジュール設計で拡張しやすい
- **3Dマテリアル**: ボタン・アイコン・カードのビジュアルに質感を持たせ、タップ感・操作感を視覚的に強化
- **インタラクティブスクロール**: ゲームのロードスクリーン・回想シーン・地図ナビゲーションへの応用
- **アクセシビリティ優先**: ゲームUI設計でも「後から追加」から「最初から組み込む」へのパラダイムシフトを示す
