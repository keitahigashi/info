## 概要

ポーランドのインディーゲーム『CARRION』（Phobia Game Studio開発）は、異星生物となって研究施設から脱出するリバースホラー。レトロ調ドット絵に対してAAA級サウンドデザインを施すという意図的な乖離が、独自の没入感を生み出している。

## コア設計原則: ビジュアルとオーディオの意図的ミスマッチ

- レトロ調ドット絵グラフィックスに対し、「サウンドデザインは間違いなくAAAの雰囲気が強い」
- スター・ウォーズの伝統的オーケストラサウンドと同様の**期待の逆転 (expectation reversal)** 手法
- ビジュアルが制約された分、オーディオが世界観を拡張する役割を担う

## フォーリーサウンド (Foley) の活用

### 自作録音へのこだわり
外部サウンドライブラリではなく、チーム自作の録音を多用:

| 対象 | 素材 |
|------|------|
| 触手の動き | USBケーブルの牽引音・風切り音を変形加工 |
| 破壊音 | ガラス・果物の破壊音 |
| 生物的な音 | 人間・動物の声を加工 |

### チーム体制
- 「チームの半分がサウンド作業に専念」するほどの投資規模
- 創作過程自体がチーム活性化・モチベーション維持に寄与

## 技術実装

### FMODによるインタラクティブサウンド
- 数十種の効果音をレイヤー登録
- ゲーム進行に伴うインタラクティブな再生制御を実現
- プレイヤーの行動に応じた動的サウンド変化

## 映画的インスピレーション

| 作品 | 影響 |
|------|------|
| **エイリアン** (1979) | ミニマルシンフォニックサウンド |
| **遊星からの物体X** (1982) | クリーチャー的不気味さ |
| **ペンデレツキ** (作曲家) | 現代的シンセサウンド |

## 汎用化ポイント

1. **意図的ミスマッチ戦略**: ビジュアルスタイルとオーディオフィデリティの意図的な差異が、プレイヤーの期待を裏切り没入感を強化する汎用テクニック
2. **自作フォーリーの価値**: 既存ライブラリではなく日常物からの音作りが、ゲーム固有のサウンドアイデンティティを確立する
3. **レイヤードサウンド設計**: ミドルウェア（FMOD/Wwise）を用いた複数音源のレイヤリングとインタラクティブ再生が、リアクティブなサウンドスケープの基盤となる
4. **映画音響の参照**: 映画のサウンドデザイン哲学からの学びが、ゲーム特有のインタラクティブ性と融合することで独自の体験を生む
