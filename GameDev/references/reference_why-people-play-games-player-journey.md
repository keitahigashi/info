## プレイ動機の時間的変化分析

### 核心的洞察

「ゲームをプレイし続ける動機は、プレイを開始した動機と異なり、時間とともに変化する」

初期の動機（新鮮さ・好奇心）が満たされた後、プレイヤーが継続するには異なる種類の充足が必要になる。この変化を設計に組み込むことがリテンションの鍵となる。

### Big 5 心理学モデルのゲームデザイン適用

**O.C.E.A.N.（開放性・誠実性・外向性・協調性・神経症傾向）** をゲームデザインの文脈で再解釈する：

| Big 5 特性 | ゲームでの現れ方 | 対応する設計要素 |
|-----------|---------------|---------------|
| **開放性 (Openness)** | 新しい体験・発見の探求 | 探索・謎解き・新メカニクスの段階的導入 |
| **誠実性 (Conscientiousness)** | 目標達成・完了欲求 | コレクション・実績・進捗の可視化 |
| **外向性 (Extraversion)** | 社会的インタラクション | マルチプレイ・ギルド・ランキング |
| **協調性 (Agreeableness)** | 協力・貢献 | Co-opモード・ギルドへの貢献・ギフト |
| **神経症傾向 (Neuroticism)** | リスク管理・安全確保 | セーブポイント・保険的メカニクス |

バートル・MDA・オクタリシスと組み合わせることで、より精密なプレイヤーペルソナを構築できる。

### プレイヤージャーニー (Player Journey) の4段階

```
Discovery（発見）
  └─ 動機: 好奇心・新鮮さ
  └─ 設計: フック・ティザー・直感的な初体験

Evaluation（評価）
  └─ 動機: 「このゲームは自分のためか？」
  └─ 設計: コアループの明確な提示・障壁の最小化

Use（活用）
  └─ 動機: スキル向上・深みの探求・習慣化
  └─ 設計: 難易度曲線・マスタリーシステム・コンテンツの広がり

Affinity（親近感）
  └─ 動機: アイデンティティとの結合・コミュニティ
  └─ 設計: カスタマイズ・ファン文化サポート・長期的な関与
```

### 精密な共感 (Precise Empathy) の重要性

「ターゲットプレイヤーを明確にすること」が設計判断の質を高める。

- 漠然とした「すべてのプレイヤー」への設計 → 誰にも刺さらないリスク
- 具体的なプレイヤーペルソナへの設計 → 明確な意思決定・優先度付けが可能

**Big 5を使ったペルソナ構築手順:**
1. ターゲットとなるプレイヤー像のBig 5プロファイルを仮定
2. プレイヤージャーニーの各段階での充足欲求を特定
3. 各段階のゲームデザイン要素に紐付ける
4. プレイテストで仮定を検証・更新

### 自己決定理論 (SDT) との統合

ジャーニーの各段階において、SDTの3軸が異なる重要性を持つ：

| ジャーニー段階 | 主要なSDT要素 |
|-------------|-------------|
| Discovery | 自律性（自分が選んでいる感覚） |
| Evaluation | 有能感（「できる」と感じさせる） |
| Use | 有能感 + 自律性（成長と自由） |
| Affinity | 関係性（コミュニティへの帰属） |
