## PBRの定義

光の反射・散乱・屈折・吸収など物理現象としての光学現象を計測し、厳密な数式でモデル化したレンダリング手法。従来のLambert/Phongシェーディングとの主な違いは、より厳密な物理計算に基づいている点。

## PBRの3主要パラメータ

### 1. アルベド (Albedo)
- 受けた光をどれだけ反射するかを表す係数
- 物体の「明るさそのもの」ではなく、外部光に対する反射能
- 0〜1の範囲で、反射光は受光強度に比例

### 2. 反射率 (Metallic / Reflectivity)
- 物質のスペキュラー反射特性を制御
- 金属: 反射率が高い
- プラスチック・木材: 反射率が低い

### 3. ラフネス (Roughness)
- 表面粗さを0〜100%で表現
- 0%: つるつる（艶あり、鏡面的）
- 100%: ざらざら（マット、拡散的）

## マイクロファセット理論

一見つるつるに見える表面も、電子顕微鏡レベルでは微細な凹凸がある。この微細表面（マイクロファセット）が光をあらゆる方向に反射させることで、視覚的な質感が生まれる。

- ラフネスが低い → マイクロファセットが揃う → 鋭い鏡面反射
- ラフネスが高い → マイクロファセットがランダム → 拡散的反射

## 拡散反射と鏡面反射

### 拡散反射 (Diffuse Reflection)
- 粗い表面からの光反射（微細表面での乱反射）
- アルベドによって色味が反映される
- 従来はランバート反射モデルで近似

### 鏡面反射 (Specular Reflection)
- 鏡面的な光の反射（入射角=反射角）
- **非金属**: 反射光はアルベドの影響を受けない（白色反射）
- **金属**: 反射光に表面色がつく（色付き反射）

## Cook-Torranceモデル

```
f(l, v) = D(h) * F(v, h) * G(l, v, h) / (4 * (n·l) * (n·v))
```

| 項 | 名称 | 役割 |
|----|------|------|
| D | 法線分布関数 (NDF) | マイクロファセットの配向分布 |
| F | フレネル項 | 視線角度による反射率変化 |
| G | 幾何減衰項 | マイクロファセットの相互遮蔽 |

## NDF: Blinn-Phong vs GGX (Trowbridge-Reitz)

### Blinn-Phong NDF
```
D_Blinn(m) = (α + 2) / (2π) * (n·m)^α
```

### Trowbridge-Reitz (GGX)
```
D_GGX(m) = α² / (π * ((n·m)² * (α² - 1) + 1)²)
```

（α = ラフネスの2乗）

### 比較結果
- GGXはハイライト部分でより自然な見た目を生成
- GGXはロングテール（長い裾野）を持ち、エッジ付近で微妙な反射が残る
- 現在のリアルタイムPBRではGGXが主流

## ゲーム開発への実践的視点

### アーティスティック調整の重要性
- シミュレーションで完全にリアルにすると「かえって平凡」になる可能性がある
- 物理計算に基づきながらもアーティスティック調整を加えるアプローチを推奨
- PBRは「制約の中の自由度」を提供するフレームワーク

### 実装上の利点
- 照明環境が変わっても一貫した質感を維持
- アーティストが直感的に理解しやすいパラメータ（金属度・粗さ）
- テクスチャ制作のワークフローが標準化される
