## 第一印象の重要性

「ゲームのアートスタイルは誰かの注意を引く最初の — そして時には唯一の — 機会」。ビジュアルがゲームプレイ前に購入・ダウンロード・ウィッシュリスト登録の判断を決定する。

## アートディレクションの6原則

| 原則 | 説明 |
|------|------|
| **ゲームプレイ整合性 (Gameplay Alignment)** | ビジュアルスタイルがメカニクスを支援し妨げない |
| **ビジュアル一貫性 (Visual Consistency)** | キャラ・環境・UI全体で一貫した美学が没入感を維持 |
| **ジャンル-トーン一致 (Genre-Tone Matching)** | アートがゲームのカテゴリと感情的レジスターを補完 |
| **プレイヤー明確性 (Player Clarity)** | デザインが直感的なナビゲーションと理解を促進 |
| **クロスプラットフォーム適応 (Cross-Platform Adaptability)** | スタイルがデバイス間で効果的にスケール |
| **感情的共鳴 (Emotional Resonance)** | 色・アニメーション・雰囲気で意味のあるプレイヤー接続 |

## 一貫した美的アイデンティティ

「不整合なビジュアルはプレイヤーを体験から引き離す」。一貫性が信じられる世界の鍵。成功する一貫性はゲームプレイ・ストーリー・UXを同時に支える意図的な選択が必要。

## スタイル選択の持続性

スタイライズドな選択（セルシェーディング、ローポリ）はリアリスティックなアプローチより長寿命の利点がある。11のアートスタイル（ピクセルアート、手描き2D、リアリスティック3D、ボクセル等）がカバーされ、各々が異なるナラティブトーン・制作予算・ターゲットプラットフォームに対応。
