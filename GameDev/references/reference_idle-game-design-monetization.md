## 設計の核心原則

「ゲームデザインの中心にエンゲージメントを置くことが、プレイヤーを長期間ゲームと対話させ続ける最も確実な方法」

放置ゲームは長いプレイヤーライフスパンで知られていないが、思慮深い設計でリテンションを大幅に延長可能。

## マネタイゼーション4階層

| 優先度 | 収益源 | 説明 |
|--------|--------|------|
| **1（最高）** | Rewarded Ads（リワード広告） | 広告視聴でゲーム内報酬。支配的収益ストリーム |
| 2 | Interstitial Ads（インタースティシャル） | ゲームプレイセッション間のフルスクリーン広告 |
| 3 | In-App Purchases（IAP） | 直接支出機会 |
| 4（最低） | Banner Ads（バナー広告） | 最も効果の低い収益源 |

## マネタイゼーション組合せルール

- インタースティシャルとバナー広告の**併用を避ける**
- バナー広告/インタースティシャルはIAP（広告除去）と組合せ可能
- リワード広告は**全マネタイゼーションタイプと効果的に併用**
- 思慮深い報酬戦略の実装が前提

## 成功要因

AdVenture Capitalist、Idle Miner Tycoon 等の成功要因:

| 要因 | 説明 |
|------|------|
| トレンディ/ユニークな設定 | プレイヤーの好奇心を引く世界観 |
| 魅力的なビジュアル | 見た目のクオリティ |
| 強力なASO | アプリストア最適化 |
| バランスの取れたゲームプレイ | メカニクスの均衡 |
| 知的なマネタイゼーション | プレイヤー体験との調和 |

## オフライン進行の設計

放置ゲームの主要メカニクスの一つ。リテンション向上に不可欠。

- オフラインプレイに**制限（例: 2時間上限）** を設ける作品もあり
- アイドル報酬への依存を最小化
- ハードリミットの設定が重要（無制限は経済バランス崩壊）

## プレステージ数式

標準プレステージ公式は指数**0.5〜0.8**を使用:
- **逓減収益**を生成
- 複数プレステージサイクルを促進
- リセットごとのパワー増加が適切に管理される
