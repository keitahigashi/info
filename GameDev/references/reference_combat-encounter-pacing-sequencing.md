## 定義：ペーシングとシーケンシングの違い

- **ペーシング (pacing)**（科学的側面）: 時間に対する強度 (intensity) の流れ
- **シーケンシング (sequencing)**（芸術的側面）: 複数要素を組み合わせて戦闘を構成する創造的プロセス
- **強度 (intensity)**: プレイヤーが現在の行動を中断して新しい行動を選ぶ頻度

## 戦闘計画ピラミッド

プレイヤーが戦闘中に取れる3つの行動：

1. **Assess（評価）**: 状況判断
2. **Attack（攻撃）**: 敵への攻撃
3. **Move（移動）**: 位置変更

## ペーシングの黄金パターン（5段階）

| ステージ | 説明 | 強度 |
|---------|------|------|
| Introduce（導入） | 敵を登場させる | 低 |
| Initiate（開始） | 戦闘開始 | 中 |
| Escalate（強化） | 敵の増加・強化 | 高 |
| Midpoint（中間） | 息抜きの機会 | 中 |
| Climax（クライマックス） | 最大強度で終幕 | 極高 |

> 「優れた戦闘には流れがある。強く始まるが段階的に盛り上がり、最後にクレッシェンドに達する」— Mike Birk Head (God of War)

## シーケンシングの6要素

### 1. 敵タイプ要素 (Enemy Type Factor)
- 異なるAIアーキタイプを組み合わせる（例：銃撃手＋近接型）
- 複数距離で同時に脅威を創出
- 同時登場タイプ数は制限すること

### 2. 敵数要素 (Enemy Count Factor)
強度調整の最もシンプルなノブ（つまみ）。2つのパターン：

**トライアングルパターン**
- 敵数を段階的に増加
- 敵タイプは変わらない（最も一般的）

**ダイアモンドパターン**
- 前半：敵数増加 → 後半：敵数減少＋敵の種類・強度を向上
- ボス戦などに適用

### 3. スポーンタイミング要素 (Spawn Timing Factor)
- 早すぎ：プレイヤーがパターンを認識できない
- 遅すぎ：退屈になる
- 比喩：「暴力的なダンスの振り付け」

### 4. スポーン位置要素 (Spawn Location Factor)
人間の視線の動きに基づいた配置戦略：
- 多くの人は左から右へ読む
- 第1スポーン：左上から開始し、視線の円形軌跡（4隅を通る）に沿って配置
- 効果：画面全体での注視移動を促進し、スポーンアニメーションが自然に見える

### 5. レベルイベント要素 (Level Event Factor)
スクリプト化された環境変化：
- 例：Uncharted 3の客船シーンで水位が上昇し、プレイ可能エリアが縮小
- 敵数を減らしても強度を維持可能
- プレイヤーに再計画を強制し、自然な状況変化でペーシングを調整

### 6. ダイアログ要素 (Dialogue Factor)
- 敵AIが戦闘変化を事前告知し、プレイヤーの反応時間を確保
- 低コストで強度調整が可能

## 追加要素：音楽 (Music Factor)

- スケール感と劇的性を付与
- プレイヤーの感情レベルを上昇
- 脳が認識・学習する戦闘パターンの追加層を形成

## プレイテスト観察ポイント

照準 (crosshair) の動きを監視：
- 動きが少ない → 強度不足
- 動きが多くフラストレーション → ペース調整が必要
- バランス点：画面全体での適度な照準移動

## 結論

> 「すべての要素を組み合わせることがシーケンシングの芸術であり、発生タイミングがペーシングの科学である」

ペーシングとシーケンシングは分離されたものではなく、相互依存する連関システム。
