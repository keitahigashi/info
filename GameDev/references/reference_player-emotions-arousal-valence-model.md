## プレイヤー感情の心理学的フレームワーク（psychological framework）

**テーマ**: 心理学的感情モデルのゲームデザイン応用と感情設計の実践的体系

### 基本フレームワーク

**ラッセルの感情円環モデル（Russell's Circumplex Model of Affect）**
- 覚醒度（arousal）高/低 × 感情価（valence）快/不快 の2軸で感情を分類
- プレイヤー感情をマッピングして誘導目標を設定する設計ツールとして機能
- 4象限の例：高覚醒×快=興奮、高覚醒×不快=怒り、低覚醒×快=満足、低覚醒×不快=倦怠

**ヤーキーズ・ドットソン法則（Yerkes-Dodson Law）**
- 易タスク（easy task）は高覚醒が有利
- 難タスク（difficult task）は低覚醒が有利
- 難易度曲線（difficulty curve）設計に直接適用可能：高難度場面の前に覚醒を抑える設計

### 感情がゲーム体験にもたらす6つの恩恵

1. 記憶形成（memory consolidation）の強化
2. ポジティブムードによる問題解決力の向上
3. 覚醒による重要場面への注意集中
4. パフォーマンス（performance）の最適化
5. モチベーション（motivation）の持続
6. 怒り・罪悪感による目標行動の誘発

### ハロー効果（halo effect）の実践的活用

- 「ユーレカ体験（eureka moment）」の喜びが試行錯誤の苦痛を上書きする（例：Antichamber）
- 「良いフラストレーション（good frustration）」と「悪いフラストレーション（bad frustration）」の設計上の峻別
- ポジティブ感情のキャリーオーバー（carryover）効果を次のセクション設計に活用

### 感情測定手法（measurement methods）

- **PANAS**（Positive and Negative Affect Schedule）：ゲーム後のアンケート形式感情スケール
- **FACS**（Facial Action Coding System）：表情分析による非侵襲的感情測定
- **実践的応用**: 操作性・UI問題など「悪い感情」の除去がリテンション（retention）向上に直結
- 感情設計はゲームプレイ設計と分離できない：システムが感情を生む
