## 伏線の定義と役割

「後の物語で起こる出来事を暗示する文学的手法」。ゲームナラティブでの役割:
- 物語の継続性と一貫性の創出
- サスペンス・緊張感の構築
- プレイヤーのエンゲージメント向上
- 没入感のある体験の実現

## 伏線手法の3カテゴリ

### 1. 微妙なヒントと提案
| 手法 | 説明 |
|------|------|
| 対話を通じた示唆 | キャラクターが後に重要となる出来事や人物に言及 |
| アイテム説明 | 将来の重要性を暗示する記述 |
| 環境的手がかり | 謎のシンボルや異常なオブジェクト配置 |

### 2. 視覚・音響的手法
- ライティングと影の活用 → シニスターな雰囲気の創出
- サウンドエフェクト → 不安感や緊張の演出
- カメラアングルの工夫 → 不穏さの表現

### 3. 環境設定とセットアップ
- 環境ストーリーテリング（壁画、落書きなどの物語的表現）
- セットアップとペイオフの構造（早期設定 → 後の回収）

## 高度な伏線技法

### ミスディレクションとダミー手がかり
- 虚偽の手がかりでプレイヤーの推測を別方向へ誘導
- 注意散漫の仕掛けで真の物語からの乖離を演出
- 「赤いニシン」的要素で推理を攪乱

### プレイヤー心理の活用
- プレイヤーの期待を逆用する展開
- 恐怖や不安の心理的活用
- ジャンル慣例の裏切り

### 伏線とサプライズのバランス
「事後的には気づけるが、当初は隠れている」— 微妙さと明白さの最適バランス。

## ゲーム事例

| ゲーム | 伏線手法 |
|--------|---------|
| **ゼルダの伝説** | 環境線索とアイテム説明による間接的暗示 |
| **The Last of Us** | 微妙な視覚・音響的手がかりの積み重ね |
| **Portal** | ライティングと環境設定で予期しない結末への準備 |

## ベストプラクティス

| 原則 | 説明 |
|------|------|
| 過度な明白性の回避 | 重い手法や明らかな伏線は避ける |
| 一貫性と整合性 | 物語世界との整合性維持が必須 |
| プレイテスト検証 | 多様なプレイヤーからのフィードバックで反復改善 |
