## OCEANモデルのゲームデザインへの応用

Ubisoft の Jason VandenBerghe が GDC 2012 で発表。心理学の「ビッグ5性格特性 (Big Five / OCEAN)」をプレイヤー動機の分析フレームワークとして再解釈した。

### OCEAN → ゲーム動機の変換

| 性格特性 | ゲーム動機 | デザイン含意 |
|---------|-----------|------------|
| **開放性 (Openness)** | 新規性 (Novelty) | 探索・発見・未知の体験 |
| **誠実性 (Conscientiousness)** | チャレンジ (Challenge) | 達成・習熟・目標設定 |
| **外向性 (Extroversion)** | 刺激 (Stimulation) | スピード・アクション・興奮 |
| **協調性 (Agreeableness)** | 調和 (Harmony) | 協力・コミュニティ・共感 |
| **神経症傾向 (Neuroticism)** | 脅威 (Threat) | 恐怖・プレッシャー・生存 |

### 6つの側面への細分化

各特性は6側面に分解され、計30の動機軸が生まれる。例えば「開放性」は以下の側面を持つ:
- 想像力 / 芸術的関心 / 冒険心 / 感情豊かさ / 知的好奇心 / 価値の柔軟性

この細粒度の分析により「プレイヤーは〇〇を望む」という一枚岩の議論を、「セグメントAは〇〇を求め、セグメントBは××を求める」という具体的な対話に転換できる。

## 核心知見

> "プレイは人生と同じ理由で行われる（Play is done for the same reasons as life）"

日常生活では満たせない動機を充足する手段としてゲームを位置づける。これにより:
- 同じゲームで正反対の動機を持つプレイヤーが共存できる理由が説明できる
- 「広いプレイヤー層へリーチする」ために**各側面の両端（高・低）に訴求**する設計の重要性が見えてくる

## 実践的示唆

- **デザイン議論の客観化**: 「俺はこれが面白い/つまらない」という感情論から、「この機能は誠実性高×外向性低セグメントに訴求する」という構造的分析へ
- **ターゲット設定の明示化**: 主要ターゲットのOCEANプロファイルを定義してからメカニクスを設計する
- **多様性の担保**: 単一特性に最適化するとニッチになる。複数特性への訴求がIPの広がりを生む
