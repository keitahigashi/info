## 定義と目的

ポストプロセスは「メイン3Dレンダリングシーンが描画された後、フレームごとに適用されるビジュアルエフェクトのセット」。画面空間操作でワールドジオメトリを変更せず、雰囲気と映画的品質を向上させる仕上げ処理。

## レンダリングパイプライン統合

ジオメトリ・ライティング・シェーディングパスの後に配置。エフェクトはチェーンで動作し、前のエフェクトの出力が次のエフェクトの入力となるため、**順序が最終的な外見に重大な影響**を与える。

## コアビジュアルエフェクト

### 輝度・光エフェクト

| エフェクト | 説明 | 注意点 |
|-----------|------|--------|
| **ブルーム (Bloom)** | 光がソースを超えて滲む効果。ハロー生成 | 過度にすると細部が消失 |
| **レンズフレア** | カメラ内での光散乱を模倣。ハイコントラスト時のドラマ | — |

### 色彩・トーン

| エフェクト | 説明 |
|-----------|------|
| **トーンマッピング** | HDR画像を表示可能範囲（LDR）に変換。リアルな明るさ再現 |
| **カラーグレーディング** | 色相・彩度・コントラスト調整でムード確立（冷たいSF / 暖かいアドベンチャー） |

### 深度・フォーカス

| エフェクト | 説明 |
|-----------|------|
| **被写界深度 (DoF)** | 焦点面外のオブジェクトを選択的にブラー。注意誘導と映画的分離 |
| **アンビエントオクルージョン (SSAO)** | 隙間や角にシェーディング追加。追加ジオメトリなしで深度知覚を向上 |

### 動き・スムージング

| エフェクト | 説明 |
|-----------|------|
| **モーションブラー** | 移動オブジェクトにトレイル。速度感を伝達。高速アクションでは細部がスメア |
| **アンチエイリアシング (AA)** | TAA/FXAA等。ジャギーを平滑化。明瞭さとテクスチャ保持のバランス |

### スタイリスティック

| エフェクト | 説明 |
|-----------|------|
| **フィルムグレイン** | 伝統的フィルムストックを模倣するテクスチャ |
| **ビネット** | 画像周辺を暗くし中央に注意を集中 |
| **色収差** | レンズ分散を模倣する微妙な色ずれ |

## パフォーマンス最適化

- 高コストエフェクトのLODスケーリング
- パフォーマンスヘッドルームに基づくアダプティブ品質
- 大画面向けハーフ解像度/タイルドレンダリング
- プラットフォーム別の調整（コンソール/モバイル）

## ジャンル別応用

| ジャンル | 戦略 |
|---------|------|
| **FPS** | DoF・モーションブラーを最小化し、敵の明瞭性と武器レスポンスを保持 |
| **RPG/アドベンチャー** | 攻撃的なカラーグレーディングとブルームでムード確立。インベントリ/ダイアログの可読性維持 |
| **レース/アクション** | モーションブラーで速度感を強調。HUD隠蔽を防止 |
| **インディー/スタイライズド** | ブルームやカラーシフトを誇張し独自の芸術的アイデンティティを確立 |

## プレイヤーアクセシビリティ

- 個別エフェクトの粒度あるトグル・プリセットを提供
- モーションブラーの軽減オプション（感受性への配慮）
- ハイコントラスト設定・明るさ調整
- 全美的設定で読みやすいUIを維持
