# マムシとヤスリ

## あらすじ

一匹のマムシが鍛冶屋の仕事場に入り込み、そこにあった道具を舐め回し始めた。やがてヤスリを見つけると、それに噛みついた。するとヤスリは少しも傷つかず、逆にマムシの舌は血だらけになった。

マムシはますます怒り、ヤスリを相手に格闘を挑んだが、噛めば噛むほど自分の口が裂けていくばかりで、ヤスリには何の変化もなかった。

## 教訓

自分よりはるかに硬い相手、あるいは傷つけようのない相手に攻撃を仕掛けても、損害を受けるのは自分だけである。怒りに任せて無謀な戦いを挑むことは、敵を傷つけるどころか自らを滅ぼす愚行にほかならない。戦う相手を見極める冷静さが必要である。
