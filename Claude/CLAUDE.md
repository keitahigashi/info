# CLAUDE.md

## このリポジトリについて

Claude Code CLIのナレッジベース。記事・公式情報を収集・整理し、実践的なガイドとして蓄積する。

## 絶対ルール（必ず守ること）

- 収集済みURLへの重複追加は禁止（`REFERENCES.md` で重複チェック）
- 1回の実行で最大5記事まで
- ナレッジベースファイル（`00-11`）への追記は所定フォーマットに従う
- ログは必ず `log/{YYYY-MM-DD}.md` に作成する

## Skillsの使い方

タスク開始前に必ず該当するSKILL.mdを読んでから作業すること。

| やること | 参照するSKILL.md |
|----------|-----------------|
| 記事収集・自動収集ワークフロー | `skills/article-collector/SKILL.md` |

## ファイル構成（参照用）

| パス | 役割 |
|------|------|
| `00-11` のナンバリングファイル | ナレッジベース本体 |
| `README.md` | 目次・システム概要 |
| `REFERENCES.md` | 収集済み記事テーブル（重複チェックの単一ソース） |
| `references/` | 記事詳細ファイル（Git管理） |
| `log/` | デイリー収集ログ |
| `skills/article-collector/` | 記事収集ワークフロー定義 |
