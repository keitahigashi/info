## 概要

### SpaceXとの提携内容

- **対象施設**: SpaceXが展開するAIスーパーコンピューター「Colossus 1」
- **容量**: 300メガワット超の演算能力を5月中に利用可能に
- **GPU構成**: NVIDIA H100/H200/GB200など計22万基超を搭載

### Claude Code制限の変更

- 5時間あたりのレート制限を**2倍**に引き上げ
- ピーク時間帯の制限低下を撤廃

### Claude API制限の緩和

- 各ティアで入出力トークン数が大幅に増加
- Anthropicが同時に発表した変更点として記載
