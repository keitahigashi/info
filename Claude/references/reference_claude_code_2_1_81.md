## アップデート内容（2026-03-20リリース）

### 1. --bareフラグ追加
スクリプト実行時のオーバーヘッドを軽量化。開発効率向上。

### 2. --channels導入
モバイルデバイス経由での権限承認が可能に。多様なデバイスからのアクセスに対応。

### 3. OAuth安定性向上
外部サービス連携時の信頼性が強化。より安全に利用可能に。
