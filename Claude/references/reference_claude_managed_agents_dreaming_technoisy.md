## 概要

2026年5月6日公開。Code with Claudeカンファレンスで発表されたClaude Managed Agentsの3つの新機能について、技術仕様と実装事例の数値データを詳述。

## 新機能一覧

| 機能 | ステータス |
|------|----------|
| Dreaming（ドリーミング） | リサーチプレビュー |
| Outcomes（アウトカムズ） | パブリックベータ |
| Multiagent Orchestration | パブリックベータ |

## 1. Dreaming（ドリーミング）

**定義**: エージェントが過去セッションを振り返り、記憶を整理・更新するスケジュール実行プロセス

### 機能詳細

- 重複エントリの統合
- 古い情報・矛盾データの更新
- パターン発見（反復ミス、共通ワークフロー）

### 技術仕様

| 項目 | 詳細 |
|------|------|
| 実行方式 | 非同期ジョブ |
| 処理時間 | 数分〜数十分 |
| 対応モデル | claude-opus-4-7、claude-sonnet-4-6 |
| 最大入力 | 100セッション/実行 |

## 2. Outcomes（アウトカムズ）

**概念**: 開発者がルーブリック（評価基準）を定義し、独立したグレーダーが客観的に判定

### 評価対応範囲

- 客観的仕様（ファイル形式、必須項目）
- 主観的基準（ブランドボイス適合、ビジュアルガイドライン遵守）

### 実績

社内ベンチマークで「タスク成功率が**最大10ポイント向上**」

| ファイル形式 | 改善率 |
|-------------|--------|
| docx | +8.4% |
| pptx | +10.1% |

## 3. Multiagent Orchestration

**概要**: リードエージェントが複数スペシャリストエージェントに並列委任

- 共有ファイルシステム上で**並列実行**
- 各エージェントが作業履歴を保持
- Claude Consoleで完全なトレース可能
