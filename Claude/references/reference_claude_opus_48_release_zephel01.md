## Claude Opus 4.8概要

### リリース日
2026年5月28日

### 位置づけ
Opus 4.7からの据え置き価格での機能強化版。「実用性・信頼性強化」にフォーカスしたアップデート。

## 主要な改善点

### 正直さの向上
- コードの欠陥を見逃す確率が**Opus 4.7の約1/4に低下**
- 不確実な点を自ら「不確実」と表明するようになった
- 「確証のない報告」問題が大幅に改善

### 判断力の強化
- エージェント作業での精度が向上
- 自律コーディングで複雑なタスクの完了率が改善
- 早期テスターの評価：「適切な質問をし、自分のミスに気づき、計画がおかしいときは押し返す」

### ブラウザ操作性
- ウェブ操作ベンチマークで**84%**を記録
- 現時点で最強のコンピューター操作モデルと評価

## 新機能

| 機能 | 概要 |
|------|------|
| Dynamic Workflows | 数百のサブエージェントの並列実行に対応 |
| Effort Control | 思考の深さを調整するコントロール追加 |
| システムプロンプト途中更新 | エージェント実行中の動的更新が可能 |

## 価格変更
- Fast Mode: **旧比1/3**に値下げ（$30/$150 → $10/$50）
- 通常版Opus 4.8とFast Modeの価格差は2倍
