2026年5月6日（米国時間）にサンフランシスコで開催された「Code w/ Claude 2026」開発者会議のライブブログ。AI開発者・研究者のSimon Willisonによる詳細な技術レポート。

- 登壇者: Ami Vora（最高製品責任者）

## 主要発表内容

### インフラ拡張

- **SpaceXのColossusデータセンター**（メンフィス）とのパートナーシップを発表
- 300MW超・NVIDIA GPU 22万基以上のコンピューティング容量を確保

### Claude Managed Agentsの新機能

| 機能 | ステータス | 概要 |
|------|-----------|------|
| Dreaming | リサーチプレビュー | 過去セッションを振り返り自己改善を実行 |
| Outcomes | パブリックベータ | 成功の定義を自動反復評価 |
| Multiagent Orchestration | パブリックベータ | リードエージェントが複数専門エージェントに並列委任 |

### Claude Code の機能拡張

- デスクトップアプリケーション対応
- **コードレビュー自動化**機能（全社展開）
- **CI自動修正**機能
- **ルーチン機能**: 非同期自動化タスクのクラウド設定

## 企業採用事例
