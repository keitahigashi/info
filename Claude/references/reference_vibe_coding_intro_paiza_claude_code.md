## 講座の概要

- **提供者:** Python VTuber スプー
- **形式:** 完全無料のプログラム
- **定義:** 「AIを使用して自然言語プロンプトから機能コードを生成する新しいソフトウェア開発手法」
- **対象者:** 生成AI開発に関心のあるエンジニア、プロダクトマネージャー、企画者

## 習得スキル

- Pythonおよび VSCode のセットアップ方法
- 自然言語によるアプリケーション開発プロセス
- Claude Codeの実践的な活用技法

## 実習ステップ例（レッスン1：ダッシュボードアプリ作成）

1. **環境構築** - Python・VSCode・Claude Code のセットアップ
2. **APIキー取得** - Claude API認証情報の取得
3. **データ取得処理** - 外部データソースからのデータ読み込み
4. **グラフ可視化** - Matplotlibやその他ライブラリでの視覚化
5. **AI分析機能統合** - Claude APIを使った自動分析・インサイト生成
