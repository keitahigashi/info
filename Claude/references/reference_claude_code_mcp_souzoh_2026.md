## MCPの概要と設定

### MCPとは
「AIと外部ツールをつなぐ業界標準規格」。設定はコマンド1行で完了。

### スコープの種類
| スコープ | 対象 | 用途 |
|---------|------|------|
| `local` | 自分のみ | 個人用ツール |
| `project` | プロジェクト内共有 | チーム共有 |
| `user` | 全プロジェクト | グローバル適用 |

### 最初に導入すべき3点セット
1. **業務データベース**: データ読み取り・書き込み
2. **ブラウザ操作**: Web閲覧・操作自動化
3. **検索機能**: 情報収集

これにより「台帳を読む→調べる→操作する→書き戻す」という閉じたワークフローが実現。

### セキュリティ要点
- 公式が「信頼できるサーバーのみ接続」を警告
- APIキーは環境変数経由で渡す
- 権限は最小限に絞る

非エンジニア向けにノーコード視点から解説した実践ガイド。
