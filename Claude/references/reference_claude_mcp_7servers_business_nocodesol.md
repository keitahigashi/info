## 業務部門別MCPサーバー活用ガイド

### MCPの定義

「Claudeが外部ツールやデータに安全にアクセスするための中継サーバー」

### サーバー選定の5つの評価軸

- 接続先の適合性（DB/SaaS/ファイル/RPA）
- 認証方式（APIキー/OAuth/SSO）
- 権限分離（読み取り専用など）
- 監査ログの有無
- 運用性（デプロイ・更新対応）

### 業務別活用事例と工数削減効果

| 部門・業務 | 削減率 | 削減時間 |
|-----------|--------|---------|
| 営業提案作成 | 34% | 3.2h → 2.1h |
| 経理月次締め | 35% | 40h → 26h |
| 人事問い合わせ対応 | 40% | 15分 → 9分 |
| カスタマーサポート一次切り分け | 36% | 25分 → 16分 |
| 品質保証報告書作成 | 33% | 90分 → 60分 |
| 法務契約レビュー | 35% | 120分 → 78分 |
| 開発部門障害対応 | 39% | MTTA 18分 → 11分 |

### 導入5ステップ

1. KPI設定
2. 接続要件整理
3. PoC実施
4. 運用設計
5. 本格展開

### 費用感

- 小規模PoC: 初期30〜120万円・月額3〜15万円

### 誤操作リスク低減の原則

「参照→下書き→人が確定」の三段階分離を推奨
