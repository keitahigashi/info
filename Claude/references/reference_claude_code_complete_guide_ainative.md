## 詳細

### Claude Codeの本質
ターミナルレベルで自律的に動作するAIコーディングエージェント。ファイルシステム操作、コード実行、Gitオペレーションを独立して管理。

### 6つの主要機能
1. **コード操作**: 生成・編集・リファクタリング、import依存追跡
2. **ファイル管理**: Read/Write/Glob/Grep/Bash
3. **Git自動化**: Conventional Commits、PR作成、ブランチ管理、コンフリクト解消
4. **テスト**: フレームワーク自動検出（Vitest/Jest/pytest）、生成・実行
5. **MCP統合**: Playwright/GitHub/Slack/DB/Figma
6. **サブエージェント**: 探索・計画・テスト・コードレビューの並列実行

### 2026年新機能
- Web版 `/teleport`: ターミナルセッションをブラウザに転送
- Computer Use: マウス・ブラウザ・ファイル操作の自律実行
- Voice Mode: Push-to-Talk、20言語対応
- /loop: 定期バックグラウンドタスク
- Prompt Caching: コンテキスト再利用で高速化・コスト削減
- Extended Thinking: 設計・デバッグ向け推論強化
