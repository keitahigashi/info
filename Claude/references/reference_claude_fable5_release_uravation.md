## Claude Fable 5 リリース速報（2026年6月）

### 公開日
2026年6月10日（更新：6月11日）

### 性能と位置づけ

- **SWE-Bench Pro**: 80.3%（GPT-5.5 58.6%・Opus 4.8 69.2%を上回る）
- **コンセプト**: 限定公開の「Mythos」モデルを初めて一般向けに提供するバージョン
- **強み**: 長くて複雑なタスクを途中で崩れずやり切る力

### 料金体系

| 項目 | 料金 |
|------|------|
| 入力トークン | $10 / MTok |
| 出力トークン | $50 / MTok |
| Opus 4.8比 | 約2倍 |
| 無料期間 | claude.ai 全プランで6月22日まで無料 |

### 利用経路

- Claude API
- claude.ai（Web・Desktop・Mobile）
- Amazon Web Services Bedrock
- Google Cloud Platform
- GitHub Copilot

### 安全機構（3層セーフガード）

1. **通常フィルタリング**: 標準的な有害プロンプトチェック
2. **ドメイン特化フォールバック**: サイバー・生物化学領域ではOpus 4.8へ自動フォールバック
3. **データ保持**: 30日間保持（ゼロデータ保留は非対象）

### 法人導入の判断軸

- **向いている業務**: 複雑な調査・マルチファイルコーディング・エージェントワークフロー
- **コスト管理**: 業務の「重さ」で使い分け、軽い作業は引き続きOpus 4.8/Sonnet 4.6
- **落とし穴**: 「全部Fable 5に乗り換える」はコスト爆発の典型パターン

### 6月22日以降の課金モデル

6月22日以降は使用量クレジット制に移行（全プラン）。6月15日から導入済みの「対話/自動化2プール分離」との組み合わせ管理が必要。
