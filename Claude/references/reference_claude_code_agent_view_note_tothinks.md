## Agent View徹底解説

### 基本情報

- 公開日：2026年5月12日
- 対象バージョン：v2.1.139以上

### Agent Viewのコア機能

```bash
claude agents    # ダッシュボード起動
```

- 進行中・待機中・完了済みセッションを1画面で一覧表示
- 複数セッションの並列タスク管理
- バックグラウンドへのセッション送信

### 同時発表の新機能

#### /goalコマンド
- 「完了条件を満たすまで自律継続」する仕様
- 長期タスク自動化（夜間実行など）に対応
- 従来の一問一答型から脱却した持続型タスク実行

#### MCPサーバー並列接続
- 外部ツール連携の処理速度向上
- 複数MCPサーバーへの同時接続が可能

#### 50のCLI改善
- システムプロンプト最適化
- セキュリティ強化
