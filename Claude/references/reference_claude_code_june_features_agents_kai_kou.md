## Claude Code 2026年6月 新機能まとめ

**公開日：** 2026年6月20日

### 1. `/cd`コマンド（v2.1.169）

セッションを維持したままディレクトリ切り替えが可能に。

- **従来**：新しいセッションを起動する必要があった
- **新機能**：会話履歴と読み込んだファイルのコンテキストを保持しながら複数リポジトリを行き来可能
- **活用**：複数リポジトリを横断する作業でセッション再起動が不要

### 2. ネストサブエージェント5段階対応（v2.1.172）

エージェントが最大5階層までの入れ子構造をサポート。

- 複雑な自律タスクを階層化して安定動作
- 設計時の注意：各エージェント間では必要最小限の情報伝達に絞ることを推奨
- バックグラウンドチェーンは5レベルの深さでキャップ

### 3. Auto-retryとフォールバックモデル

長時間自動実行のパイプライン安定性を向上：

- レート制限ヒット時に指数バックオフで自動リトライ
- 軽量モデルへの自動切り替え（`fallbackModel`で最大3つのフォールバックモデルを設定）

```json
// settings.json例
{
  "fallbackModel": ["claude-sonnet-4-6", "claude-haiku-4-5"]
}
```

### 4. AWS Bedrock自動設定

- `~/.aws/config`を自動読み込み
- 手動でリージョン設定する手間が削減
- Bedrock・Vertex・Foundryでのauto modeが利用可能

### Week 24（6月8〜12日）その他の新機能

- **`--safe-mode`**：すべてのカスタマイズを無効にした状態でClaude Codeを起動（トラブルシューティング用）
- **サブエージェントの独自サブエージェント生成**：階層型タスク分解が可能に
