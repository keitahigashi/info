## 出典

Zenn (genda_jp): https://zenn.dev/genda_jp/articles/9558ecec45beb5

## 変更の概要

2026年6月15日より「対話利用」と「自動化」の課金が分離。

| 利用形態 | 課金区分 |
|---------|---------|
| Web/アプリでのチャット | サブスク枠（変更なし） |
| Claude Code | サブスク枠（変更なし） |
| Claude Cowork | サブスク枠（変更なし） |
| `claude -p`（非対話モード） | 月次クレジット（従量課金） |
| Agent SDK | 月次クレジット（従量課金） |
| GitHub Actions連携 | 月次クレジット（従量課金） |
| サードパーティ経由の利用 | 月次クレジット（従量課金） |

## 影響確認手順

1. **自分の使い方を確認** — 対話のみなら対応不要
2. **自動実行箇所を洗い出す** — cron・CI・スクリプト・サードパーティツールを確認
3. **認証経路を確認** — サブスク認証か、APIキー認証かを判定
4. **該当者向け対応** — クレジット請求の設定、使用制限のON/OFFを実施

## プランごとの月次クレジット

- **Proプラン**: 月 $20 相当のクレジット
- **Maxプラン**: 月 $100 相当のクレジット

## 注意点

- **opt-inが必要**: メール案内からopt-inして初めてクレジットを受け取れる（見逃し注意）
- **超過課金OFF設定**: クレジット枯渇時に自動化が停止するよう設定可能
- 「自動化のコストが定額に隠れていた時代が終わる」ことを意味する重要な変更
