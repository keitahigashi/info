## 法人導入実践ガイド：稟議テンプレ・セキュリティ・段階展開

### 目的

企業がClaude Codeを安全に導入・運用するための実践ガイド。情シス部門の承認取得に使える稟議テンプレートを提供。

### プラン別セキュリティ機能比較

| プラン | データ学習 | 管理機能 | 認証 |
|--------|----------|---------|------|
| Free/Pro | 利用の可能性あり | なし | 標準 |
| Team | 使用されない | 管理ダッシュボード | SSO対応 |
| Enterprise | 使用されない | フル管理 | SSO+HIPAA+データレジデンシー |

Enterprise: SOC 2 Type II準拠
