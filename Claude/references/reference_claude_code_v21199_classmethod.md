## Claude Code v2.1.199 の主要アップデート（2026年7月3日）

### リリース概要
2026年7月2日リリース。主にバグ修正に焦点を当て、バックグラウンドエージェント・サブエージェントの安定性向上とネットワーク信頼性改善が中心（24件の変更）。

### 新機能
- **スラッシュスキルのチェーン機能強化**：`/skill-a /skill-b do XYZ` のようにチェーンする際、先頭1スキルだけでなく最初の5スキルまで同時ロード可能に

### 主要な改善事項
- サーバー側レート制限エラー（429）の自動リトライ機能追加
- 環境変数 `CLAUDE_CODE_RETRY_WATCHDOG` のデフォルトリトライ数向上
- `claude agents` セッション行のプルリクエストリンク表示改善

### 重要なバグ修正

| バグ | 内容 | 修正内容 |
|------|------|---------|
| Linuxバックグラウンドデーモン強制終了 | 約50秒ごとに全エージェントが強制終了 | 異常終了後の破損レコード処理を改善 |
| サブエージェントエラー処理 | APIエラー（使用量上限等）が成功として報告 | エラーを親エージェントに正しく伝達 |
| ストリーミングレスポンス破棄 | サーバーエラー時に部分レスポンスが失われる | 部分レスポンス保持機能を実装 |
| SSL証明書エラー処理 | TLS検査プロキシ環境での即座失敗 | 修正ヒント表示と処理改善 |

### 利用者への影響
並列エージェントを多数走らせているユーザーにとって、特にLinuxデーモン修正とサブエージェントエラー正確伝達が重要。
