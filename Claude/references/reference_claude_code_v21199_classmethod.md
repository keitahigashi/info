## Claude Code v2.1.199 主要アップデート（2026年7月3日）

### 概要
- 公開日：2026年7月3日
- 変更件数：24件
- 主軸：バックグラウンドエージェント・サブエージェントの安定性、ネットワーク信頼性

### 新機能

- **スラッシュスキル連続呼び出し強化**：先頭に並べたスキルを最大5個まで同時読み込み可能に（従来は1個のみ）

### 重大な不具合修正

| 修正内容 | 詳細 |
|---------|------|
| Linux強制終了バグ | バックグラウンドエージェントデーモンが約50秒ごとに自身と稼働中の全エージェントを強制終了していた問題を解決 |
| サブエージェントAPIエラー誤報 | 利用上限到達などのAPIエラーを「成功結果」として親エージェントに報告していた問題を改善 |

### セキュリティ修正
- プランモードが状態変更系ブラウザツール呼び出しに確認を要求するよう改善
- 読み取り専用の `browser_batch` 呼び出しのみ自動許可

### ネットワーク・安定性改善

| 区分 | 内容 |
|------|------|
| レート制限対応 | 一時的なサーバー側429エラーを自動リトライ |
| リトライ設定 | `CLAUDE_CODE_RETRY_WATCHDOG` のデフォルト回数を300に引き上げ、`CLAUDE_CODE_MAX_RETRIES` 上限（15）を撤廃 |
| ストリーミング対応 | API応答途中のエラーでも部分応答を保持 |
| SSL証明書エラー | リトライ前に即座に失敗し修正ヒントを表示 |
| 設定ファイル復旧 | リセット前にバックアップを自動取得 |

### その他
- macOS SSH対応改善
- `claude stop` 競合解決
- Hook stderr抑制
