## Claude 2026年6月 アップデートまとめ

### 公開日
2026年6月9日

### 主要アップデート一覧

#### モデル強化
- Claude Opus 4.8：Max・Team Premiumの新デフォルトモデル
- `/effort` スライダーでインタラクティブに調整可能
- Fable 5（Mythos級）：6月9日より一般公開

#### Microsoft 365 連携
- SharePoint・OneDrive・Outlook・TeamsをClaudeで検索・参照可能
- Office アドイン：Excel・PowerPoint・Word・Outlookのファイルを基に資料作成・要約・編集

#### Artifacts
- 文章・コード・図表を別画面で編集・プレビュー可能
- 生成物のリアルタイム確認と修正が容易に

#### Claude Cowork（有料プラン向け）
- 複数ステップの業務タスクに対応
- 人間のワークフローに近い段階的実行

#### Claude Code 拡張
- 開発者向け機能を継続強化
- Dynamic Workflows・エージェントビュー等の新機能追加

#### スマホアプリ連携
- 位置情報・カレンダー・リマインダーとの統合
- モバイルファーストの業務利用が可能に
