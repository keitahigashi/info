## 変更の核心：2プール分離

2026年6月15日より、Claudeの利用枠が「対話（Interactive）」と「自動化（Programmatic）」に完全分離される。

### 影響を受けない利用方法
- チャットインターフェース（claude.ai）での会話
- Claude Code の対話的使用（人が操作しながら使う場合）
- Claude Cowork
→ 従来通りのサブスク枠から消費。変化なし。

### 影響を受ける利用方法
- Claude Agent SDK
- ターミナルコマンド `claude -p`（ヘッドレスモード）
- GitHub Actions 連携
- OpenClawなどの自動実行ツール
→ 新設の「Agent SDK月額クレジット」から消費。

## クレジット配分（プラン別）

| プラン | 月間Agent SDKクレジット |
|--------|------------------------|
| Pro | $20相当 |
| Max 5x | $100相当 |
| Max 20x | $200相当 |

- 毎月の請求サイクルでリセット
- 余ったクレジットの繰越不可
- クレジット超過分は標準API料金での従量課金

## 著者の評価

「全員の値上げではなく、ピンポイントな変更」。チャットやClaude Code対話利用のみのユーザーには影響ゼロ。自動化ツールを多用する開発者や企業が主な影響対象。

自分の使用方法を確認する際の判断基準：「人がキーボードを操作しているか、プログラムが自動実行しているか」で二分できる。
