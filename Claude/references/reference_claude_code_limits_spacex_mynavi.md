## Claude Code・API制限変更詳細（2026年5月7日）

### Claude Codeの変更

- 5時間あたりのレート制限を2倍に引き上げ
- ProおよびMaxプランのピーク時間帯における利用枠引き下げ措置を完全撤廃
- Team、シートベースのEnterpriseプランも対象

### Claude API（Opusモデル）の大幅な制限引き上げ

| 指標 | 変更前 | 変更後 |
|------|--------|--------|
| 1分あたりの最大入力トークン数 | 800,000 | 5,000,000 |
| 最大出力トークン数 | 160,000 | 400,000 |

### SpaceXパートナーシップ詳細

- 300メガワット超の計算容量
- NVIDIA製GPU換算で22万基以上
- Colossus 1データセンターの容量を活用

### 技術的意義

入力トークンが800kから5Mへ約6.25倍拡張されたことで、大規模コードベース全体の一括解析や長大なドキュメント処理が現実的に。出力トークン2.5倍増加で長文コード生成・詳細解説も向上。
