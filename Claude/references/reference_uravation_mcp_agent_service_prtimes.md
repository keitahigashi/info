## 出典

PR TIMES: https://prtimes.jp/main/html/rd/p/000000155.000137731.html

## 概要

- **公開日**: 2026年6月3日
- 株式会社Uravationが「汎用AIの導入だけでは現場が動かない」という課題に対応し、Claude Code・MCP・Skillsを組み合わせた業務特化型AIエージェント開発サービスを強化・発表。

## サービスの特徴

- **技術スタック**: Claude Code + MCP（Model Context Protocol）+ Skills
- **アプローチ**: 各社の業務フローに合わせてエージェントを設計・開発
- **提供範囲**: 要件ヒアリングから研修・伴走支援まで一貫提供（「開発して終わり」ではなく現場定着まで支援）

## 想定される活用領域

| 部門 | 活用例 |
|------|--------|
| 営業 | 提案資料作成・顧客リサーチ補助 |
| 経理・バックオフィス | 定型書類作成・データ整理 |
| カスタマーサポート | FAQ回答案生成 |
| リサーチ・企画 | 情報収集・調査要約 |

## 業界動向としての意味

Claude Code・MCP・Skillsを組み合わせた業務特化型エージェントが商用サービスとして本格展開されるフェーズに入ったことを示す事例。ベンダー各社がAPI利用だけでなく、Claude Codeのエコシステム（MCP・Skills・Hooks）を組み合わせたソリューション開発を本格化している。
