## 概要

2026年5月のClaude Codeアップデートを実務視点でまとめた記事。派手な新機能より「実務利用に効く改善」が中心の週となった。

## 主要アップデート

### 1. Fast mode のデフォルトモデル変更
- Opus 4.6 → **Opus 4.7** にアップグレード
- 軽微な修正やリファクタリングなど速度重視タスクで出力傾向が変わる可能性あり

### 2. /goal コマンド追加
- 完了条件を設定し、それを満たすまで複数ターン自動継続
- テスト成功や特定仕様達成まで自律的に進める使い方に最適

### 3. claude agents（Research Preview）
- 複数セッションの状態管理ビューが追加
- 実行中・完了・待機中タスクを一覧表示
- 並列実行時の状況把握が容易に

### 4. インフラ安定化
- プラグイン依存関係の強化
- バックグラウンドセッションの信頼性向上
- コンテキストコスト見積もりの可視化
- 企業利用を視野に入れた改善

## 企業展開の進展

PwCとの戦略的パートナーシップ拡大が最大のニュース。米国チームから導入を開始し、数十万人規模のプロフェッショナルへグローバル展開予定。保険引受が10週間から10日に短縮されるなど、本番業務での実績が報告されている。

## 実務ポイント

- Fast modeの出力傾向変化に注意（既存ハーネスのテスト推奨）
- /goalはCI/CDパイプラインとの統合で特に威力を発揮
- claude agentsビューで並列開発の管理コストが大幅減
