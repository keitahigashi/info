## 新デザイン機能一覧

| 機能 | 詳細 |
|------|------|
| **新サイドバー** | 全セッションを一元管理。ステータスフィルタリング・プロジェクト別グループ化・自動アーカイブ機能 |
| **ドラッグ&ドロップレイアウト** | ターミナル・プレビュー・チャットなどのペインを自由配置可能なカスタマイズUI |
| **統合ターミナル** | アプリ内にターミナル機能を搭載。ウィンドウ切り替え不要に |
| **ファイルエディタ統合** | スポット編集がアプリ内で完結 |
| **HTMLプレビュー** | Webアプリの変更をリアルタイム確認可能 |
| **サイドチャット** | メインタスク継続中に別スレッドで質問可能 |
| **テーマ自動切り替え** | ターミナル設定に合わせて自動でライト/ダークモードを切り替え |

## 対応プラン

Pro・Max・Team・Enterprise（2026年4月より順次展開）

## 並列エージェント対応

新サイドバーにより複数セッションを同時並走でき、Agent Teams機能との親和性が向上。

## 旧UIとの主な違い

- セッション一覧の永続表示（従来は都度開く必要）
- ペインレイアウトのカスタマイズ性（ドラッグ&ドロップ）
- サイドチャットによるコンテキスト分離
