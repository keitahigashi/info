## 主要発表

### 1. セルフホストサンドボックス（パブリックベータ）

**概要**: Claude Managed Agents のツール実行を、ユーザーが管理するインフラ上で動作させる仕組み。

**アーキテクチャ**:
- ツール実行 → ユーザー管理インフラ（Cloudflare・Daytona・Modal・Vercelなどのマネージドプロバイダーまたは自社インフラ）
- エージェントループ（オーケストレーション・コンテキスト管理・エラーリカバリ） → Anthropic インフラ

**利点**:
- 既存のネットワークポリシー・監査ログ・セキュリティツールをそのまま適用
- ファイル・リポジトリを社内ネットワーク外に出さない
- 計算コストの大きい作業向けにコンピュートサイジングとランタイムイメージを制御可能

**既に採用している企業**: Amplitude・Clay・Rogo

### 2. MCPトンネル（リサーチプレビュー）

**概要**: プライベートネットワーク内の MCP サーバーに、パブリックインターネット経由なしでアクセスする仕組み。

**仕組み**:
- ユーザーがデプロイする軽量ゲートウェイが**単一のアウトバウンド接続**を確立
- インバウンドファイアウォールルール不要
- パブリックエンドポイント不要
- エンド・ツー・エンド暗号化

**利点**:
- 社内ネットワーク上の MCP サーバー（社内 DB・社内 API など）に安全にアクセス
- パブリックインターネットへの露出なし

## ボリス・チェルニーのキーノート主旨

プログラミングの歴史的変遷:
1. **初期のプログラミング**: 直感的・魔法的な体験（シンプルな抽象レイヤー）
2. **現代のプログラミング**: 複数の抽象レイヤーの複雑な積み重ね
3. **エージェントベースのアプローチ**: アイデアを思いついてから実行に移るまでのギャップを縮小 → 初期のプログラミングの即時性を復活
