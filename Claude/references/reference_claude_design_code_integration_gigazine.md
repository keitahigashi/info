## Claude Design × Claude Code 連携強化（2026年6月17日発表）

### 発表概要

Anthropicが2026年6月17日にClaude Designの機能強化を発表。デザインシステムに沿った制作、キャンバス上での直接編集、Claude Codeとの同期、外部ツール連携に対応。

### 主要な機能強化

#### デザインシステム管理

- GitHubやデザインファイルから複数のデザインシステムをインポート可能
- 管理者ロールで標準デザインシステムを承認・制限

#### デザイン・コード双方向連携

- `/design-sync` でデザインシステムをリポジトリに取り込み
- `/design` コマンドでコードからデザイン操作が可能
- 成果物をシームレスに引き継ぎ可能

#### エディター強化

- キャンバス上での直接操作（ドラッグ、リサイズ、整列）
- 数百件の安定性修正

### 外部ツール連携

対応サービス：Adobe、Canva、Lovable、Replit、Vercel、Wixなど

### その他の改善

- 利用上限をClaude Cowork・Claude Codeと共有
- 平均的なトークン消費量が減少
- PDF・PowerPoint書き出し対応
