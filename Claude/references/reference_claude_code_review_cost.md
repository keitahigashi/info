## 概要
Claude Code新機能「Code Review」を2つのPRで実測し、$100超のコストが発生した体験レポート。マネージド版とプラグイン版の比較、コスト削減戦略を提案。

## 詳細

### マネージド Code Review の特性
- 複数エージェントが並列でバグ探索を実行
- セキュリティリスク評価と精査レベルの自動判定
- 「Extended reasoning」で思考過程を可視化

### 実測コスト
- 70行のMarkdown差分: **$31.54**（30分2秒）
- 大規模Goコードベース: **$77.96**（47分14秒）
- 公式記載の$15〜25を大幅に超過
- 月200PR規模での試算: **月額$15,000超**

### プラグイン版（code-review プラグイン）
- コマンド: `/code-review:code-review` または `--comment` オプション付き
- 5つの独立レビューエージェントが並列起動
- CLAUDE.mdコンプライアンスチェック、バグ検出、gitヒストリー分析
- Confidenceスコア80以上で報告
- **$0追加コスト、4分23秒**で完了

### カスタマイズ
- CLAUDE.md: プロジェクトコンテキストとコーディング規約
- REVIEW.md: レビュー固有チェック項目、言語指定（「日本語でレビュー」等）

### 運用戦略
プラグイン版をメインに運用し、マネージド版はコスト最適化待ちが現実的。組織的標準化やセキュリティ監査記録が必要な場合はマネージド版を検討。
