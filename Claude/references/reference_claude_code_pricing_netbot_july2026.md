## 料金体系と実測レポート

- サブスク定額: Pro $20 / Max 5x $100 / Max 20x $200
- API従量課金: サブスク上限超過後も継続利用可能
- 無料プランではClaude Code利用不可

### 著者の実測データ

- 月$290（AI関連経費計）で従来月20〜40万円相当の業務を一人運営
