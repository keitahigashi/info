## 2026年4月時点の主要アップデート

### Routines（クラウド常駐自動化）
- **スケジュール（cron）**：時間・日次・週次で自動実行
- **API**：外部サービスからのトリガー
- **GitHub（Webhook）**：PR/コミットをトリガーに自動実行

### デスクトップアプリ刷新
- 並列エージェントのサイドバー管理
- 統合ターミナル・改良差分ビューア
- HTML・PDFプレビュー機能

### Opus 4.7 + xhigh Effort
- xhighがcoding/agenticタスクのデフォルト推奨
- 長時間エージェント運用に最適化

### Agent Teams
- 複数エージェントの同一ウィンドウ並列実行
- 双方向コラボレーション型マルチエージェント

## 対象読者と活用場面

- 個人開発者・チーム開発・エンタープライズ導入の全段階に対応
- インストールから本番運用まで一冊で参照できる実用的リファレンス
