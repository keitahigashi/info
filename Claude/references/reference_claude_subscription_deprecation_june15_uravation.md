## 2026年6月15日の3大変更

### 変更1：旧モデルIDのAPI廃止
`claude-sonnet-4` / `claude-opus-4` のAPIエイリアスが廃止。
バージョン固定ID（例: `claude-opus-4-8-20261001`）への変更が必要。

### 変更2：サブスク利用枠の2プール分離
- **インタラクティブプール**: チャット・Claude Code対話的使用
- **プログラマティックプール**: Agent SDK・claude -p・GitHub Actions

### 変更3：自動化処理の従量課金移行
CI/CD統合や自動実行処理はAgent SDKクレジットが尽きた後、標準API料金での従量課金。

## コスト影響の試算

| 使用パターン | 月間変化 |
|-------------|---------|
| チャット・対話利用のみ | 変化なし |
| CI/CD月1,000回・軽量タスク | +$15〜30程度 |
| 本格自動化（月10,000回以上） | 実質15〜30倍のコスト増の可能性 |

## 5項目チェックリスト（企業向け）

1. **モデルID確認**: コードベース内に `claude-sonnet-4` / `claude-opus-4` が含まれていないか
2. **自動実行の把握**: GitHub Actions・バッチ処理・スケジュール実行の一覧化
3. **クレジット枠の見積もり**: 月間API呼び出し回数とトークン量の概算
4. **プラン再検討**: 自動化量に応じたMax/Team/Enterpriseプランの最適化
5. **フォールバック設計**: クレジット超過時の動作（エラーか従量か）の確認

## よくある誤解

- **誤**: Claude Code が有料になる → **正**: 対話的使用は従来通り
- **誤**: 全ユーザーへの値上げ → **正**: 自動化処理の実費化
- **誤**: サブスク廃止 → **正**: 利用枠の分類変更のみ
