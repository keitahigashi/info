## Claude Design 大規模アップデート（2026年6月18日）

### アップデート概要

AnthropicのAIデザインツール「Claude Design」に複数の新機能が追加された。

### 新機能

#### 1. デザインシステムの統一

- GitHub、デザインファイル、画像などから複数のデザインシステムをインポート可能
- 組織のガイドラインに沿ったデザイン統一が容易に

#### 2. Claude Code との連携強化

- `/design-sync` コマンドでシームレスな移行が可能
- デザイン完成後、自動でコード化段階へ移行

#### 3. キャンバス上での直接編集

- デスクトップアプリのサイドバーに「デザイン」を追加
- 迅速なアクセスを実現

#### 4. エクスポート機能の拡張

- PDF、PowerPoint出力対応
- Adobe、Canva、Figmaなど連携先を準備

### 利用条件

Claude Pro（月20ドル）以上のプラン契約者が対象
