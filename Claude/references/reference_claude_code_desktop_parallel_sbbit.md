## 主要内容

### デスクトップアプリ刷新（2026年4月）
AnthropicがClaude Codeのデスクトップアプリを大幅刷新し、並列エージェントによるマルチタスク機能を強化。

### 新サイドバー
- アクティブなセッションと最近使用したセッションを一覧表示
- セッション間の移動が容易に

### 並列エージェント対応
- Subagent @mention機能でメインセッションからサブエージェントにタスク委任
- 複数プロジェクトの並行処理が可能

### 作業効率化機能
- Windows: `Ctrl + ;` でサイドチャットを起動
- 統合ターミナルによるテスト・ビルドの並行実行
- アプリ内ファイルエディタと改良された差分ビューア
- HTML・PDFのプレビュー機能強化
- CLIプラグインの互換性向上

### 表示モード
詳細・通常・概要の3段階表示に対応。

### 対象プラン
Pro、Max、Team、Enterprise プランのユーザー。
