## Claude Opus 4.8概要

**公開日：** 2026年5月29日

## 主なトピック

- あらゆる領域のベンチマークで改善（コーディング・推論・エージェントスキル）
- GPT-5.5やGemini 3.1 Proを複数タスクで上回る
- 不確実性を自ら指摘する「正直さ」の強化
- 欺瞞への協力がOpus 4.7より大幅に低減

## 新機能

### Dynamic Workflows
単一セッション内で数百のサブエージェントを並列実行可能。大規模タスクの自動分解・並列処理が実現

### Effort Control（5段階）
| レベル | 特徴 |
|--------|------|
| Low | 速度優先 |
| Medium | バランス |
| **High**（デフォルト） | 標準品質 |
| Extra | 高品質 |
| Max | 最高品質 |

## Mythosへの言及

- 欺瞞・悪用への協力がOpus 4.7比で大幅低減
- 「**数週間以内にすべての顧客にMythosクラスのモデルを提供できる見込み**」とAnthropicが言及

## 価格情報

- 入力：$5 / 100万トークン（出力：$25）
- Fast Mode：入力$10、出力$50（2.5倍速）
