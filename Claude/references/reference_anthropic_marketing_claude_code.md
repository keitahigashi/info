## 概要

Anthropicのグロースマーケティングチーム（非技術者1人）がClaude Codeをガチ運用。エンジニアなしで検索広告・SNS広告・ASO・メール・SEOを担当。

## 4つのユースケース

### 1. Google Ads クリエイティブ自動生成
- 既存広告CSV読み込み→パフォーマンス低下広告を特定
- 見出し用（30文字制限）と説明文用（90文字制限）に別々のサブエージェント配置
- 複数キャンペーン分を数分で生成

### 2. Figmaプラグイン開発
- Claude Codeでプラグインを自作
- デザインフレーム自動認識→最大100パターンの広告バリエーション生成
- 1バッチ0.5秒、クリエイティブ生産量10倍

### 3. Meta Ads MCPサーバー構築
- MCPでMeta Ads APIを連携
- Claude Desktopから直接キャンペーン分析
- ダッシュボードアクセス不要

### 4. メモリシステムによる自己改善
- 過去の仮説と実験結果を記録→新規広告生成時に参照
- テストのたびに精度向上

## ワークフロー

1. claude.aiで全体設計
2. タスクごとにサブエージェント分割
3. Claude Codeへ渡す
4. 一発完成を狙わない

## 成果

- 広告コピー作成: 2時間→15分
- クリエイティブ生産量: 10倍
- 1人で大規模チーム並みの処理量

## 3つのコツ

1. API保有ツールの反復作業を狙う（広告PF、デザインツール、分析基盤）
2. タスク分割（専門エージェント配置で品質確保）
3. 段階的設計（即座にコーディングしない、まず全体像を固める）
