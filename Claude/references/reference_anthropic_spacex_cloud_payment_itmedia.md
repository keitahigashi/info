ITmedia NEWS: https://www.itmedia.co.jp/news/articles/2605/21/news083.html

## 記事概要

- **公開日**: 2026年5月21日 13時05分
- **情報源**: SpaceX の IPO 関連目論見書（SEC 提出）

## 契約の主要データ

| 項目 | 内容 |
|------|------|
| 月額契約金額 | 12億5000万ドル（約1987億円） |
| GPU 台数 | NVIDIA 製 22万台以上 |
| 対象インフラ | COLOSSUS・COLOSSUS II |
| 導入スケジュール | 2026年5月〜6月に段階的運用開始 |
| 契約条項 | 90日間の通知期限で終了可能 |

## 背景・経緯

- SpaceX が IPO 準備中に SEC へ提出した目論見書で契約内容が明らかになった
- Claude の利用需要増大に伴い、Anthropic が外部スーパーコンピュータへのアクセスを確保
- 2026年5月に Claude Code の5時間レート制限を2倍にした原動力のひとつ
- Intel もスーパーコンピュータ関連プロジェクトに参画

## 戦略的意義

- Anthropic の計算リソースが不足しており、外部クラウドで補完する戦略
- 月額1987億円という規模は AI インフラ投資の新たな水準を示す
- SpaceX の COLOSSUS（テキサス州メンフィス）はトレーニングとインファレンスの両方に対応
- Claude の週次利用制限緩和（50% 増加、7月13日まで）の継続にも貢献予定
