## Claude Mythos とは

2026年4月に発表されたセキュリティ特化型 Claude モデル。2026年6月後半の一般公開が予定されている。

## 主要スペック

| 項目 | 内容 |
|------|------|
| SWE-bench Verified | 93.9%（世界最高水準のコーディング性能） |
| 特化領域 | サイバーセキュリティ・脆弱性発見 |
| 現在の提供状況 | Project Glasswing 経由で 150 組織・15カ国超に限定提供 |
| 一般公開条件 | 「十分なセーフガード整備後」（具体日程未発表） |

## 一般公開が見送られた理由

「ゼロデイ脆弱性を自律発見・悪用する能力」があるため、安全保障上のリスクから慎重な段階的リリースを採用。Anthropic は Glasswing 参加組織でのユースケース検証と安全基準の整備を優先している。

## Project Glasswing の概要

- 対象: 企業・研究機関・政府機関
- 規模: 150 組織以上、15カ国超
- 目的: 実環境での安全性検証とセーフガード設計
- 参加申請: Anthropic 公式サイト経由（審査あり）

## 中小企業向け3ステップ準備アクション

1. **セキュリティ棚卸し** — 現在の脆弱性管理フロー・ペネトレーションテスト頻度を文書化
2. **Opus 4.8 での準備** — Mythos の前身モデルで自動化パイプラインを構築・慣れておく
3. **Glasswing 動向モニタリング** — Anthropic の公式ブログ・ニュースレターを定期チェック
