## 概要
国土交通省が2026年2月26日に「地理空間MCP Server」（α版）を公開。不動産情報ライブラリAPIの35種データのうち25種に対応し、自然言語で地理空間情報を取得可能。

## 詳細

### 対応データ（25種）
- 地価公示、都道府県地価調査、不動産取引価格
- 防災情報、地形情報
- 周辺施設、人口情報

### 利用方法
- 「新宿駅周辺の地価公示データを教えて」のような自然言語でLLMに問い合わせ
- MCPサーバーが該当データを取得して回答を返す

### 前提条件
- 不動産情報ライブラリのAPI利用申請（APIキー取得）が必要

### MCPの役割
AIと外部サービス間の「通訳係」として機能。自然言語リクエストをAPIが理解できる形式に翻訳。

### 関連取り組み
- 2025年11月: 「MLIT DATA PLATFORM MCP Server」公開済み
- 今後: PLATEAUなど他地理空間データとの連携拡大が焦点
