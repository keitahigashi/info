## 概要
Claude Codeの5つの拡張機能（CLAUDE.md・Skills・スラッシュコマンド・サブエージェント・Hooks）を「コンテキスト効率」を軸に使い分けるガイド。

## 詳細

### 使い分けの判断基準

| 機能 | ロードタイミング | 適用シーン |
|------|----------------|-----------|
| CLAUDE.md | 常時読み込み | プロジェクト全体で必要な知識（概要・ガイドライン） |
| Skills | 必要時に自動読み込み | 特定タスク専用の知識（コードレビュー等） |
| スラッシュコマンド | 手動で明示的に呼び出し | PR作成等の定期的・意図的な処理 |
| サブエージェント | 独立タスクで過程不要 | 翻訳・大量ファイル調査（結果のみ親に返す） |
| Hooks | JSON設定で確実に自動実行 | 曖昧さを排除した特定タイミングの処理 |

### 核心的な使い分け原則
- **常に必要** → CLAUDE.md
- **特定タスクで必要** → Skills（コンテキスト圧縮可能）
- **人間が意図的に起動** → スラッシュコマンド
- **作業過程の記憶が不要** → サブエージェント
- **確実な自動実行** → Hooks
