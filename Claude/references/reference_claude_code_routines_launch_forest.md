## Routines（ルーチン）機能の公式発表

### 基本概念
- 一度設定すれば**スケジュール・API・イベント**に応答してクラウドで自動実行
- 端末を起動し続ける必要なし（クラウドインフラ常駐）
- 2026年4月15日よりリサーチプレビューとして提供開始

### 3つのトリガー方式

| トリガー | 説明 | 例 |
|---------|------|-----|
| スケジュール（cron） | 指定日時・間隔で定期実行 | 毎晩2時に実行 |
| API | APIコール時に実行 | HTTPリクエスト送信時 |
| GitHub（Webhook） | リポジトリイベントに応答 | プルリクエスト作成時 |

### プラン別1日あたり実行回数上限

| プラン | 上限 |
|--------|------|
| Pro | 5回/日 |
| Max | 15回/日 |
| Team | 25回/日 |
| Enterprise | 25回/日 |

### 対応プラン
Pro、Max、Team、Enterprise で現在テスト中（リサーチプレビュー段階）
