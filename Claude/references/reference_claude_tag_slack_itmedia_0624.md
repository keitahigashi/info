## Claude Tag：Slackをチームとして動くAIプラットフォームへ

**公開日**: 2026年6月24日

### 概要

Anthropicが2026年6月23日、SlackでClaude（AI）をチームメンバーとして扱える機能「Claude Tag」をリリース。EnterpriseおよびTeamプランのユーザー向けにベータ提供開始。

Slack内で`@Claude`とメンションするだけでタスクを委譲できる。

### 主な特徴

| 特徴 | 内容 |
|------|------|
| マルチプレイヤー対応 | チャンネル全体で1つのClaudeが機能し、全員が進捗を確認・継続可能 |
| 非同期処理 | ユーザーが別の作業をしている間にバックグラウンドで実行 |
| コンテキスト保持 | チャンネルの会話履歴を追跡し、毎回ゼロから説明が不要 |
| 自律的タスク進行 | 複数時間～数日かけて自律的にプロジェクトを進行 |
| 厳格なアクセス制御 | 管理者がチャンネルごとにツール・情報・メモリアクセスを制限 |

### 活用シーン例

**エンジニアリングチーム向け**:
- デプロイ前後のレイテンシ比較
- バグの特定・分析

**営業チーム向け**:
- CRM記録の整理・更新

### 利用条件・技術仕様

- **対象プラン**: EnterpriseまたはTeamプラン
- **使用モデル**: Claude Opus 4.8
- **既存Slack連携の変更**: 旧Claude Slackアプリから30日間の移行ウィンドウ

### Anthropicの実績

Anthropic社内では、コードの65%がClaude Codeで生成されている。Claude Tagはこのような自律的なAI活用をチームコラボレーションツールにも拡張する取り組み。
