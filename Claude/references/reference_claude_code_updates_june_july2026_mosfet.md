## 出典

MOSFET's blog（はてなブログ）: https://mosfet.hatenablog.com/entry/2026/07/06/001800

## 概要

MOSFET氏がClaude Code CLIツールの2週間分（6月22日〜7月3日）のバージョンアップデートを詳細追跡した技術ブログ記事。著者は「地殻変動が3つ同時に起きた2週間」と評している。

## 三大地殻変動

### ① Claude Sonnet 5 がデフォルトモデルに（v2.1.197）
- ネイティブ1Mトークンコンテキスト搭載
- プロモ価格: $2/$10 per Mtok（8月31日まで）

### ② サブエージェントのバックグラウンド実行化（v2.1.198）
- 親エージェントは手を止めず他の作業を継続し、完了時に通知を受ける仕様
- バックグラウンドエージェントが自動コミット・push・ドラフトPR作成

### ③ Fable 5・Mythos 5 輸出規制解除（6月30日）
- 前回の全停止から全面復帰

## 主要新機能一覧

| 機能 | 説明 | 初出バージョン |
|------|------|-------------|
| `claude mcp login/logout` | CLIからMCPサーバー認証 | v2.1.186 |
| `/rewind` 強化 | `/clear`前まで会話復元 | v2.1.191 |
| `/dataviz` スキル | チャート・ダッシュボード設計 | v2.1.198 |
| `sandbox.credentials` | サンドボックスから認証情報遮断 | v2.1.187 |
| 権限モード "Manual" | 旧「default」を明示的に改名 | v2.1.200 |

## Windows向け改善
- PowerShell gitコマンドの失敗判定修正（v2.1.196）
- CJK文字貼り付けの文字化け修正（v2.1.187）
- 日本語音声ディクテーション対応（v2.1.195）

## パフォーマンス向上
- **CPU使用量約37%削減**（v2.1.191）：ストリーミング中のテキスト更新を100ms単位に集約

## アーキテクチャ転換
従来の同期的なツリー構造（親が待機）から非同期ジョブディスパッチャへ移行。設定値の後方互換性も維持（`--permission-mode manual` と旧 `default` を併記で受理）。
