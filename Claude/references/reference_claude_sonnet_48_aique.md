## 概要

2026年5月2日公開。2026年3月のClaude Code npmパッケージのソースコードリーク（51万2,000行）から判明したClaude Sonnet 4.8の仕様と、2026年5月中旬のリリース予想をまとめた記事。

## リーク経緯

- 2026年3月31日：Claude Code npmパッケージに誤って含まれたソースマップが512,000行のTypeScriptコードを露出
- Anthropicのエンジニア Boris Cherny が「通常の開発者エラー」と確認
- コード内に `claude-sonnet-4-8` の参照が存在したことで発覚

## リリース予定

| 項目 | 内容 |
|------|------|
| リリース時期 | 2026年5月中旬予定 |
| 根拠 | Opus 4.7（4月16日）から1〜4週間後というAnthropicの従来パターン |
| バージョン番号 | Sonnet 4.7をスキップして4.8（リークコードで確認済み） |

## 期待される主な機能

### アダプティブ思考（Adaptive Thinking）
- タスク予算管理と組み合わせた複雑コーディング向け「高努力レベル」の実装
- 推論機能がデフォルトで統合（o-series型の別製品化ではなくメインモデルに組み込み）

### 性能向上領域
- ビジョン機能強化（Opus 4.7は98.5%の視覚精度）
- コーディングベンチマーク改善
- 高解像度画像サポート
- 長期コンテキスト処理
- エージェント型推論

## 価格予測

$3/$15（100万トークン入力/出力）— Sonnet 4.6から据え置き予想

- 大規模コードベース処理
- 複数文書にわたる分析
