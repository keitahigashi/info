## 記事概要

**公開日:** 2026年6月18日（アップデート発表日: 2026年6月17日）

Anthropicが「Claude Design」のベータ機能を大幅アップデート。提供開始から1週間で100万人超が利用する急速な普及を受けた機能強化で、デザインシステムの横断管理とClaude Codeとの双方向連携が主軸。

## 新機能5選

### 1. デザインシステムのインポート強化

企業ルール（色・フォント・コンポーネント）の読み込みが強化。

**インポート方法:**
- GitHub リポジトリ
- デザインファイル
- raw データのアップロード

**追加機能:**
- **自動修正機能**: ルール照合後に不一致を自動補正
- **管理者権限によるロック**: ガイドラインを全メンバーに強制統一

### 2. Claude Codeとの双方向連携

| 方向 | コマンド | 内容 |
|------|----------|------|
| デザイン→コーディング | `/design-sync` | デザインシステムをClaude Codeに取り込み |
| デザイン→コーディング | `Hand off to Claude Code` | デザイン・パーツ・意図を引き継いで実装 |
| コーディング→デザイン | `/design` | コードベースから直接デザイン編集 |

### 3. キャンバス上での直接編集

4つの編集方法:
1. テキストプロンプト入力
2. インラインコメント（要素クリック後にコメント記入）
3. テキスト直接編集
4. スライダー操作（スペース・色・角の丸み調整）

数百件の安定性修正により業務利用品質に向上。

### 4. 利用上限の統合

チャット・Claude Cowork・Claude Code・Claude Designの利用枠を一元管理。
- トークン消費量の削減
- エラー発生率の大幅低下

### 5. 外部ツール連携コネクタ（9種類）

| コネクタ | 用途 |
|----------|------|
| Adobe | クリエイティブツール連携 |
| Canva | テンプレートベースデザイン |
| Gamma | プレゼンテーション作成 |
| Vercel | Webデプロイ |
| Miro | ホワイトボード・UXマッピング |
| Wix | Webサイト構築 |
| Lovable | アプリプロトタイピング |
| Base44 | ノーコードアプリ |
| Replit | コーディング環境 |

対応書き出しフォーマット: PDF・PowerPoint

## 連携ワークフロー例

### デザイン先行パターン
1. Claude Designを開く
2. `/design-sync` でデザインシステムを取り込み
3. プロンプト入力・キャンバス編集でデザインを確定
4. Claude Codeへ引き継ぎ実装

### コード先行パターン
1. ターミナルで `/design` コマンド実行
2. コードベースからデザインプロジェクトを自動生成
3. ライブプロトタイプで確認・調整
4. そのまま実装完了
