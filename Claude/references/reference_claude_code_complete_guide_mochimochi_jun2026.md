## Claude Code完全ガイド2026 — 概要と要点

**公開日：** 2026年6月24日  
**著者：** mochimochi（PwC→マッキンゼー→Google→外資AIベンチャー役員）

### Claude Codeの基本理解

- PCファイルシステムへ直接アクセスし、指示を受けて自律的に処理を実行する「エージェント」
- ウェブ版AIとの違い：ツール連携・ファイル読み書き・コマンド実行が可能
- 非技術者の業務自動化（営業報告書自動生成、週次データ集計）にも活用可能

### 3つの主要エージェント体系（2026年時点）

| 体系 | 概要 |
|------|------|
| **Subagent** | 同一セッション内で複数専門エージェントを並列実行 |
| **Agent View** | 複数バックグラウンドセッションを一元管理（2026年5月リリース） |
| **Agent Teams** | 複雑なワークフロー実行（実験的機能） |

### コア技術機能

- **1Mトークンコンテキスト**：約50〜80万字分（文庫本5冊相当）を一度に処理
- **Routines**：定例業務の定期自動実行（スケジュール実行）
- **MCP連携**：GitHub・Slack・Notion・Google Workspace等と標準規格で接続
- **CLAUDE.md**：セッション横断的にルール・ペルソナを永続化
- **Hooks**：イベントトリガー型の自動処理

### 実務事例

- 週次レポート自動生成：作業時間93%削減（3時間→14分）
- 競合調査：5日分を1日に短縮
- 非技術者が日常業務を自動化

### 導入ロードマップ

1. **ステップ1**：インストール・初期設定（30分）
2. **ステップ2**：小規模タスクで習熟（1〜2時間）
3. **ステップ3**：CLAUDE.md作成と定例業務自動化（2〜3時間）
