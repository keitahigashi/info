## 新機能ダイジェスト（Week 13〜17）

### Week 17（2026年4月20〜24日）v2.1.114–v2.1.119

- **`/ultrareview`** がパブリックリサーチプレビューとしてオープン。バグ検出エージェントのフリートがクラウドで実行され、検出結果が自動的に CLI またはデスクトップに戻る
- **セッションの概要**：ターミナルがフォーカスされていない間に何が起こったかを表示
- **カスタムテーマ**：`/theme` またはプラグインから色パレットを構築・配布可能
- **ウェブ上の Claude Code**：新しいセッションサイドバーとドラッグアンドドロップレイアウトでリデザイン

### Week 16（2026年4月13〜17日）v2.1.105–v2.1.113

- **Claude Opus 4.7** が Max および Team Premium の新しいデフォルトとしてリリース
- 新しい `xhigh` エフォートレベルと `/effort` スライダーを追加
- **Routines**：スケジュール・GitHub イベント・API 呼び出しからテンプレート化されたクラウドエージェントを実行
- **`/usage`**：利用制限の駆動要因を表示
- CLI がネイティブバイナリに移行

### Week 15（2026年4月6〜10日）v2.1.92–v2.1.101

- **Ultraplan** が早期プレビューに。CLI からクラウドでプランを作成、ウェブエディタで確認後リモート実行またはローカルに戻すことが可能
- **Monitor ツール**：バックグラウンドイベントをストリーミングして会話に流し込み、Claude がログをリアルタイムで監視
- `/loop` が間隔を省略すると自動ペース
- `/team-onboarding` でセットアップを再生可能なガイドにパッケージ化
- `/autofix-pr` でターミナルから PR 自動修正をオン

### Week 14（2026年3月30〜4月3日）v2.1.86–v2.1.91

- **Computer use** がリサーチプレビューで CLI に登場。Claude がネイティブアプリを開き、UI をクリック、ターミナルから変更を確認
- `/powerup` インタラクティブレッスン
- ちらつきのない alt スクリーンレンダリング
- ツールごとの MCP 結果サイズオーバーライド（最大 500K）

### Week 13（2026年3月23〜27日）v2.1.83–v2.1.85

- **Auto mode** がリサーチプレビューでリリース。分類器が許可プロンプトを処理し、安全なアクションは中断なく実行、危険なアクションはブロック
- デスクトップアプリでのコンピュータ使用
- ウェブでの PR 自動修正
- `/` でのトランスクリプト検索
- Windows 用のネイティブ PowerShell ツール
- 条件付き `if` フック
