## 機能概要

`/ultrareview` はClaude Code v2.1.86で追加された機能。クラウド上のサンドボックスで複数のレビューエージェントを並列実行し、ブランチやPRのバグを検出する。全ての指摘は独立に再現・検証済みのため、スタイル提案ではなく実バグに限定される。

**背景**：Anthropic社内でコード出力量が200%増加する一方、レビュー能力が追いつかない不均衡が発生。実質的なコメントがつくPR割合が16%→54%に改善された実績から製品化。

## 技術的詳細

### 実行フロー

1. ローカルgitリポジトリをリモートサンドボックスにアップロード
2. 複数エージェントが異なる観点からdiffを探索
3. 検出されたバグは独立に再現・検証
4. 検証を通ったfindingsのみ返却

### 実行時間と料金

| 項目 | 内容 |
|------|------|
| 実行時間 | 10〜20分（バックグラウンド動作） |
| Pro/Max 無料枠 | 3回まで（ワンタイム、月次リセットなし、5/5期限） |
| 有料料金 | 1回あたり$5〜$20程度（extra usage課金） |

### 制限事項

- Claude.aiアカウント必須（APIキーのみはNG）
- Amazon Bedrock、Google Vertex AI、Microsoft Foundry経由では利用不可
- Zero Data Retention有効化組織では利用不可
- 変更差分のみが対象（フォルダ全体ではない）
- 検出のみで、修正は別途指示が必要

## 適用場面

費用対効果が高い重要な変更前に使用：

- 認証・認可ロジック変更
- データマイグレーション
- 並行処理リファクタ
- セキュリティコード書き換え
- 決済処理変更

| `/ultrareview` | クラウド | 本番マージ直前 | $5〜$20 |
