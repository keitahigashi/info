## 概要
Claude Codeの生産性向上に不可欠なコマンドとスキルを体系的に解説。セッション管理からコード品質自動化、並列処理まで網羅。スキルとコマンドの違いも明確化。

## 詳細

### セッション管理コマンド
- `/rewind`: Gitリセット相当の操作（4つのメニュー選択肢）
- `/branch`: 並行実験用ブランチ作成
- `/compact`: 長セッションの圧縮
- `/plan`: プランモード切り替え（Shift+Tabでも可）
- `/export`: 会話をMarkdown保存

### コード品質コマンド
- `/diff`: 変更内容を対話表示
- `/simplify`（v2.1.63追加）: 3つのサブエージェント並列レビュー（再利用性・可読性・パフォーマンス個別検査）
- `/review`: バグ・ロジックエラー検査（PR URL対応）

### 自動化コマンド
- `/batch`: 50ファイル同時変更、5〜30独立タスク並列処理（git worktree上実行）
- `/loop`: 定期監視（デフォルト10分、3日後自動終了）
- `/btw`: 会話汚染なし質問（プロンプトキャッシュ再利用でトークン削減）
- `/rc`: スマートフォンからリアルタイム操作

### ショートカット
- Ctrl+V: スクショ直接ペースト
- Ctrl+J / Option+Enter: 改行
- Ctrl+R: 履歴検索
- Esc×2: rewindメニュー

### スキル vs コマンドの違い
- スラッシュコマンド: ハードコード固定処理
- スキル: マークダウンファイルベースプロンプト（AI推論実行で並列エージェント生成可能）
