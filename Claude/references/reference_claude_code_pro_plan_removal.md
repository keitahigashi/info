## 概要
AnthropicがClaude CodeをProプラン（月額$20）からMaxプラン（月額$100以上）へ移行させた件を徹底調査した記事。正式告知なしの変更に対するユーザー反応とA/Bテストの可能性を分析。

## 詳細
- **変更内容**: 2026年4月21日頃、Claude CodeがProプランの機能一覧から削除され、Maxプラン以上のみに変更
- **告知方式**: 正式アナウンスなし。pricing pageを静かに編集する形で実施
- **ユーザー反応**: X・Redditで「告知ゼロのプラン改悪」「Proユーザー置き去り」と強い批判
- **A/Bテスト仮説**: 段階的な変更展開（A/Bテスト）の可能性を示唆
- **影響**: $20/月のProプランでClaude Codeを利用していたユーザーが$100+/月のMaxプランへの移行を迫られる可能性
