## Claude企業導入セキュリティ完全ガイド

### セキュリティ基盤

AnthropicはSOC 2 Type II・ISO 27001認証取得済み。Constitutional AIによる3層防御構造を採用：

- 訓練層
- システムプロンプト層
- 規約層

### データ保護方針

| プラン | 学習利用 | 暗号化 |
|--------|---------|--------|
| Free/Pro | 利用の可能性あり | TLS 1.3 + AES-256 |
| Team | 使用されない | TLS 1.3 + AES-256 |
| Enterprise | 使用されない | TLS 1.3 + AES-256 |

### 企業導入必須プラン

Teamプラン（5名以上）が最低ライン。必須設定：

- 管理者コンソール
- SSO（シングルサインオン）
- MFA強制設定

### API利用時の安全対策

- APIキーは環境変数管理、`.gitignore`で除外
- 月次予算アラート設定・レート制限による暴走対策
- プロンプトインジェクション対策を実施

### 企業導入5チェックポイント

1. DPA（データ処理契約）締結
2. 情報分類ルール策定
3. アクセス制御設計
4. インシデント対応手順
5. 従業員教育

### 実運用例（GENAI社）

情報を4段階分類し匿名化ルールで機密保持。営業資料作成90%削減を実現。
