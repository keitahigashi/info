## Claude Sonnet 4.8 概要

2026年3月31日、Anthropicの約51万行のTypeScriptコードがnpmパッケージに誤同梱されGitHub上で拡散。次期モデル「Claude Sonnet 4.8」の存在と新機能が明らかになった。

## 流出コードから確認された5つの新機能

### 1. KAIROS（永続型エージェント）

バックグラウンド常駐型のエージェントシステム。単発対話ではなく継続的なタスク管理・長時間タスク実行が可能。

### 2. Undercover Mode

セキュリティ関連機能。API呼び出し時に内部モデル名をマスクし使用モデルを非表示化。

### 3. Cardinal（視覚メモリーシステム）

画像処理機能の強化。Opus 4.7で98.5%の視覚精度を達成し、画像内容を文脈として保持する機構。

### 4. Mythos（最上位モデル候補）

Opusを超える最高位モデルの可能性がコード内の参照から確認。

### 5. その他の機能強化

- 高解像度画像対応
- 指示追従精度の向上
