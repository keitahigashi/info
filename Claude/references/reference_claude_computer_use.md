## 概要

Anthropicが2026年3月23日にClaudeにコンピューター操作機能を追加。アプリを開く、ブラウザ操作、スプレッドシート入力などのPC操作が可能に。

## 対応プラン

- Claude Pro: 月額$20
- Claude Max: 月額$200
- Mac版から提供開始（研究プレビュー）

## 利用可能なアプリ

- Claude Code
- Claude Cowork

## 主な機能

1. **スマート接続**: Slack、Gmail、Googleカレンダーなどのコネクターを優先使用
2. **画面操作**: 必要に応じてChrome/デスクトップを直接操作
3. **リモート実行（Dispatch）**: スマホからPC操作を指示可能

## 実用例

- 毎朝自動でメールチェック
- 毎週の分析取得
- レポート/プルリクエスト自動生成

## 制限事項

- まだ初期段階
- 機密データの取り扱いは非推奨
- 信頼できるアプリから開始推奨
