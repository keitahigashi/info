## Claude MCP 仕組みと広告データ分析活用法

### 公開日
2026年5月30日

### MCPの仕組み

**MCP（Model Context Protocol）** = AIと外部データ・ツールをつなぐ標準規格

- 「AIにとってのUSBコネクタ」と表現
- MCPサーバーがデータソースとの橋渡しを担当
- SQLを直接記述せず日本語で依頼可能

```
[Claude] ↔ [MCPサーバー] ↔ [BigQuery / DB / API]
```

### 3ステップ導入手順

1. **MCPサーバー準備**: 対応データソース選択（BigQuery・DB・SaaS等）
2. **Claude設定ファイルにサーバー登録**: `claude_desktop_config.json` にエンドポイント追加
3. **日本語で依頼**: 「先月のCPAは？」「媒体別に集計して」など自然言語で操作

### 広告データ分析の実装例（BigQuery連携）

**構成**: BigQueryに蓄積した広告データ → Claude MCP → 分析・レポート作成

**実現できること**:
- 媒体別（Google/Meta/LINE等）の実績比較
- 悪化キャンペーンの自動特定と改善案提示
- 複数媒体の時間軸別・チャネル別の深掘り分析
- レポート資料の自動生成

**プロンプト例**:
```
「先月のGoogle広告とMeta広告のCPA・CVRを媒体別に比較して」
「コンバージョンが前週比30%落ちた原因を調べて」
```

### 導入の2つの選択肢

| 方法 | 概要 | 向いている組織 |
|------|------|--------------|
| 自社構築 | MCPサーバーを自前で設定 | エンジニアリソースがある企業 |
| 連携基盤サービス利用 | 専門ベンダーに委託 | スピード重視・IT人材不足企業 |
