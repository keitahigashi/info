## v2.1.162 主な変更内容

### エージェント・インターフェース強化

- `claude agents --json` に `waitingFor` フィールド追加：待機中セッションのブロック理由（権限プロンプト等）を表示
- スラッシュコマンドのオートコンプリートで選択するとプロンプトに挿入されるように変更（以前は即時実行）
- `/effort` コマンドが選択レベルをデフォルトとして永続化する旨を確認表示するように
- Remote Controlがフッターに常時表示される「ピル」UIに変更（起動時メッセージから変更）

### バグ修正・信頼性向上

- configディレクトリが読み取り専用の場合の起動ハングを修正（インメモリ設定で起動し、起動エラーを表示）
- WebFetchの権限ルールがプリ承認済みドメインに適用されない問題を修正
- Windowsでのパスマッチングのバックスラッシュ・大文字小文字の問題を修正
- ターミナルのステータス表示が60〜120文字で切れる問題を修正（フル幅利用に）

### ポリッシュ・使いやすさ

- スタートアップ時の通知をグループ化して静粛化
- 起動警告を具体的な修正方法付きで簡潔化
- 失敗ターンを複数行エラーブロックではなく、コンパクトな警告行に変更
- バックグラウンドサービスの起動とバイナリ検証を改善
- Windsurf を Devin Desktop に名称変更（`/ide`メニュー・`/terminal-setup`・`/scroll-speed`）

## リリース全体のテーマ

「エージェント・バックグラウンド・スタートアップのポリッシュ」：JSONステータスの改善、信頼性の高い権限・ツール動作、明確な警告、スムーズなセッション処理、ネイティブおよびWindowsビルドでの静粛な起動を重点的に改善。

## 注目の変更

- `claude agents --json` の `waitingFor` フィールドは、マルチエージェント自動化のデバッグに有用。どのエージェントが何を待っているかを確認できる
- スラッシュコマンドが即時実行からプロンプト挿入に変わったため、誤実行を防止
