## 機能概要

2026年4月にAnthropicがリリースした`/ultrareview`は、複数のAIレビュアーエージェントがクラウド上で並列にコードを探索し、検出した問題を独立した検証プロセスで確認後に報告するコードレビューコマンド。

従来の単一パスレビュー（`/review`）と異なり、クラウドサンドボックス上でマルチエージェントが同時に変更を探索するため、シングルパスでは見落とされやすい脆弱性やロジックバグを検出できる。

## エージェント構成

複数のレビュアーエージェントがAnthropicのリモートサンドボックス上でフリート構成で起動し、並列処理によって「幅広いカバレッジ」を実現する。

## 使い方

```bash
# ブランチモード（現在のブランチとデフォルトブランチの差分をレビュー）
/ultrareview

# PRモード（GitHub PRを指定）
/ultrareview 1234
```

**前提条件**：
- Claude Code v2.1.86以降
- Claude.aiアカウント認証
- gitリポジトリ内での実行

## 業種別ユースケース

- **フィンテック**：認証フローやトランザクション処理の変更検証
- **SaaS企業**：重要度の高いPRやAPIフロー変更の深層レビュー
- **研究機関**：データパイプライン移行前やサブミッション前のバグ検出
