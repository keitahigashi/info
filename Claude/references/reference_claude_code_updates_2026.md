## 概要

Claude Code（Anthropic公式CLI）の2026年1〜3月アップデートを7カテゴリ別に日本語で解説。毎週更新されるリファレンス記事。

## 著者の実務的推奨

- Opus 4.6 Fastモード: 日常的なコード修正にはコスト効率重視でFast推奨
- Agent Teams: プレビュー段階のため本番ワークフロー全面採用は避けるべき
- MCP自動モード化: Slack・DB・API連携を前提としたエージェント設計が第一候補

## 2026年3月アップデート（v2.1.66〜v2.1.71、17件）

### AIモデル・コンテキスト管理
- Opus 4.6デフォルトeffort設定がmediumに変更
- プロンプトキャッシュ効率化、セッション再開時のトークン消費約600トークン削減

### エージェント・マルチエージェント
- `/loop`コマンド: 指定間隔での自動繰り返し実行
- Cronスケジューリングツール実装
- サブエージェントレポートの簡潔化

### MCP・プラグイン・拡張機能
- VS Code内のネイティブMCP管理ダイアログ追加
- スキル自己参照変数 `${CLAUDE_SKILL_DIR}` 対応
- InstructionsLoadedフックイベント実装

### IDE連携・セキュリティ
- VS Code拡張でセッション一覧表示
- スキル読み込み時のgitignore対応強化
- シンボリックリンク経由の権限バイパス修正

### パフォーマンス・安定性
- React CompilerのmemoCacheリーク修正
- REPLレンダースコープの約35MBメモリ蓄積問題解消
- stdin凍結問題の根本修正

### 開発ツール・ワークフロー
- `/claude-api`スキル追加
- Bash自動承認に11個のユーティリティコマンド追加
- Remote Controlの名前付きセッション対応

### ターミナルUI・入力・音声
- 音声STT対応言語が10言語追加（合計20言語）
- Windows/WSLのCJKクリップボード修正
- voiceキーバインドのカスタマイズ対応
