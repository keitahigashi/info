## Claude Code v2.1.178 主要アップデート（2026年6月16日）

### 変更概要
- 22件の変更を実装
- 破壊的変更・非推奨事項なし
- セキュリティ改善とバグ修正が中心

### 新機能

#### パーミッションルール「Tool(param:value)」構文
- ツールの入力パラメータを照合する新構文
- 特定モデル使用のサブエージェント起動をブロック可能
- 例: `Bash(command:rm*)` で `rm` コマンドを含むBash実行を拒否

#### ネストした `.claude` ディレクトリ対応強化
- スキル分離・モノレポ対応が強化
- サブプロジェクトごとの設定管理が容易に

### セキュリティ改善
- autoモードでの起動前評価によるセキュリティギャップ解消
- サブエージェント関連の複数不具合修正

### バグ修正
- メモリ不足によるクラッシュ対応
- 認証情報リフレッシュ後のエラー修正
- コンパクション処理の改善
- CJK IME操作の不具合解消
