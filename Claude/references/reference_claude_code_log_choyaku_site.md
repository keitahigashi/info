## 出典

CCログ超訳（非公式追跡サイト）: https://claude-code-log.com/

## 概要

Claude Code公式変更履歴を毎時自動取り込みし、AIが日本語で分析・解説する非公式追跡サイト。機能別分類・週次まとめを提供。

## サイトの主要機能

- **毎時自動更新**: 公式changelogをリアルタイムで監視・反映
- **機能別分類**: Settings / Permissions / MCP / Plugins / Sub-agents / Skills / Hooks など12以上のカテゴリで分類
- **週次まとめ記事**: 主要変更点を凝縮して提供

## 最近のアップデートトレンド

- **エージェントビューの強化**: 待機中エージェント数表示・ツール実行時間リアルタイム表示
- **セキュリティ**: シェルインジェクション対策・高コストワークフロー誤作動防止
- **安定性**: メモリリーク修正・バイナリ最適化
- **操作性**: スクリーンリーダーモード・Vimモード柔軟キーマッピング追加

## 利用上の注意

AI生成のため内容が不正確な場合があり、正確情報は公式リリースノート確認を推奨。ナレッジベースのchangelog補完リソースとして活用できる。
