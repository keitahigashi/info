## Claude Fable 5 企業導入ガイド

### 概要

- Anthropic初の最上位モデル「Mythos-class」として2026年6月9日に一般公開
- SWE-Bench Pro: 80.3%（ソフトウェア開発エージェント能力）
- Stripe社は5,000万行規模のコード移行を1日で完了

### 料金体系と課金切替

```
入力: 100万トークン当たり10ドル
出力: 100万トークン当たり50ドル
（Opus 4.8比で2倍の価格）

無料試用期間: ～6月22日
課金開始: 6月23日〜有料従量課金
```

### コスト削減策

- **プロンプトキャッシング**: 最大90%割引（1ドル/100万トークン）
- **ルーティングソリューション**: 外部ゲートウェイ利用で最大40%削減可能

### 提供形態

| 提供形態 | 特徴 |
|---------|------|
| Claude Platform | 最新機能をいち早く活用 |
| AWS Bedrock | セキュリティ境界内、リージョン選定可能 |
| Google Cloud | GCP統合運用が可能 |

### ガバナンス上の重要変更点

#### データ保持仕様（ZDR対象外）

- 送信されたすべてのプロンプトおよび出力を最大30日間、サーバー側に安全監視目的で保持
- 従来のゼロデータ保持前提のポリシー見直しが必須

#### 自動ルーティング機能

特定領域では自動的にOpus 4.8へ切り替わる:
- サイバーセキュリティ関連
- 生物学・化学領域
- モデル蒸留（Distillation）関連

### 情シス実務チェックポイント

- 法務部連携による社内ポリシー改定
- 23項目の企業導入チェックリスト実施
- Usage Credits設定の見直し
- 開発環境をサンドボックスに限定
