## Claude Code が効果を出す3条件

1. 入力データが構造化されている
2. 出力フォーマットが定型化できる
3. 効果測定が時間単位で可能

## 失敗パターン3つ

- ツール導入が目的化する
- 業務分解が不足する
- 効果測定を始める前に撤退する
