## 概要
Anthropicが2026年4月14日に発表したClaude Codeデスクトップアプリの大規模UI再設計。複数プロジェクトの並列作業を効率化する「並列エージェント時代」向けの設計。

## 詳細

### 並列セッション管理
- 複数のClaudeセッションを1つのウィンドウから並列実行
- サイドバーで全セッションを一元管理
- ステータス・プロジェクト・環境でフィルタリング、プロジェクト別グループ化
- 完了したPRは自動アーカイブ

### 統合開発ツール（1ウィンドウ内）
- **統合ターミナル**: テスト・ビルドを並行実行
- **ファイルエディタ**: コード直接編集
- **高速diffビューア**: パフォーマンス向上
- **拡張プレビューペイン**: ローカルアプリ・HTML・PDF対応
- 全パネルがドラッグ&ドロップで配置変更可能

### ワークフロー改善
- **Command+;**: 作業中断なしのクイックチャット
- **3段階表示モード**: Verbose / Normal / Summary
- **Command+/**: キーストロークドキュメント強化
- リアルタイムレスポンスストリーミング

### インフラサポート
- SSH経由リモートセッション（Mac・Linux）
- CLI・組織・ローカルデプロイ間のプラグイン互換性
- コンテキストウィンドウ使用状況モニタリングボタン

### 対象プラン
Pro、Max、Team、Enterprise（デスクトップアプリまたはClaude API経由）
