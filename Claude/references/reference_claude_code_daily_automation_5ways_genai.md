## Skills・MCP・Routinesによる5業務自動化

### 概要

「中断から深い集中状態に戻るまで平均23分かかる」という研究データを起点に、Claude Codeの4つの機能（Skills / MCP / MDファイル / Routines）を組み合わせた5業務の自動化手順を自社実運用データ付きで解説。

### 4つの実装要素

- **Skills**: 業務手順書をAIに登録。メール返信ならパターン5種（依頼/断り/進捗報告/お礼/リスケ）を定義
- **MCP**: Gmail・Googleカレンダー・Googleドライブと接続して実アクションを自動実行
- **MDファイル**: 顧客履歴・プロジェクト背景・社内ルールを事前インプット
- **Routines（自動実行）**: Cron指定で毎朝7時など定時処理

### 5業務の自動化詳細

| 業務 | 実装 | 効果 |
|------|------|------|
| 朝のタスク整理 | 議事録MD読み込み+カレンダーMCP取得→優先度順リスト生成 | 完全自動 |
| メール返信 | 未読取得→5パターンSkillで下書き保存 | 1.5時間→5分 |
| ミーティング準備 | 過去議事録の宿題抽出+背景インプット→1ページ準備資料 | 30分→5分 |
| スケジュール調整 | 空き時間抽出→候補メール送信→確認後カレンダー登録 | 15分→30秒 |
| プレゼン資料 | 過去資料MDから背景収集→定型構成でGoogleスライドAPI直接生成 | 3時間→15分 |

### 自社GENAI実績（Claude Max 20x / 月約3万円）

- 月間削減時間: 約60〜80時間
- メール返信: 1.5h→5min、資料作成: 3h→15min

### 非エンジニア向け導入ロードマップ

- Week1: メール返信（手動起動）
- Week2: タスク整理（自動実行追加）
- Month2: プレゼン作成自動化

### 重要な実装Tips

- Skillは3行から始めて段階的に改善
- MDファイルに契約金額・個人連絡先は記載しない
- プレゼン生成はブランドカラー等は手動調整が必要
