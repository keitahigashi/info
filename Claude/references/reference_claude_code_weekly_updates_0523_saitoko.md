## 主要アップデート（2026/05/23週）

### 1. 週次利用制限の50%拡大

全プランの週次総量上限が50%拡大（7月13日まで時限措置）。SpaceX Colossus 1データセンターの利用契約（NVIDIA GPU 22万台超・300MW以上の計算資源）に基づく措置。

### 2. Computer Use（CLI macOS版）

CLIからmacOSのUI操作（クリック・スクロール・スクリーンショット）が可能に。
- 対象バージョン: v2.1.85以降
- 対象プラン: Pro/Maxのリサーチプレビュー
- 主な用途: GUIテスト自動化、レガシーアプリ操作

### 3. `/skills` コマンド強化

マーケットプレイスのスキル・プラグインをリアルタイムで絞り込み検索できるように。キーワード検索による効率化が実現。

### 4. Claude Security（パブリックベータ）

コード脆弱性の自動スキャンと修正提案機能がEnterprise向けに公開。Project Glasswingとの連携で実装された機能。
