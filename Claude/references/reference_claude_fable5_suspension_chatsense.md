ChatSense Blog: https://chatsense.jp/blog/claude-fable-unavailable

## 停止の概要

- **停止日**：2026年6月12日〜（全面停止）
- **原因**：米国政府の輸出管理指示によりAnthropicが全顧客向けに停止
- **対象外モデル**：Claude Opus 4.8 など他のClaudeモデルは停止対象外

## 「使えない」の2パターン

| パターン | 内容 |
|---------|------|
| 完全停止 | 政府指令による全面的なアクセス無効化（現在） |
| 自動切り替え | 3層安全分類器が発動しOpus 4.8へ自動フォールバック（通常時） |

## 規制への対処

- **規制を回避する方法は存在しない**
- Claude Opus 4.8 への切り替えが推奨される代替策

## 法人・ビジネス利用者向け実践的対応

| 対策 | 内容 |
|------|------|
| 代替モデル選定 | Claude Opus 4.8 や他のAnthropicモデルへ切り替え |
| 社内データ管理 | モデル依存の業務プロセスの見直し |
| 権限管理 | 利用可能なモデルの管理・ポリシー更新 |
| 利用ログ確認 | 停止前後での利用状況の把握 |

## 重要な教訓：特定モデルへの業務依存を避ける
企業は「特定の1モデルに業務を依存させない」体制構築が重要。AI利用基盤の設計にフォールバック機構を組み込むことが必須。

```yaml
# agents.md でフォールバック対応の例
models:
  primary: claude-opus-4-8        # 現在の推奨
  fallback: claude-sonnet-4-6     # 軽量タスク向け
  # fable-5 は現在利用停止中
```
