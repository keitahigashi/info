## Claude Code v2.1.194–v2.1.196 主要アップデート（2026年6月30日）

### 対象バージョン
- v2.1.195：2026年6月26日リリース
- v2.1.196：2026年6月29日リリース

### 新機能

| 機能 | 説明 |
|------|------|
| 組織デフォルトモデルサポート | 管理者がコンソールで組織全体のデフォルトモデルを設定可能。未選択時は「Org default」表示 |
| セッションへの読みやすい自動命名 | 自動で説明的な名前が付与され識別・管理が容易に |
| チャット内ファイル添付のクリック可能化 | Cmd/Ctrl+クリックでFinder/Explorerに表示 |
| `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` 追加 | フルスクリーンモードでマウス操作を無効化する環境変数 |

### セキュリティ対応
- `claude mcp list/get` での自己承認サーバー起動防止（リポジトリコミットの `.claude/settings.json` 経由の自己承認を無効化）
- 外部プラグインへのインストール同意の必須化

### 主要な改善

| 項目 | 内容 |
|------|------|
| バックグラウンドセッション信頼性 | 長時間コマンドがセッション停止・再起動を超えて永続化 |
| バックグラウンドエージェント | 停止後エージェントビュー開時に自動再開 |
| `/code-review` トークン使用量 | 約25%削減 |
| ターミナルUI | ストリーミング中の未変更サブツリー走査をスキップしてパフォーマンス改善 |

### 挙動変更（要注意）
- **フックマッチャー**：部分マッチ→完全マッチに変更。ハイフン含む識別子は `mcp__brave-search__.*` 形式が必要
- **ストリーミングアイドルウォッチドッグ**：デフォルト有効化。5分間イベント無生成時に中断・再試行。無効化は `CLAUDE_ENABLE_STREAM_WATCHDOG=0`
- **リモートコントロール**：`ANTHROPIC_BASE_URL` がAnthropic以外を指す場合は無効化
