## 主要アップデート内容（2026年4月）

### 1. Subagent @mention機能

- メインセッションから別サブエージェントへタスクを委任
- 並列処理が可能に
- 使い方：`@{サブエージェント名}` に続けてタスク指示

**主な用途例：**
- 複数ファイルの同時編集（フロントエンド・バックエンド同時作業）
- リサーチとコーディングの並行実行
- テスト生成の自動化

### 2. フリッカーなし Alt-screen レンダリング

- ターミナル表示のちらつきを解消
- 長時間セッション時の視認性向上
- スクロール位置の保持改善

### 3. PowerShell ネイティブ対応

- Windows環境でPowerShellコマンドをネイティブ実行
- WSLセットアップ不要
- Windows固有の管理タスク自動化が可能に

### 4. その他の改善

- 大規模コードベースでのメモリ最適化
- ファイル検索速度改善

## ビジネスへの影響

1. エンジニア1人が「2〜3人分のタスクを同時進行」可能
2. DX人材不足補填の現実的選択肢
3. Windows環境の中小企業での導入ハードル低減
