**結論:** 非エンジニアには Claude Code を強推奨。デスクトップ版は ChatGPT と同じ感覚で使用可能。

## Claude Code が最適な3つの理由

1. **ターミナル完結**: VS Code インストール・設定が不要。フォルダ指定だけでファイルが正しい場所に生成される
2. **CLAUDE.md による品質基準化**: 「外部ライブラリは最小限」「パスワード非直書き」などのルールを CLAUDE.md に記述
3. **自律実行**: ファイル作成からテスト実行まで完成品が返される

## 実装5ステップ

```
Step 4: 修正も日本語指示（例: 「色を赤に変更」）
Step 5: CLAUDE.md で品質基準を定義
```

## 失敗回避の5注意点

- **具体的な指示**: 「いい感じに」ではなく「背景白・赤アクセント・Noto Sans JP」と明示
- **動作確認の実施**: 生成コードをブラウザで実際に確認
- **機密データの保護**: テストはダミーデータ使用、パスワードは環境変数管理
- **AIの限界認識**: 大規模システム設計・高度セキュリティは人間判断が必要
- **1機能ずつ開発**: 「小さく作→確認→次へ」のアジャイル手法
