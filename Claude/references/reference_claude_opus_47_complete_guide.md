## 概要
2026年4月16日リリースのClaude Opus 4.7を全機能網羅で解説。3つの進化軸、ベンチマーク比較、4つの新機能、価格据え置き、移行時の注意点をカバー。

## 詳細

### 3つの進化軸
1. **長時間タスク安定性**: 複雑な多段階処理の劣化防止・自己検証メカニズム強化
2. **指示追従精度**: 「知的推測」行動の抑制、ユーザー指示のリテラル解釈
3. **ビジョン処理飛躍**: 最大2,576px（約3.75MP）対応、前世代の3倍

### ベンチマーク比較

### 4つの新機能
1. **/ultrareview**: Claude Code内の専用コードレビューセッション
2. **Cyber Verification Program**: 悪意あるサイバー攻撃リクエストの自動ブロック
3. **xhigh effort level**: highとmaxの間の推論深度（複雑分析にmax遅延不要）
4. **Task Budget（パブリックベータ）**: タスクごとのトークン消費制限

### 料金（据え置き）
- 入力: $5/Mトークン
- 出力: $25/Mトークン
- 最大出力: 128Kトークン / コンテキスト: 1M

### 移行時の重要注意点
- **トークナイザ変更**: 新トークナイザでトークン数1.0〜1.35倍増（日本語/英語/コード依存）→ 実質月額コスト最大35%増の可能性
- **指示精度**: 「行間を読む」行動が減少 → 明示的な指示が必須
- **サイバータスク自動ブロック**: セキュリティリサーチ/ペネトレーションテストにはCyber Verification Program登録が必要
