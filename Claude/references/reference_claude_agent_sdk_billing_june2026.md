## Agent SDK 課金変更 2026年6月15日施行（2026年6月9日公開）

### 変更概要

2026年6月15日より、エージェント利用がチャット利用と完全に分離され、専用の月次クレジット制へ移行。

### クレジット額の設定

| プラン | 月額 | 付与クレジット |
|--------|------|--------------|
| Pro | $20 | $20相当 |
| Max 5x | $100 | $100相当 |
| Max 20x | $200 | $200相当 |

クレジットは個人単位で付与・毎月リセット・繰り越し不可。

### 影響対象

- Claude Agent SDK使用
- `claude -p`コマンド（プログラマティックモード）
- Claude Code GitHub Actions
- OpenClawなどのサードパーティ自動化アプリ

### 非影響（従来通り）

- Webチャット・アプリでの対話的利用
- Claude Codeの対話的使用

### コスト影響

- 重いワークロードで最大150倍以上のコスト増加の可能性
- 「定額使い放題」が終了し、実費化へ移行

### デフォルト動作

クレジット枯渇時は即停止（Extra Usage Credits無効時）。超過課金を避けるには設定確認が必須。

### 必要な対応

1. 6月8日頃のAnthropicからのメール確認
2. クレジット申請・設定確認
3. 旧モデルID（`claude-sonnet-4` / `claude-opus-4`）をバージョン固定IDへ更新
4. 月間API呼び出し量の概算とプラン再検討
5. クレジット超過時動作確認（エラーか従量か）

### よくある誤解

- 「Claude Codeが有料化」→ 誤（対話的使用は従来通り）
- 「全ユーザー値上げ」→ 誤（自動化処理の実費化）
