- [ ] SSO（SAML/OIDC）が必要か

### 失敗パターン4選

1. **無料続行**: 利用量が増えても課金しない→コンテキスト不足で品質低下
2. **過度なアップグレード**: 利用パターン分析なしでMax 20x即選択→過剰投資
3. **経費管理の非効率化**: 個人クレカ→清算の繰り返し→TeamプランでCard一元化を推奨
4. **為替対応欠落**: ドル建て料金の円換算見積もりを怠りコスト予測がずれる
