## 概要

2026年5月14日、Anthropicはサブスクリプション体系の大幅刷新を発表。エージェント的利用（Agent SDK・`claude -p`コマンド等）を従来の対話型利用枠から分離し、別枠の月間クレジットから消費する制度を2026年6月15日より適用する。

## 変更内容

### エージェント利用の別枠化

**対象となる利用方法**
- Claude Agent SDK（Python/TypeScript）
- `claude -p` コマンド（非インタラクティブモード）
- Claude Code GitHub Actions
- Agent SDK経由のサードパーティアプリ

**月額クレジット付与額**

| プラン | クレジット |
|--------|----------|
| Claude Pro | $20 |
| Max 5x | $100 |
| Max 20x | $200 |

### 利用ルール

- クレジットは月単位リセット（繰り越しなし）
- 超過分は従量課金（標準APIレート）で追加請求
- 対話型利用（Web・IDE）は従来通りサブスク枠

## ユーザーへの影響

エージェント機能を多用するユーザーにとっては、月次クレジットを早期に消費する可能性があり**実質的な値上げ**となるケースがある。従来の週次リセットと異なり月次リセットのため、使い方によっては月末にクレジット不足が発生する。

## 適用日

**2026年6月15日**
