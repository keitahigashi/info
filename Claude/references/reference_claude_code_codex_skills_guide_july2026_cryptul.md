## 概要

## Skillsの基本概念

- Agent Skillsは「作業のやり方をフォルダにまとめてエージェントに配る仕組み」
- 中核ファイルは `SKILL.md`（テキストファイル）
- フォルダ内に指示・スクリプト・リソースをまとめる構造

## 探し方（3ルート）

| 入手先 | 信頼度 | 説明 |
|--------|--------|------|
| 公式・ベンダー公式 | 最高 | AWS・Shopify・Vercel・Cloudflare等 |
| キュレーションリスト | 高 | awesome系。mattpocock/skills（約156,000スター）等 |
| マーケットプレイス | 中 | skills.sh（Vercel Labs, 2026年1月公開）等 |

## カテゴリ別スキルカタログ（15カテゴリ）

開発の型・実装 / テスト・ブラウザ自動操作 / 文書・表計算 / スライド・プレゼン / UI・デザイン / 動画・メディア / 翻訳・アクセシビリティ / SEO・GEO / コンテンツ・ライティング / マーケティング・広告 / データ分析・可視化 / リサーチ・ナレッジ管理 / プロダクトマネジメント / バックオフィス・業務 / セキュリティ・監査

## インストール方法（3種類）

1. フォルダに直置き（`~/.claude/skills` または `.claude/skills`）
2. プラグイン経由で導入
3. インストールツール（skills.sh）活用

## セキュリティチェック項目

導入前確認事項：外部通信の有無 / 危険コマンドの存在 / 認証情報要求 / プロンプトインジェクション

## 運用のコツ

- `description` で発火条件を明確に書く
- 数を絞る（入れ過ぎ禁止）
- チーム共有での定期棚卸し

## Codexとの違い

- 録画機能による自動スキル生成が可能（Claude Codeには未実装）
