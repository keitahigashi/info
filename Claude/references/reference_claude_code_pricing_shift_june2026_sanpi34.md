## 6月15日施行：変更の骨子

2026年6月15日から、対話的なClaude Code利用と自動化・プログラム的利用が分けて扱われる。単なる値上げではなく、AIが「個人ツール」から「開発インフラ」へ移行する象徴的な転換。

## 利用形態別の扱い

| 分類 | 具体例 |
|------|--------|
| **従来通り**（サブスク利用枠） | Claude.aiでの会話、IDE上での対話的利用、Claude Cowork |
| **新クレジット対象** | Agent SDK、`claude -p`、GitHub Actions連携、第三者エージェント |

## プラン別の月次クレジット配分

| プラン | 月次クレジット |
|--------|--------------|
| Pro | $20 |
| Max 5x | $100 |
| Max 20x | $200 |
| Team Premium | $100 |

## ユーザー層別の影響

- **ライトユーザー**: 軽微な影響
- **中規模ユーザー**: タスク設計の工夫が必須
- **ヘビーユーザー**: 予算管理型への転換が必要

## 開発者に必須の新スキル

1. タスクのAI向け分解力
2. コンテキスト設計力
3. ROI判断力

## 実務アクション

- 現在の利用方法の棚卸し
- AIに任せるタスクの分類・優先順位付け
- チーム運用ルールの策定
