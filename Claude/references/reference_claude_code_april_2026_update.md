## 概要
Claude Code v2.1.89〜v2.1.92の4バージョンで追加された新機能・パフォーマンス改善・安定性修正の総括。/powerupインタラクティブ学習、MCP 500K文字上限、defer権限決定、名前付きサブエージェントなど。

## 詳細

### /powerup コマンド（v2.1.90）
- Claude Code内蔵のインタラクティブ学習システム
- アニメーションデモで機能をウォークスルー（外部ドキュメントへの誘導ではなくツール内で完結）

### MCP 500K文字制限（v2.1.91）
- ツール結果サイズが最大500,000文字に設定可能
- 設定方法:
```json
{
  "_meta": {
    "anthropic/maxResultSizeChars": 500000
  }
}
```
- ログ分析・DB クエリ・コード読取ツールの制約を解消

### defer 権限決定（v2.1.89）
- PreToolUseフックにallow/denyに加え第3の選択肢「defer」を追加
- 実行を一時停止し外部シグナルで再開を待機
- CI/CD承認ゲートやマルチエージェント連携に活用

### /cost コマンド強化（v2.1.92）
- モデル別・キャッシュヒット別のコスト分析（従来のセッション合計から刷新）
- Proサブスクライバー向けプロンプトキャッシュ有効期限インジケーター

### 名前付きサブエージェント（v2.1.89）
- `@mention`構文でサブエージェントを直接指定可能
- 匿名エージェント選択から名前指定へ

### パフォーマンス改善
- Writeツール: diff計算が60%高速化
- SSEトランスポート: O(n²)→O(n)に計算量改善
- 長時間セッション: レスポンス劣化問題を改善

### 安定性修正（v2.1.89〜v2.1.92）
- tmuxウィンドウ終了後のサブエージェント再生成
- `--resume`セッションでのプロンプトキャッシュミス回帰（20+バージョン遅延）
- auto-compact無限ループ修正
- CJK文字・絵文字のプロンプト履歴欠落修正
- ネストされたCLAUDE.mdコンテキスト再注入修正

### 削除された機能
- `/tag` コマンド
- `/vim` キーバインディングサポート
- `/release-notes` はインタラクティブバージョンピッカーに再設計
