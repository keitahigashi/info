## ベストプラクティス3原則

1. CLAUDE.mdにコーディング規約・テスト要件を明記して生成品質を向上
2. 小タスクから段階的に開始してスコープを拡大
3. セキュリティスキャン+人間レビューを必ず併用

Auto Modeで自律実装を進める間に並行作業を行い、「生産性2〜3倍向上」を達成できると示唆。
