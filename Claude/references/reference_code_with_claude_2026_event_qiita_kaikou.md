## SpaceX提携の技術詳細

- 「Colossus 1データセンターの全計算資源へのアクセス権」を獲得
- GPU種別: NVIDIA H100・H200・GB200
- 規模: 22万台以上のGPU、300MW以上の電力容量
- 利用開始: **1ヶ月以内**に計算容量を展開

## Claude Codeのレート制限改善内容

| 項目 | 変更内容 |
|------|---------|
| 5時間レート制限 | 全プラン（Pro/Max/Team/Enterprise）で**2倍に拡大** |
| ピーク時間制限 | Pro・Maxで**完全撤廃** — 時間帯による制限なし |
| Claude Opus APIレート | 増量（入力800k→5M、出力160k→400kトークン） |

## その他の主要発表

- **Code Reviewの全社展開**: Claude Code CLIのコードレビュー機能をEnterpriseに全面展開
- **Claude Code CLIの複数アップデート**: プラグインURLロード、自動更新機能追加
- **金融サービス向けエージェントテンプレート**: 10種類（ピッチビルダー・決算レビュー等）
- **Microsoft 365アドインのGA**: Office製品群とのネイティブ統合
