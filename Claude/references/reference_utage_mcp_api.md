## 概要
国産MAツール「UTAGE」（株式会社Fountain、累計17,000社導入）がAPI・MCPを公開。Claude/Claude CodeからLP自動作成、ファネル構築、ページコンテンツ配置、配信管理が自然言語指示で実行可能に。

## 詳細

### 対応AIツール
- Claude、Claude Code、OpenClawなどのAIエージェント

### 公開機能
- **ファネル管理**: 作成・一覧取得・更新、ステップの作成・並び替え・更新
- **ページ管理**: LP・セールスページ・サンキューページの作成・取得・更新・削除
- **ページコンテンツ構築**: セクション・行・カラム配置、テキスト・画像・ボタン・フォーム・動画・カウントダウンタイマー等、HTML直接編集
- **メディア管理**: 画像・動画・音声の検索・一覧取得・フォルダ管理
- **配信管理**: メール配信・LINE配信アカウントの作成・一覧取得、配信シナリオ管理

### 活用シナリオ
- セミナー集客LP自動作成（自然言語指示→ファネル作成→ステップ追加→ページ作成→コンテンツ配置を自動実行）
- オンライン講座販売ファネル（複数ステップ自動設計）
- 既存ファネル一括修正（画像差し替え・CTA統一）
- 外部システム連携（CRM・広告管理・予約・EC）

### 今後の拡張予定
メール自動配信、LINE自動化、会員サイト構築、オートウェビナー管理、決済連携、顧客管理、アクセス解析、コンバージョンデータ取得
