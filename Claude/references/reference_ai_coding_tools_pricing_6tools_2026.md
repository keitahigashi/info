## 見落とされがちな隠れコスト3点

1. **クレジット/トークン上限超過料金** — プランの月額以外にAPI超過分が課金される場合がある
2. **モデル選択による消費量の違い** — 上位モデルを選ぶと同じ作業量でもトークン消費が急増
3. **年払い割引の活用** — 年払いで最大20%の節約が可能（Claude Code Teamなど）

## Claude Code固有の注意点

- Pro/Max切り替えはプランのアップグレードで対応（月途中の変更は日割り計算）
- 2026年6月時点の「バケット型料金」（使用量に応じたクレジット消費）に移行済み
- 企業導入ではTeam Standard（$25/席）またはEnterprise（要見積もり）が選択肢
