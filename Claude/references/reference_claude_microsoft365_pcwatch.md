## Claude for Microsoft 365 の正式公開

### 公開日
2026年5月8日（Anthropicの発表は5月7日）

### 概要
Anthropicが「Claude for Microsoft 365」アドインを正式版として公開。Excel・Word・PowerPoint向けアドインが正式版、Outlook向けアドインはパブリックベータ版として提供開始。有料プランユーザーがWindows/macOS版Officeで利用可能。

## 各アドインの機能

| アドイン | 主な機能 |
|---------|---------|
| Claude for Excel | 数式を維持しながらの更新・モデル構築 |
| Claude for Word | 変更履歴を通じた編集・コメント返答 |
| Claude for PowerPoint | テンプレートを使用したスライド作成 |
| Claude for Outlook | 受信トレイ整理・返信下書き（パブリックβ） |

## 特筆機能：アプリ間コンテキスト維持
- 複数ファイルを並べて開いた際に自動的に変更内容が反映される
- アプリをまたいでチャットのコンテキストを保持
- 例: Wordで作成した文章をExcelのデータと連携して分析可能

## 利用条件
- Claudeの有料プランユーザー
- Windows/macOS版Microsoft Office
- Excel・Word・PowerPointは正式版、Outlookはベータ版
