## 料金体系

| プラン | 月額/人 | 特徴 |
|---|---|---|
| Standard | $25 | 基本使用量（2026年1月に$50→$25値下げ） |
| Premium | $125 | Standard比5倍の使用量（旧$150） |

**コスト構造：** 「シート基本料金 + 追加使用量（従量課金）」の二層構造  
Standardで追加$100使うなら Premium移行が経済合理的。

## 管理画面の主要セクション

| セクション | 主な機能 |
|---|---|
| 組織 | メンバー役割設定、シート割当 |
| 請求 | プラン管理、支払い方法、自動チャージ |
| 使用量 | 利用上限設定（組織全体/シート種別/個人別の3段階） |
| IDとアクセス管理 | SSO、ドメイン認証、セキュリティ制御 |
| Claude Code | 環境別ON/OFF、権限モード、Code Review管理 |

## Bedrockからの移行ジャーニー（カンリー実例）

| フェーズ | 時期 | 内容 |
|---|---|---|
| Phase 1 | 2025年6月〜 | AWS Bedrock（従量課金）から開始 |
| Phase 2 | 2025年末 | ヘビーユーザーをTeamプランに切り替え |
| Phase 3 | 2026年1月 | 全員Teamプランへ統一 |

移行トリガー：2026年1月の値下げ（Standard $50→$25）＋StandardでClaude Code対応。

## 運用上の注意点

### 新機能のサイレント追加問題
- Pro/Maxより数日〜数週間遅れて展開される
- 管理画面にデフォルトブロック状態で追加されることが多い
- 新機能キャッチアップエンジニア ≠ 管理画面管理者 → 機能未活用リスク

### 詳細分析の限界
- 個人別支出は管理画面で確認可能
- セッション単位の詳細分析にはOpenTelemetry連携が必要

## セキュリティ推奨設定

| 設定 | 推奨 |
|---|---|
| 招待リンク・メンバー招待 | OFF推奨 |
| 「組織の作成を制限」 | ON推奨（シャドーIT対策） |
| バイパス権限モード | OFF推奨（セキュリティリスク高） |

## 利用可能範囲

- **できること**：メンバー管理、請求管理、機能制御、基本アナリティクス
- **できないこと**：セッション単位詳細分析、5人未満・151人以上での利用、リアルタイム監視
