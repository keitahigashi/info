## 概要

2026年5月6日、Anthropicが金融機関向けのClaudeエージェントテンプレート10種類を発表。Claude Cowork・Claude Code・Claude Managed Agentとして利用でき、Microsoft Office製品（Excel・PowerPoint・Word・Outlook）と連携可能。

## リサーチ・顧客対応機能（5種）

| テンプレート | 主な機能 |
|------------|---------|
| ピッチビルダー | ターゲットリスト作成、比較対象企業分析、ピッチブック草案作成 |
| ミーティング準備 | 顧客・取引相手の概要資料作成 |
| 決算レビュー担当 | 決算資料読み込みと投資テーマ関連変更点抽出 |
| モデル・ビルダー | 財務モデルの作成・維持管理 |
| マーケット・リサーチャー | セクター動向追跡とリスク項目抽出 |

## 財務・業務機能（5種）

| テンプレート | 主な機能 |
|------------|---------|
| バリュエーション・レビュー | 企業評価の検証 |
| 総勘定元帳照合 | 勘定照合の自動化 |
| 月次決算処理 | 月次クロージング支援 |
| 財務諸表監査レビュー | 監査レビュー補助 |
| KYCスクリーニング | 顧客確認プロセス自動化 |

- **新規追加**: Dun & Bradstreet、Fiscal AI、IBISWorld
