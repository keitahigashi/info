## 概要

Anthropicが2026年4月28日発表した「Claude for Creative Work」の徹底解説記事。9コネクタの設定手順、各ツール別実践ユースケース、ハンズオンデモ（テキストプロンプトだけでローポリ城のBlenderシーン生成）を収録。

## 対象コネクタ一覧

| コネクタ | 主な用途 |
|----------|----------|
| Adobe Creative Cloud | 50以上のツール横断（Photoshop・Illustrator・Premiere等）の一括操作 |
| Blender | シーン解析・スクリプト生成・Python API自然言語インターフェース |
| Autodesk Fusion | 会話形式での3Dモデル設計 |
| Ableton | ドキュメントアシスタント（公式ドキュメントに基づく回答） |
| Affinity by Canva | バッチ画像処理・レイヤー管理・ファイルエクスポート自動化 |
| SketchUp | 部屋・家具・サイトの3Dモデル生成 |
| Splice | ロイヤリティフリーサンプルの自然言語検索 |
| Resolume Arena | 映像パフォーマンスワークフロー統合 |
| Resolume Wire | 映像エフェクトチェーン管理 |

## 主な機能カテゴリ

1. **ツール習得支援** - ソフトウェアの使い方をコンテキスト依存で解説
2. **カスタムコード生成** - ツール固有スクリプト（BlenderのPython等）の自動生成
3. **複数アプリ間データ連携** - ソフトウェア横断でのアセット変換・転送
4. **反復タスク自動化** - バッチ処理・フォーマット変換・ファイル整理

## 実践ユースケース

| 業種 | 活用例 |
|------|--------|
| 映像制作 | Premiereでの複数クリップ一括編集・カラーグレード適用 |
| 建築設計 | SketchUp/Autodesk Fusionでの概念→3Dモデル変換 |
| グラフィックデザイン | Illustratorでのアイコンセット一括生成・ブランドスタイル適用 |
| 音楽制作 | Abletonの操作を自然言語で質問・解決 |

## 利用要件・料金

- 全Claudeプランで利用可能（Freeプラン含む）
- Adobe署名不要で開始可能（サインインで上限・ツール数・セッション保存が拡張）
- グローバル提供（地域制限なし）

## 技術基盤

- MCP（Model Context Protocol）上に構築
- オープン設計：Claude以外のLLMからも利用可能
- Anthropicはパートナー企業（特にBlender）への投資も実施
