## 背景・意義

従来のチャット形式AIとは異なり、Claude Codeはタスクを受け取り複数ファイルにまたがって自律的に編集・実行・確認を繰り返し完了まで自走できるエージェント。本書は日本語圏でのClaude Code普及を加速する重要な学習リソースとして位置づけられる。
