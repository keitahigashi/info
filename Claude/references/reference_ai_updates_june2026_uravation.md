## Anthropicの主要アップデート

### Anthropic Marketplace開始
- Claude向けサードパーティツール・拡張機能の統一プラットフォーム
- 周辺ツール選定コストが低減、エコシステムの集約が進む

### Claude Code Dynamic Workflows
- **最大1,000並列処理**対応
- 大規模リスト処理（営業先リスト分析・コードベース監査等）が現実的な時間内で完了可能に

### Computer Use API実用段階化
- PC操作自動化がWebアプリ定型業務レベルで「人が監督する運用」が必須な実用段階に到達
