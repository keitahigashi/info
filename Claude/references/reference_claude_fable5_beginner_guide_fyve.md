## Claude Fable 5 入門ガイド

### 公開日
2026年6月10日

### Fable 5 の正確な位置づけ

> 「"全部これに乗り換える"のは間違い」

Fable 5は**重い仕事専用の特別なツール**。軽い作業（メール返信・簡単な要約・コードの小修正）では従来モデル（Sonnet・Haiku）で十分。コスト効率を考えると使い分けが必須。

### 読み方・基本情報

- **読み方**: クロード・フェイブル・ファイブ
- **リリース**: 2026年6月9日
- **位置づけ**: Mythos 級モデルの初の一般公開版
- **特性**: 「賢いが遅く、料金が約2倍」

### 使い分け判断基準

| 用途 | 推奨モデル |
|------|-----------|
| 日常的なチャット・軽い質問 | Haiku 4.5 / Sonnet 4.6 |
| 通常の開発・文章生成 | Opus 4.8 |
| 複雑なマルチファイル実装・長時間エージェント | **Fable 5** |
| 調査→実装→テストの一連フロー | **Fable 5** |

### 3ステップ活用法（初心者向け）

1. **6月22日までの無料期間に試す**
   - claude.ai の全プランで無料アクセス可能
   - 普段Opus 4.8でやっている重めのタスクをFable 5で比較

2. **コスト感覚をつかむ**
   - トークンカウンターで同じ作業のコストを比較
   - Fable 5 vs Opus 4.8 の差額を記録

3. **使い分けルールを決める**
   - 「30分以上かかる作業 → Fable 5」などマイルールを設定
   - 週次でコストを確認し、費用対効果を評価

### よくある誤解と正解

| 誤解 | 正解 |
|------|------|
| すべての作業がFable 5で最良 | 軽い作業はSonnet/Haikuの方がコスパ良好 |
| 無料期間中は使い放題 | 6月22日以降は従量クレジット制に移行 |
| MythosとFableは同じ | Mythosは安全装置なし（限定提供）、Fableは安全装置付き（一般公開） |
