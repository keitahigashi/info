## 概要
Claude Codeの全機能を1ページにまとめた包括的チートシート。毎日CHANGELOGをチェックし新機能に「NEW」バッジを付与して自動更新。単一HTMLファイルで軽量・無料・サインアップ不要。印刷可能（A4横向き、Ctrl+P）。

## カバー範囲
- キーボードショートカット（Mac/Windows両対応）
- スラッシュコマンド一覧
- CLIフラグ
- MCPサーバー設定
- メモリファイル（CLAUDE.md等）
- Skills システム
- Agents（カスタムサブエージェント）
- 環境変数
- ワークフロー

## 特徴
- 毎日自動更新で常に最新
- モバイルでも動作
- Hacker Newsで高評価
- 一部誤りの指摘あり（作者が継続修正中）

## 活用方法
日常のクイックリファレンスとして、またClaude Code新機能のキャッチアップに有用。
