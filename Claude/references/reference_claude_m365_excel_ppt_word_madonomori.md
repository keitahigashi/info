## Claude for Microsoft 365 概要

Anthropicが2026年5月8日、「Claude for Microsoft 365」として以下のアドインを一般提供開始:
- **Claude for Excel**（一般提供）
- **Claude for PowerPoint**（一般提供）
- **Claude for Word**（一般提供）
- **Claude for Outlook**（ベータ版）

入手: Microsoftマーケットプレースから無償ダウンロード可（利用には有料プラン加入が必要）

## アプリ間コンテキスト共有機能

```
Excel → PowerPoint → Word間で
会話内容と文脈を自動引き継ぎ
```

- Excelで集計したデータをPowerPointやWordで参照可能
- 一方のアプリでの操作が他のアプリのClaudeにも反映される
- シームレスなクロスアプリワークフローを実現

## Claude Outlook版（ベータ）の機能

- 受信トレイを自然言語で分類・優先順位付け
- メール返信の下書き自動生成
- カレンダー連携による空き時間検索

## 搭載モデル

Claude Opus 4.7を活用。

## 関連する背景

Claude Connectors（2026年4月末発表）の一環として、AdobeやMicrosoft 365との統合が進んでいる。Claude Codeとは独立した、ビジネスユーザー向けの拡張機能として提供。
