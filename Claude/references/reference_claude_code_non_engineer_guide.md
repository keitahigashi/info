## 概要
営業・マーケ・経営企画担当者向けに、Claude Codeを日本語プロンプトで活用する5つの業務シーンを紹介。100社の研修実績に基づく失敗パターンも掲載。

## 詳細

### 5つの業務シーン
| シーン | 削減率 | 削減時間 |
|-------|--------|---------|
| CSVデータ分析→グラフ生成 | 80-85% | 60分→5-15分 |
| 競合調査レポート自動生成 | 75-80% | 4時間→30-45分 |
| 議事録→アクションアイテム抽出 | 75-80% | 60分→15分 |
| 週次レポート自動作成 | 85-90% | 200分→15-20分 |
| メール返信ドラフト一括生成 | 83-85% | 100-120分→15-20分 |

### 活用の3つの型
- 型1: ファイル処理型（CSV加工）★☆☆
- 型2: 情報収集型（URL調査）★★☆
- 型3: 自動化型（定期処理）★★★

### 避けるべき5つのミス（100社研修実績）
1. ファイルパスの不完全指定
2. 権限設定の無視（削除実行の危険）
3. 出力情報の無検証使用（特に数字）
4. プロンプト過剰で処理中断
5. 日本語ファイル名の文字化け

### 対策
「削除や移動は必ず確認を取ってから」と事前に指示。重要な数字は原典確認必須。
