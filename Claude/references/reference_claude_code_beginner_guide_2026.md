## 概要

## 詳細

### Claude Codeの定義
Anthropic社開発の「ターミナルベースのAIエージェント」。チャットではなく「実際に手を動かすアシスタント」。2026年3月時点でOpus 4.6がデフォルトモデル。

### 主要機能
- **コーディング**: バグ修正、新機能実装、テスト自動生成、セキュリティスキャン（SWE-bench Verified 80.9%）
- **ライティング**: Markdown直接保存、メール・企画書ドラフト
- **データ分析**: CSV/Excel読込、決算レポート、グラフ自動生成
- **ファイル管理**: フォルダ整理、一括リネーム、バックアップ自動化
- **業務自動化**: MCP経由でGoogle/Notion/Slack連携

### 2026年新機能
- ボイスモード（20言語対応）
- /loopコマンド（繰り返しタスク自動実行）
- 100万トークンコンテキスト
- MCP Elicitation（インタラクティブ操作）
- Coworkデスクトップアプリ

### 料金体系
| プラン | 月額 | Claude Code |
|--------|------|-------------|
| Free | $0 | 利用不可 |
| Pro | $20 | 基本利用可 |
| Max 5x | $100 | Opus 4.6安定アクセス |
| Max 20x | $200 | 大規模プロジェクト向け |

### インストール
- 前提: Claude Proプラン以上 or APIクレジット
- Windows: Git for Windowsの事前インストール必要
- 基本コマンド: `/plan`（計画）、`/voice`（音声）、`/effort`（思考量調整）

### 競合比較
- **Claude Code**: 複雑なコード変更・大規模ファイル処理で優位
- **ChatGPT**: UI表示・マルチモーダル性
- **Gemini CLI**: 無料枠・Google連携が強力
