## Claude Code × MCPおすすめサーバー一覧

### コンセプト

MCPサーバーを活用することでClaude Codeが「賢いアシスタント」から「実働するチームメンバー」に変わる。

### 推奨サーバー10選

| # | サーバー名 | カテゴリ | 必須度 | 用途 |
|---|-----------|---------|--------|------|
| 1 | GitHub MCP | コード管理 | ★★★ | Issue・PR・ブランチを自動操作 |
| 2 | Context7 | ドキュメント参照 | ★★★ | 最新ドキュメントをリアルタイム参照 |
| 3 | Playwright MCP | ブラウザ操作 | ★★★ | 自動テスト・UI確認を実行 |
| 4 | Brave Search MCP | Web検索 | ★★☆ | インターネット検索機能を追加 |
| 5 | PostgreSQL MCP | DB管理 | ★★★ | 自然言語でDB操作 |
| 6 | SQLite MCP | DB管理 | ★★☆ | ローカル開発向け |
| 7 | Figma MCP | デザイン連携 | ★★☆ | デザインデータをコードに反映 |
| 8 | Slack MCP | チーム連携 | ★★☆ | メッセージ送信を自動化 |
| 9 | Notion MCP | ドキュメント | ★★☆ | 仕様書と連携 |
| 10 | Memory MCP | 永続記憶 | ★★☆ | セッション間での記憶保持 |

### セットアップ基本コマンド

```bash
claude mcp add <サーバー名> --scope user -- <起動コマンド>
```

### スコープ3種類

- **ユーザーグローバル**: 全プロジェクトで利用可能
- **ローカル**: 現在のプロジェクトのみ
- **プロジェクト共有**: チームメンバーと設定共有

### セキュリティ原則

- 最小権限の原則を適用
- 本番環境への接続は慎重に
- 読み取り専用スコープから始める
