## デザイナー向けClaude Code 18コマンド完全解説

### 概要

- **著者**：大原かほ（UIデザイナー）
- **公開日**：2026年5月11日
- **対応バージョン**：Claude Code v2.1.x / Opus 4.7
- **主張**：ショートカットの引き出しの多さで効率が上がる。同じコマンドでも習熟度で作業効率が大きく変わる

### 18コマンド（4カテゴリ）

#### 日常編（1〜7）

| コマンド | 機能 |
|---------|------|
| `Esc Esc` | 巻き戻し機能 |
| `Shift + Tab` | 権限モード切替 |
| `/compact` | コンテキスト圧縮 |
| `@ メンション` | ファイル/URL読み込み |
| `! プレフィックス` | シェルコマンド直挿し |
| `/resume` / `claude -r` | セッション再開 |
| `\\ + Enter` / `Ctrl + G` | 長文入力 |

#### 品質向上編（8〜11）

| コマンド | 機能 |
|---------|------|
| `/effort` | 推論強度調整 |
| `/ultrareview` | マルチエージェント・クラウドレビュー |
| `/security-review` | セキュリティ専門レビュー |
| `/loop` | 自動反復改善 |

#### 並列・自動化編（12〜15）

| コマンド | 機能 |
|---------|------|
| `/batch` | 並列処理 |
| オートランモード | 自律実行 |
| `fewer_permission_prompts` | 許可プロンプト削減 |
| `cc max` でターミナル分割 | 並列セッション管理 |

#### 隠れ便利編（16〜18）

| コマンド | 機能 |
|---------|------|
| `/btw` | サイドタスク処理 |
| `/insights` | 使い方分析レポート |
| `design.md` | デザインガイドファイル |

### 学習優先度

- **初心者向け**：「今日から使うべき5つ」から段階的に紹介
- **中級者向け**：「月単位で活用したい8つ」まで段階的にステップアップ
