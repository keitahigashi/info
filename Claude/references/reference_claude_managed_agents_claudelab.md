## Claude Managed Agentsとは

2026年4月8日開始のパブリックベータ。「AIエージェントを10倍速く本番へ」をキャッチフレーズとし、これまで自前構築が必要だったエージェントインフラをマネージドサービスとして提供。

## 提供される主なインフラ機能

| 機能 | 説明 |
|---|---|
| サンドボックス実行 | 安全な隔離実行環境 |
| 認証管理 | OAuth・APIキー管理を自動化 |
| チェックポイント | 長時間タスクの中断・再開 |
| セッション管理 | 状態保持・コンテキスト管理 |

## 料金体系

- **基本料金**：トークン消費量（通常API料金と同じ）
- **インフラ料金**：**$0.08/セッション時間**
- 長時間の自律実行タスク・複雑なワークフロー自動化に最適

## Messages APIとの比較

| 項目 | Messages API | Managed Agents |
|---|---|---|
| インフラ構築 | 自前実装が必要 | マネージド（不要） |
| 長時間実行 | 手動管理 | チェックポイント自動化 |
| セッション管理 | 自前実装 | 組み込み済み |
| 料金 | トークンのみ | トークン + $0.08/時間 |

## 想定ユースケース

- 複雑なワークフロー自動化
- 7〜24時間にわたる長時間エージェント実行
- コードレビュー・PRマージ・デプロイパイプライン
- データ収集・変換・分析の自律エージェント

## 採用企業（ベータパートナー）

Notion・楽天・Asana等がベータパートナーとして採用。

## 注意事項

- 2026年4月時点はリサーチプレビュー段階
- 本番環境での利用は段階的ロールアウトを推奨
